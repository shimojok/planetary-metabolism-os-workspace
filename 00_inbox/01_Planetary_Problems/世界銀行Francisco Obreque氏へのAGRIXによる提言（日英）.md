下記は、私達のMBT Sustainable Cycle の内容です。

  

AGRIX Projectで採用するMBT Sustainable Cycle は、MBT55微生物群(NB菌)を発酵機で廃棄物と混合させ、24時間で完全に分解するシステムです。分解により生成される機能性堆肥は、農作物に栄養素を与え、肥料を削減することができ、土壌の炭素量も増加させ、さらに、腐植質生成により、長期的に貯留させる特徴もあり、温暖化抑制に貢献します。機能性飼料の生産も可能です。

使用する廃棄物は、食品残渣、農作物残渣、海産物残渣、食品加工における廃棄物、鶏糞や豚糞や牛糞などの家畜の排泄物、果実の皮などの非可食部、海産物の甲羅、鶏や豚の骨などです。

廃棄物は、通常、燃焼または埋め立てをするため、二酸化炭素の排出源となります。それを燃焼・埋め立てをせずに、発酵機により24時間で発酵・分解し資源化します。

また、MBT機能性堆肥は、生産性と品質を向上させるほか、農作物の根を構成するスベリンを5倍程度に増加させます。スベリンは炭素化合物であり炭素を固定します。また、MBT堆肥は土壌中で鉄、アルミニウム、その他の元素との合成により腐植質を生成し、長期的に炭素を固定させると言われています。

## Francisco Obreque 氏への提言：AGRIX Project と世界銀行の連携による食料安全保障と気候変動対策の強化

### はじめに

私は、AGRIX Project のメンバーとして、農業・気候変動対策の責任者を務めております。貴殿が執筆された記事や世界銀行の取り組みを拝見し、AGRIX Project が世界銀行の目標達成に大きく貢献できる可能性を確信しました。本提言では、世界銀行が抱える課題と AGRIX Project の優位性を明らかにし、両者の連携による食料安全保障と気候変動対策の強化を提案いたします。

### 1. 世界銀行が解決したい課題

世界銀行は、以下の課題解決に取り組んでいます。

* **食料安全保障の強化:** 世界人口の増加と気候変動の影響により、食料安全保障が脅かされています。特に、西アフリカ地域では、肥料不足、土壌劣化、干ばつなどの問題が深刻化しており、食料危機が懸念されています。
* **気候変動対策:** 農業部門は、温室効果ガス排出量の約10～12%を占め、気候変動の主要な原因の一つとなっています。土壌劣化や森林伐採は、さらに温室効果ガス排出量を増加させ、悪循環を生み出す可能性があります。

### 2. 解決のための施策

世界銀行は、以下の施策を実施しています。

* **肥料への投資:** 肥料利用効率の向上や、持続可能な肥料生産技術の開発・普及を支援しています。
* **土壌健康の改善:** 土壌管理技術の改善や、有機農業への移行を支援しています。
* **気候変動対策への投資:** 農業部門における温室効果ガス排出量削減技術の開発・普及を支援しています。

### 3. 実現できていない内容と原因

これらの施策は一定の効果を上げているものの、以下の課題があり、十分な成果が得られていない状況です。

* **技術の普及:** 新技術やノウハウの普及が遅れており、多くの農家が恩恵を受けられていません。
* **資金不足:** 途上国には、持続可能な農業への投資に必要な資金が不足しています。
* **政策・制度の整備:** 持続可能な農業を推進するための政策・制度が整備されていない地域があります。

### 4. AGRIX Project の優位性

AGRIX Project は、以下の点において、世界銀行の課題解決に貢献できる可能性を秘めています。

* **高い技術力:** MBT55微生物群を用いた発酵技術は、有機廃棄物を24時間で完全に分解し、高品質な肥料と飼料を生産することができます。
* **低コスト:** 発酵機は、シンプルな構造で低コストで作ることができ、途上国でも導入しやすい。
* **環境負荷低減:** 廃棄物の燃焼や埋め立てによる二酸化炭素排出量を削減し、土壌の炭素固定量を増加させます。
* **食料安全保障の強化:** 機能性堆肥は、農作物の収穫量と品質を向上させ、肥料使用量を削減することができます。
* **気候変動対策:** 温室効果ガス排出量削減に貢献します。

### 5. 世界銀行と AGRIX Project の連携

世界銀行と AGRIX Project は、以下の点において、共通の目標を持ち、互いに補完関係にあります。

* **共通の目標:** 食料安全保障の強化と気候変動対策
* **世界銀行の強み:** 資金力、政策・制度の整備力
* **AGRIX Project の強み:** 技術力、低コスト

### 6. 提案

世界銀行と AGRIX Project が連携することで、以下の効果が期待できます。

* **技術普及の加速:** 世界銀行の資金力と政策・制度の整備力により、AGRIX Project の技術を途上国の農家に迅速に普及させることができます。
* **資金調達の円滑化:** 世界銀行が AGRIX Project に資金援助を行うことで、プロジェクトの推進に必要な資金を確保することができます。
* **政策・制度の改善:** 世界銀行と AGRIX Project が協力して、持続可能な農業を推進するための政策・制度の整備を支援することができます。

### 7. 結論

AGRIX Project は、世界銀行の食料安全保障と気候変動対策の取り組みを大きく加速させる可能性を秘めています。両者が連携することで、より多くの農家が恩恵を受け、持続可能な農業の実現に向けて大きな一歩を踏み出すことができると確信しています。

### 今後の具体的な連携方法

* **技術普及のためのパイロットプロジェクトの実施:** 西アフリカ地域など、肥料不足や土壌劣化が深刻な地域で、AGRIX Project の技術を導入したパイロットプロジェクトを実施します。
* **政策・制度の改善に向けた共同研究:** 持続可能な農業を推進するための政策・制度の改善に向けた共同研究を実施します。
* **資金調達のための共同キャンペーン:** AGRIX Project の普及に向けた資金調達のための共同キャンペーンを実施します。

### 期待される効果

* **食料安全保障の強化:** 農作物の収穫量と品質の向上、肥料使用量の削減により、食料安全保障の強化に貢献します。
* **気候変動対策:** 温室効果ガス排出量削減、土壌の炭素固定量増加により、気候変動対策に貢献します。
* **農家の所得向上:** 農作物の収穫量と品質の向上、肥料コストの削減により、農家の所得向上に貢献します。

### 提案のまとめ

世界銀行と AGRIX Project の連携は、食料安全保障と気候変動対策という共通の目標達成に向けた、非常に効果的な手段であると考えられます。貴殿のご検討をよろしくお願いいたします。
