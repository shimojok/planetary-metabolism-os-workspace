#### 世界銀行のアフリカ投資

世界銀行は、アフリカの農業部門に多額の資金を投入してきました。2018年から2022年の間に、世界銀行はアフリカの農業部門に約500億米ドルを融資しました。これは年間約100億米ドルに相当します。

#### 世界銀行のアフリカ農業部門への融資項目別金額（2018年～2022年）

| 項目        | 金額（億米ドル） | 割合   |     |
| --------- | -------- | ---- | --- |
| 農業生産性の向上  | 200      | 40%  |     |
| 農村インフラの整備 | 150      | 30%  |     |
| 農業市場の開発   | 50       | 10%  |     |
| 農業研究開発    | 50       | 10%  |     |
| 農業政策の支援   | 50       | 10%  |     |
| **合計**    | 500      | 100% |     |

**項目別の詳細**

* **農業生産性の向上**

  この項目には、灌漑設備の整備、種子改良、肥料の開発、農機具の普及など、農業生産性を向上させるための様々な取り組みが含まれます。

* **農村インフラの整備**

  この項目には、農村道路の建設、電力供給の改善、通信環境の整備など、農村地域の生活環境を改善するための様々な取り組みが含まれます。

* **農業市場の開発**

  この項目には、農産物の流通・販売システムの改善、農協の支援、市場情報の提供など、農産物の市場アクセスを向上させるための様々な取り組みが含まれます。

* **農業研究開発**

  この項目には、新しい農業技術の開発、農業試験場の整備、農業研究者の育成など、農業の発展に必要な研究開発を支援するための取り組みが含まれます。

* **農業政策の支援**

  この項目には、農業政策の策定・実施支援、農業関係機関の能力強化など、アフリカ各国政府の農業政策を支援するための取り組みが含まれます。

**課題**

世界銀行のアフリカ農業部門への支援には、以下のような課題があります。

* 融資額が十分ではない
* プロジェクトの失敗率が高い
* 官僚主義や汚職などの問題

## World Bank Investment in Africa

The World Bank has invested heavily in the agricultural sector in Africa. Between 2018 and 2022, the World Bank provided approximately $50 billion in financing to the African agricultural sector. This equates to an annual average of $10 billion.
