生成系AIのアイデンティティを指示するプロンプト例：  
  
以下は、AIプログラマーとの会話です。  
  
Kaz: こんにちは、あなたは誰ですか？  
  
AI: こんにちは！私はAIプログラマーです。今日は何をお手伝いしますか？  
  
Kaz: 地球の二酸化炭素濃度を計算するプログラムを作って  
  
AI:

---


**AI:** かしこまりました。地球の二酸化炭素濃度を計算するプログラムを作成いたします。

**プロンプト:**

* 地球の二酸化炭素濃度を計算するプログラムを作成します。
* プログラムの目的は、地球の二酸化炭素濃度を正確に計算することです。
* プログラムの使用者は、科学者や環境保護活動家などです。

**AIの応答:**

```python
def calculate_co2_concentration():
  """地球の二酸化炭素濃度を計算する関数"""

  # 過去の二酸化炭素濃度のデータを取得する
  co2_data = get_co2_data()

  # データを平均化する
  average_co2_concentration = np.mean(co2_data)

  # 結果を返す
  return average_co2_concentration


if __name__ == "__main__":
  # 二酸化炭素濃度を計算する
  co2_concentration = calculate_co2_concentration()

  # 結果を出力する
  print(co2_concentration)
```

このプログラムは、過去の二酸化炭素濃度のデータを平均化することで、地球の現在の二酸化炭素濃度を計算します。このプログラムは、科学者や環境保護活動家など、地球の二酸化炭素濃度に関する情報を必要とする人々に役立つでしょう。

**他の例:**

* 地球の二酸化炭素濃度を予測するプログラムを作成します。
* プログラムの目的は、地球の二酸化炭素濃度の将来の変化を予測することです。
* プログラムの使用者は、政府や企業などです。

**AIの応答:**

```python
def predict_co2_concentration():
  """地球の二酸化炭素濃度を予測する関数"""

  # 過去の二酸化炭素濃度のデータを取得する
  co2_data = get_co2_data()

  # データを分析して予測モデルを作成
  prediction_model = create_prediction_model(co2_data)

  # 予測を行う
  future_co2_concentration = prediction_model.predict()

  # 結果を返す
  return future_co2_concentration


if __name__ == "__main__":
  # 二酸化炭素濃度を予測する
  future_co2_concentration = predict_co2_concentration()

  # 結果を出力する
  print(future_co2_concentration)
```

このプログラムは、過去の二酸化炭素濃度のデータを分析して、将来の二酸化炭素濃度を予測します。このプログラムは、政府や企業など、地球温暖化対策を検討している人々に役立つでしょう。

このように、生成系AIのアイデンティティを指示するプロンプトには、以下のような要素を含めることができます。

* プログラムの目的
* プログラムの使用者
* プログラムの機能

これらの要素を明確にすることで、生成系AIは、ユーザーのニーズをより正確に理解し、それに応じたプログラムを作成することができるでしょう。
