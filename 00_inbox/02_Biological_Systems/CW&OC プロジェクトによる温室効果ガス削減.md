#news #2025-10-14 

提供いただいた「CW&OC Initiative_ja.pptx」に基づき、このプロジェクト（衣料廃棄物再利用とオーガニックコットン活用によるMBT Sustainable Cycle）の**年間温室効果ガス（GHG）削減量の推計値**を以下のように算出します。

---

## 📌 前提条件（スライドからの要点）

1. **世界の衣料廃棄物：約1億トン／年**（全体規模）
2. **日本の衣料廃棄物：約50万トン／年**（焼却・埋立：90%以上）
3. **衣服のライフサイクル（製造〜廃棄）でのCO₂排出：9,500万トン／年（日本国内）**

したがって：

* **50万トンの衣料廃棄物 ≒ 9,500万トンのCO₂排出**に寄与
* つまり、**1トンの衣料廃棄物 ≒ 190トンのCO₂排出に相当**（概算）

この推計が正しいとすれば、以下のように計算できます。

---

## 🧮 年間GHG削減量の推計（最大理論値）

仮に本プロジェクトが：

* **年間100万トンの衣料廃棄物をMBT資材化して再利用（焼却を回避）**できれば、
* 回避されるCO₂排出量は：

### ✅ 計算式：

```
GHG削減量 = 衣料廃棄物量 × CO₂排出係数
           = 1,000,000トン × 190トン-CO₂／トン-廃棄物
           = 1億9,000万トン-CO₂／年
```

---

## 🎯 現実的シナリオ（初期フェーズ）

仮に、プロジェクト初期に以下の規模で展開できるとすると：

* **初年度対象廃棄物量：10万トン**
* GHG削減量：

```
10万トン × 190トン-CO₂／トン = 1,900万トン-CO₂／年
```

これは、**ナイロビMSCモデル7〜8都市分に相当**するインパクトとなります。

---

## ✅ 結論：年次温室効果ガス削減量（概算）

| プロジェクト規模   | 衣料廃棄物量 | 削減CO₂量（年）          | 備考          |
| ---------- | ------ | ------------------ | ----------- |
| 小規模（10万t）  | 10万トン  | **1,900万トン-CO₂**   | 初年度想定       |
| 中規模（50万t）  | 50万トン  | **9,500万トン-CO₂**   | 日本全体相当      |
| 大規模（100万t） | 100万トン | **1億9,000万トン-CO₂** | アジア・アフリカ展開型 |

## ✅ 廃棄物処理領域における削減ポテンシャルの全体像

| 廃棄物の種類   | 年間発生量（概算）   | CO₂排出係数             | 削減機会         | MSCによる代替手段     |
| -------- | ----------- | ------------------- | ------------ | -------------- |
| 食品廃棄物    | 約13億トン【FAO】 | 1トンあたり1.9〜2.5t-CO₂e | 約20〜30億t-CO₂ | MBT堆肥化・微生物分解   |
| 農作物残渣    | 数十億トン規模     | 地域によって異なる           | 数億t-CO₂      | 発酵→堆肥→腐植化      |
| 衣料廃棄物    | 世界で約1億トン    | 約190t-CO₂／t（加工含む）   | 最大19億t-CO₂   | 建材化・ボード化       |
| 廃棄木材・紙   | 数億トン規模      | 0.5〜1.2t-CO₂／t      | 数億t-CO₂      | MBT再発酵→土壌改良材   |
| 下水汚泥・生ゴミ | 数億トン規模      | メタン換算で極めて高い         | 高インパクト       | 発酵・固定化（バイオ炭含む） |

---

## 🔁 通常処理 vs MSC処理の違い

| 処理方法               | 処理後         | 炭素動態        | 温室効果               |
| ------------------ | ----------- | ----------- | ------------------ |
| 焼却・埋立              | CO₂またはCH₄排出 | 完全放出        | 非常に高い              |
| コンポスト化             | 一部放出＋栄養還元   | 一部固定        | 中程度                |
| **MSC（MBT発酵＋腐植化）** | 土壌中に炭素固定    | **高い長期安定性** | **非常に低い（マイナスに寄与）** |

## 🌍 インパクトの核心

* **「廃棄物＝負債」から「廃棄物＝資産・炭素貯留源」へのパラダイム転換**。
* つまり、**MSCは“温室効果ガスを出さない技術”ではなく、“削減して取り込む技術”**です。
