🌳 樹皮・幹の主要な問題

樹皮は、病害虫や環境ストレスから木を守る重要な器官ですが、以下の問題が発生します。

問題の種類 原因・病原体 症状・影響
甲虫による穿孔（コーヒーボーラー） 昆虫（Xylosandrus morigerus 等） 幹や主枝に小さな穴を開けて侵入し、内部を食害。木の導管を破壊し、枝枯れや全体の衰退を引き起こす。ストレスを受けた木が特に被害を受けやすい。
樹皮のキャンカー（潰瘍病） 真菌（Fusarium spp. 等） 樹皮が陥没・割れ、患部から樹脂（ゴム）が滲み出す。患部の上部が枯死し、生産量が激減する。剪定などの傷口から感染する。
樹皮の剥離と衰退 生理的障害（栄養不良、干ばつ、過湿、根腐れ） 樹皮がひび割れたり、剥がれやすくなったりする。これは病気の結果ではなく、木が全体的なストレス状態にあるサイン。根本的な環境改善が必要。

⏳ コーヒーの木の寿命と近年の傾向

コーヒーの木の寿命は栽培方法や環境によって大きく左右されます。

· 従来の寿命（理想的条件下）：
  · 適切に管理された伝統的な農法（日陰栽培など）では、経済生産年限は20～30年、樹齢自体は50年以上生きることもあります。
  · 高品質を重視する産地では、樹齢が上がるにつれて収量は落ちるものの、複雑な風味が増すとされ、古木（オールドツリー）として珍重されることもあります。
· 近年の短命化傾向とその要因：
  1. 集約的・太陽光栽培の増加：直射日光下で密植して高収量を追求する現代農法では、木に大きなストレスがかかり、経済寿命が15～20年程度に短縮されています。
  2. 気候変動のストレス：干ばつ、豪雨、異常高温などが木の体力を奪い、病害虫への抵抗力を低下させ、寿命を縮める主要因となっています。
  3. 病害（特にさび病）の影響：深刻な病害に繰り返し感染すると、木は回復できずに早期に衰退・枯死します。
  4. 土壌疲労と栄養管理不足：連作による地力低下や適切な施肥管理が行われない場合、木は衰弱し寿命が短くなります。

💡 持続可能性への取り組み

これらの課題に対処するため、以下のような持続可能な生産管理が求められています。

· 総合的有害生物管理（IPM）：農薬に依存せず、耐病性品種、生物的防除、適切な剪定・施肥を組み合わせた管理。
· 気候変動適応策：日陰樹の導入、保水性向上、耐乾性品種への更新。
· 土壌健康管理：被覆作物、有機物投入、適正施肥による土壌生態系の改善。
· 農家の経済的支援：苗木更新や農法転換に必要な資金・技術支援。

結論として、コーヒーの木は複数の生物的・環境的ストレスに直面し、その寿命は生産の在り方と気候変動に大きく影響されています。持続可能なコーヒー産業の未来は、単なる病害防除ではなく、木そのものの健康とレジリエンス（回復力）を高める総合的な農地管理にかかっています。

### **1. コーヒー炭疽病(Coffee Anthracnose) — _Colletotrichum_ spp.**

#### **A. 病害の生態学的特性**

**病原体**: _Colletotrichum coffeanum_, _C. gloeosporioides_(半活物寄生菌)  
**感染様式**:

```
付着器形成 → クチクラ穿孔 → 細胞間隙侵入 
→ 細胞壁分解酵素(ペクチナーゼ・セルラーゼ)分泌 
→ 壊死斑形成
```

**環境依存性**:

- 分生子発芽最適条件: 25-28℃、相対湿度>95%、葉面水膜6-12時間
- 多湿期の降雨飛沫で伝播→果実・葉・枝の傷口から侵入

**経済的被害**:

- 果実感染 → 「黒熟病」:落果率30-50%、品質低下(カップスコア-10点)
- 枝病変 → 翌年の結果母枝枯死:収量20-30%減

### **2. コーヒーピンク病(Pink Disease) — _Erythricium salmonicolor_**

#### **A. 病害の特異性と脅威**


- 初期症状(桃色菌糸)は目立つが、発見時には既に内部組織が壊死している場合が多い
