# 共通フレーム（まず全体像）

ほぼすべての分野は、次のレイヤー構造を持ちます。

```
生産者
  ↓
一次加工（選別・乾燥・搾油・醸造など）
  ↓
二次加工・ブランド化
  ↓
流通（卸・輸出・小売）
  ↓
消費者
```

病害虫・病気の影響は  
👉 **上流ほど「数量」**  
👉 **下流ほど「品質・価格・ブランド」**  
に転化します。

---

# 🐝 養蜂

### プレイヤー構造

```
養蜂家
  ↓
蜂蜜一次処理（採蜜・濾過）
  ↓
加工業者（瓶詰・ブレンド・機能性加工）
  ↓
流通・小売
```

### 特殊な関係（超重要）

```
養蜂家 ──→ 受粉サービス ──→ 農作物生産者
```

### 被害の波及

- 蜂群崩壊  
    → 蜂蜜減少（直接損失）  
    → 受粉不足（**第三者被害＝外部不経済**）
    

👉 **生産者以外に最大損失が出る唯一の分野**

---

# 🍄 シイタケ栽培

### プレイヤー構造

```
原木・菌床生産者
  ↓
シイタケ生産者
  ↓
選別・乾燥（一次加工）
  ↓
加工食品・業務用（二次加工）
  ↓
卸・小売
```

### 被害の波及

- カビ・害虫  
    → 規格外・廃棄（一次で止まる）  
    → 下流は「供給不安定」「価格変動」
    

👉 **被害は上流集中型**

---

# ☕ コーヒー

### プレイヤー構造（典型）

```
小規模農家
  ↓
精製所（ウォッシュド／ナチュラル）
  ↓
輸出業者
  ↓
焙煎業者（二次加工）
  ↓
ブランド・小売
```

### 被害の波及

- 病害虫  
    → 収量減（農家）  
    → 欠点豆増加（精製）  
    → 焙煎ロス・ブランド毀損（下流）
    

👉 **全レイヤーに累積被害**

---

# 🌿 オリーブ

### プレイヤー構造

```
オリーブ農家
  ↓
搾油所（一次加工）
  ↓
ブレンド・熟成（二次加工）
  ↓
輸出・小売
```

### 被害の波及

- 果実病害  
    → 収量減（農家）  
    → 酸度上昇（搾油）  
    → 格下げ・ブランド価値低下（下流）
    

👉 **品質規格が価格を支配**

---

# 🍇 ワイン用ブドウ

### プレイヤー構造（高付加価値）

```
ブドウ栽培者
  ↓
ワイナリー（一次＋二次加工）
  ↓
熟成・ブランド管理
  ↓
流通・輸出
```

### 被害の波及

- 病害  
    → 糖度・酸度変化  
    → 醸造設計変更  
    → **ヴィンテージ価値低下**
    

👉 **1年の病害が10年以上の価値に影響**

# 分野別：被害の「集中点」

|分野|被害集中点|特徴|
|---|---|---|
|養蜂|社会全体|外部不経済最大|
|シイタケ|生産者|上流完結|
|コーヒー|全層|累積型|
|オリーブ|加工・規格|品質主導|
|ワイン|ブランド|長期影響|

# 数理モデルへの接続（重要）

この関係性は、被害額モデルで次のように使います。

[  
D = \sum_{k} w_k D_k  
]

- (k)：プレイヤー層
    
- (w_k)：価値分配率・影響係数
    

👉 **誰が損をし、誰がリスクを負担していないか**が明確になります。
