```

#### **2. Metabolic モジュール（栄養カスケード）**
```
高分子基質 ───┐
             ▼
[一次分解] → 中分子中間体
             │
             ▼
[二次転換] → 低分子基質
             │
             ▼
[三次合成] → 高機能産物

      +---------------+
             │
┌──────┴──────┐
▼                  ▼
[Microbial層]    [Metabolic層]
```
