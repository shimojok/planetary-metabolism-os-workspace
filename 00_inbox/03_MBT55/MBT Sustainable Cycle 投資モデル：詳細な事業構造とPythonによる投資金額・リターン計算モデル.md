BioValleyでは、MBT Sustainable Cycleシステムを導入し、食品廃棄物(家畜の排泄物も含む)の堆肥化と飼料化により、農業生産とMBT機能性堆肥・飼料の販売、廃棄物処理コスト削減により、総合的に利益を創出します。例えば、ナイロビの人口440万人をモデルにし試算すると、1,400億円から最大3,500億円の経済効果(廃棄物処理コスト削減を含む)があります。

  

食料問題や食料安全保障、栄養改善が必要であり、劣化農地の修復を急ぎ生産性を向上させる必要があるアフリカから手始めに、BioValley を展開する計画です。これは気候変動対策としても重要な案件です。

## MBT Sustainable Cycle 投資モデル：詳細な事業構造とPythonによる投資金額・リターン計算モデル

**2.2 ファンド**

* ファンドは、投資家から集めた資金を運用し、以下の事業への投資を行います。

    * 炭素クレジットの管理

**3. 投資金額・リターン計算モデル（Python）**

以下のPythonコードは、MBT Sustainable Cycle 投資モデルにおける投資金額とリターンの計算例を示しています。

```python
# 投資金額
investment_amount = 1000000000  # 円

# 投資期間
investment_period = 10  # 年

# サブスクリプションフィー
subscription_fee = 1000000  # 円/BioValley/年

# AGRIX Platform サービス利用料
agrix_platform_fee = 500000  # 円/BioValley/年

# 炭素クレジット収益
carbon_credit_revenue = 100000  # 円/BioValley/年

# 想定BioValley数
estimated_biovalley_count = 100

# 想定農業生産法人数
estimated_farm_count = 1000

# 想定地方政府数
estimated_local_government_count = 10

# 投資利益率（IRR）
irr = 0.1

# 投資収益の現在価値
discounted_investment_return = 0

# 年間キャッシュフロー
annual_cash_flow = 0

# 投資期間の各年のキャッシュフロー計算
for year in range(investment_period):
  # サブスクリプションフィー収入
  subscription_fee_revenue = estimated_biovalley_count * subscription_fee

  # AGRIX Platform サービス利用料収入
  agrix_platform_fee_revenue = estimated_biovalley_count * agrix_platform_fee

  # 炭素クレジット収益
carbon_credit_revenue = estimated_biovalley_count * carbon_credit_revenue

  # 年間キャッシュフロー
  annual_cash_flow = subscription_fee_revenue + agrix_platform_fee_revenue + carbon_credit_revenue

  # 投資収益の現在価値計算
  discounted_investment_return += annual_cash_flow / ((1 + irr) ** year)

# 投資利益率（IRR）計算
irr = np.irr(np.array([investment_amount * -1] + [annual_cash_flow for _ in range(investment_period)]))

# 結果出力
print("投資金額:", investment_amount, "円")
print("投資期間:", investment_period, "年")
print("想定BioValley数:", estimated_biovalley_count)
print("想定農業生産法人数:", estimated_farm_count)
print("想定地方政府数:", estimated_local_government_count)
print("投資利益率（IRR）:", irr)
print("投資収益の現在価値:", discounted_investment_return, "円")
```
