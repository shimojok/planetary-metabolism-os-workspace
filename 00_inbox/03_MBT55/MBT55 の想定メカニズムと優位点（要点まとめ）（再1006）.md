# 4) Green Premium（簡易算出例：方法と数値）

以下は「Green Premium を定量的に議論するための**簡易モデル**」です。重要な点：**前提を明示**すれば、誰が見ても比較可能になります。ここでは代表的な要素（化学肥料の製造排出削減、土壌固定量、炭素価格）を使った例を示します（保守的な前提で計算）。

**前提（例）**

1. 合成窒素系肥料の生産に伴う CO2e：**2.6 tCO2e / トン肥料**（生産段階の代表値）を使用。([carbonchain.com](https://www.carbonchain.com/blog/understand-your-synthetic-fertilizer-emissions?utm_source=chatgpt.com "Understanding fertilizer emissions for carbon regulation"))
    
2. MBT 製品導入により、化学肥料使用が **0.5 トン/ha/年** 減る（例示）。
    
3. MBT による土壌の有機炭素増（腐植等）での**実効的な実測相当の CO2e 固定量を 1.0 tCO2e/ha/年**と仮定（現実は地域・施用量で幅あり）。([Frontiers](https://www.frontiersin.org/journals/sustainable-food-systems/articles/10.3389/fsufs.2023.1234108/full?utm_source=chatgpt.com "Quantifying soil carbon sequestration from regenerative ..."))
    
4. 炭素価格（カーボン価値）は便宜的に **$50 / tCO2e** を使用（感度シナリオで $20–$100 なども考慮）。
    

**算出（保守例）**

- （A）化学肥料生産由来の削減価値（炭素換算）
    
    - 肥料削減 0.5 t/ha × 2.6 tCO2e/ton = 1.3 tCO2e/ha/年
        
    - これの経済価値 = 1.3 × $50 = **$65 / ha / 年**
        
- （B）土壌固定（腐植増加）からの価値
    
    - 固定 1.0 tCO2e/ha/年 × $50 = **$50 / ha / 年**
        
- （C）合計（A + B）＝**$115 / ha / 年**（この例では、1 haあたりの CO2e 価値として）
    

（上の数値は保守的な想定：肥料生産排出の範囲や土壌固定量の範囲次第で上下します。たとえば土壌固定が 0.5–2.5 tCO2e/ha/年であれば、腐植価値は $25–$125/ha/年 に拡がります。([Frontiers](https://www.frontiersin.org/journals/sustainable-food-systems/articles/10.3389/fsufs.2023.1234108/full?utm_source=chatgpt.com "Quantifying soil carbon sequestration from regenerative ..."))）

**一方で「Green Premium＝消費者が追加で支払っても良い価格差」を考える場合の指標：**

- MBT 機能性飼料による「死亡率低下・生産性向上・品質向上」の直接的経済効果を合算して、これが製造コスト差や市場価格差（＝プレミアム）を正当化できるかを示す必要があります。
    
- 例：ある養鶏場で MBT 飼料により年間卵生産が 2% 向上、廃棄率が 1% 減少、鶏1羽当たりの出荷価値増が $0.5 とすると、面積／頭数あたりの増収と上の炭素価値を合算して、**消費者向けのプレミアム（例：1割〜3割の価格上乗せ）**が十分説明可能か検討します。現場データで示すと説得力が増します。
    

**短い数値サマリ（前提に基づく試算）**

- 肥料 1 t 回避 → CO2e 回避価値 ≒ **$130 / t**（＝ 2.6 tCO2e × $50）。([carbonchain.com](https://www.carbonchain.com/blog/understand-your-synthetic-fertilizer-emissions?utm_source=chatgpt.com "Understanding fertilizer emissions for carbon regulation"))
    
- 土壌腐植（0.5–2.5 tCO2e/ha/年）→ カーボン価値 **$25–$125 / ha / 年**。([Frontiers](https://www.frontiersin.org/journals/sustainable-food-systems/articles/10.3389/fsufs.2023.1234108/full?utm_source=chatgpt.com "Quantifying soil carbon sequestration from regenerative ..."))
    
- 肥料削減 0.5 t/ha ＋ 腐植 1.0 t/ha の組合せ（上の保守例）→ **$115 / ha / 年**（炭素価値ベース）。
    

> 注意：上の数値は「炭素価値に換算した Green Premium の一部分」を示すに過ぎません。実際の商用プレミアムは、品質向上分（市場の受容度）、死亡率低下に伴う補償回避、保険料低下、サプライチェーン上のリスクプレミアム低下なども含めて算出すべきです。
