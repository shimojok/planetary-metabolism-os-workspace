MBT55微生物群について

MBT55(NB菌)微生物群は、でんぷん分解菌、タンパク質分解菌、脂質分解菌、セルロース分解菌の４分野と好気性微生物55%、嫌気性微生物45%のバランスが、土壌中のあらゆる有機物、あらゆる食品残渣、家畜の排泄物を発酵・分解が可能なことです。
分類は下記の通り
好気性菌　（酵母菌、セルロース分解菌、窒素固定菌）
嫌気性菌　（セルロース分解菌、窒素固定菌）
乳酸菌群
糸状菌（芳香族化合物分解菌）
放線菌（キチン分解菌）
マンガン還元菌（黒カビ族群ー原生担子菌類）
マンガン酸化菌（有機栄養菌）
アンモニア酸化菌（亜硝酸菌）
硫黄細菌（硫化水素を水素供与体として利用する細菌群、緑色硫黄細菌科類）
硫酸菌
セルロース放線菌
鉄酸化菌
硝化生成菌
セルロース糸状菌
鉄還元菌

硫酸還元菌

## 🔎 1. MBT55微生物群における窒素固定菌

ご提示の分類から：

- **好気性菌群**：窒素固定菌（例：_Azotobacter_, 一部の放線菌 _Frankia_）
- **嫌気性菌群**：窒素固定菌（例：_Clostridium_, 光合成細菌の一部 _Rhodospirillum_, 緑色硫黄細菌など）
    
つまり、**好気性・嫌気性の両方に窒素固定菌が含まれる**構成になっています。

## 🔎 2. 好気性と嫌気性の窒素固定菌

### 好気性窒素固定菌

- **代表例**: _Azotobacter_, _Frankia_（放線菌）
- **環境**: 酸素存在下でも、特殊な呼吸活性や保護タンパク質によりニトロゲナーゼを酸素から守る
- **役割**: 土壌中で自由生活し、有機物分解と同時に固定窒素を供給
    
### 嫌気性窒素固定菌

- **代表例**: _Clostridium_ 属、嫌気性光合成細菌 (_Rhodospirillum_, _Chromatium_)
- **環境**: 酸素がない環境（嫌気的堆積層、発酵槽、嫌気性ポケット）
- **役割**: 発酵や硫黄代謝と連動しながら窒素固定を行う
    
## 🔎 3. 窒素固定のメカニズム

窒素固定は、**窒素固定酵素（ニトロゲナーゼ複合体）**によって行われます。

[  
\text{N}_2 + 8\text{H}^+ + 8e^- + 16 \text{ATP} ; \rightarrow ; 2\text{NH}_3 + \text{H}_2  
]

- **基質**: 大気中の窒素分子（N₂）
- **産物**: アンモニア（NH₃）、副産物として水素（H₂）
- **エネルギー要求**: 多量のATP（16分子以上）
- **電子供与体**: 有機酸、糖、H₂ など、発酵や分解で生じる代謝産物が使われる
    
## 🔎 4. 窒素固定菌が「作る窒素代謝物」

- **アンモニア（NH₃）**：水中ではアンモニウムイオン（NH₄⁺）として存在し、微生物や植物に直接利用される
- **アミノ酸**：窒素固定菌は固定窒素を自己のアミノ酸合成に利用する
- **間接的代謝物**：アンモニアが硝化菌によって亜硝酸（NO₂⁻）、硝酸（NO₃⁻）に変換される
    
## 🔎 5. 植物や微生物間での「立ち位置」と目的

- **窒素固定菌の目的**: 自らの生存に必要な窒素源を確保することが第一。ただし、その副産物が周囲の生態系を支える。
    
- **植物にとっての立ち位置**:
    
    - 直接 → 根圏でアンモニアやアミノ酸が植物に供給される
    - 間接 → 硝化菌・脱窒菌とのネットワークで「窒素循環」が完成する
        
- **微生物間の立ち位置**:
    
    - 有機物分解菌（セルロース分解菌、タンパク分解菌）から電子供与体や有機酸を受け取る
    - その代謝物としてアンモニアを出し、他の微生物や植物が利用
    - つまり「窒素の入口」となる一次供給者
        
## 🔎 6. まとめ：MBT55における窒素固定菌の役割

1. **大気窒素（N₂）を生物が利用可能な形（NH₃/NH₄⁺）に変換**する
2. **発酵・分解で生じた有機酸や糖を電子源として利用**し、窒素固定を進める
3. **微生物群集内で窒素をシェア**し、植物成長や他の微生物活動を支える
4. **好気性・嫌気性の両方の窒素固定菌が存在**することで、酸素濃度の異なる環境でも窒素固定が継続する
