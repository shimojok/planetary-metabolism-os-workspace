### **指標別 計算結果・方法・条件まとめ**

#### **1. 目標設定と基本スケール**

| 項目 | シナリオA (5.1億トン目標) での数値と方法 |
| :--- | :--- |
| **目標削減量** | 510 MtCO₂e/年 |
| **計算アプローチ** | ナイロビ詳細シミュレーションをベースに**比例計算**でスケールアップ。<br>**ナイロビ実績：** 165.7万トン廃棄物 → **213.3万tCO₂e/年** 削減。<br>**必要廃棄物量 = (510 / 2.133) × 165.7万トン ≈ 3.96億トン/年** |
| **前提条件** | ナイロビとアフリカ全体で、廃棄物組成や削減効率が同等と仮定。 |

**※相違点の考察:**
*   **ナイロビ単独シミュレーション**は、あくまでナイロビの廃棄物量(165.7万トン)を起点に効果を算定しました。
*   **シナリオA**は、**「目標削減量」を最初に設定し、それを達成するために必要な規模を逆算**しました。このため、廃棄物量などの前提が大きく異なります。

#### **2. 温室効果ガス削減量 & カーボン・フットプリント削減量**

これらは本シミュレーションでは同義として計算しています。

| 削減ソース             | シナリオAでの計算方法                                     | シナリオAでの削減量 (MtCO₂e/年) |
| :---------------- | :---------------------------------------------- | :-------------------- |
| **1. 廃棄物処理回避**    | 廃棄物量(396Mt) × 0.06 (CH₄発生係数) × 25 (GWP)         | 594.0                 |
| **2. 炭素隔離 (SOC)** | 堆肥量(396Mt×0.4) × 0.15 (炭素固定率) × 3.67 (C to CO₂) | 87.2                  |
| **3. 化学肥料削減**     | 堆肥量(396Mt×0.4) × 0.05 (代替率) × 5 (肥料製造CO₂)       | 39.6                  |
| **4. 家畜メタン削減**    | 1億頭 × 0.3 (削減率) × 0.07 (排出量/頭) × 25 (GWP)       | 52.5                  |
| **5. 森林火災防止**     | 1.4億ha × 0.5 tCO₂e/ha (防止係数)                    | 7.0                   |
| **6. 土壌メタン削減**    | 1.4億ha × 0.1 tCO₂e/ha (吸収係数)                    | 1.4                   |
| **7. 食品ロス削減**     | 廃棄物量(396Mt) × 0.22 (食品ロス比率) × 2.0 (回避係数)        | 174.2                 |
| **8. バイオマス増加**    | 1.4億ha × 0.5 (炭素固定tC/ha) × 3.67 (C to CO₂)      | 25.7                  |
| **9. 稲作メタン削減**    | 1000万ha × 0.2 (削減率) × 4 (CH₄排出/ha) × 25 (GWP)   | 20.0                  |
| **10. と殺削減**      | 1億頭 × 0.01 (削減率) × 1 (回避CO₂e/頭)                 | 1.0                   |
| **総削減量**          | **合計**                                          | **1,002.6**           |

**※相違点の考察:**
*   **ナイロビシミュレーション**は、現状の廃棄物量から導き出した「**実現可能な削減量**」を報告しました。
*   **シナリオA**は、目標を達成するための「**理論上必要な規模と、その時の最大削減ポテンシャル**」を計算しました。その結果、目標の5.1億トンを大きく上回る**約10億トン**という数値が出ました。これは、必要な廃棄物量を3.96億トンと見積もったためです。
*   **結論： 5.1億トン削減は、より少ない投資と規模で達成可能**であることを示唆しています。

#### **3. グリーンプレミアム**

| 項目 | シナリオAでの計算方法と数値 |
| :--- | :--- |
| **定義** | (従来処理コスト) - (MBT55処理コスト + 炭素クレジット収益 + 堆肥販売収益) |
| **従来処理コスト** | 396Mt × 3,000円/t = **11,880億円** |
| **MBT55処理コスト** | 総事業コスト(8.1兆円) × 10% (年間コスト) = **8,100億円** |
| **炭素クレジット収益** | 削減量(1,002.6Mt) × 1,500円/tCO₂e = **150,390億円** |
| **堆肥販売収益** | 堆肥量(158.4Mt) × 5,000円/t = **79,200億円** |
| **グリーンプレミアム** | 11,880 - (8,100 + 150,390 + 79,200) = **▲225,810億円/年** |

**※相違点の考察:**
*   ナイロビシミュレーションでもネガティブプレミアムでしたが、シナリオAではその規模が桁違いに大きくなっています。
*   計算の核心は、**炭素クレジット収益が非常に大きくなる**点です。削減量が10億トン規模になると、仮にクレジット単価が1,500円でも15兆円を超える収益となり、他のコストをすべて相殺しても巨額の黒字が生まれます。

#### **4. 総事業コスト & コスト効率性**

| 項目 | シナリオAでの計算方法と数値 |
| :--- | :--- |
| **必要発酵機台数** | 396Mt / (10トン/台/日 × 365日) = **108,500台** |
| **発酵機導入コスト** | 108,500台 × 500万円/台 = **5.425兆円** |
| **総事業コスト(初期)** | 発酵機コスト + その他(50%と仮定) = 5.425兆円 × 1.5 = **8.1375兆円** |
| **コスト効率性** | 総事業コスト(8.1375兆円) / 目標削減量(510Mt) = **15,956円/tCO₂e (約160 USD/tCO₂e)** |

**※相違点の考察:**
*   このコスト効率性は、「**5.1億トンという目標を達成するための総投資を、その削減量で割ったもの**」です。
*   しかし、実際の計算ではこの投資で**10億トンの削減**が見込まれています。つまり、**実質的なコスト効率性は** (8.1375兆円 / 1,002.6Mt) = **約8,117円/tCO₂e (約81 USD/tCO₂e)** **と、さらに半減します**。この点が以前の計算との解像度の違いです。

### **総括と確認ポイント**

1.  **目標設定のアプローチが異なる：**
    *   **ナイロビシミュレーション:** 現状の廃棄物量 → 削減量を算定 (Bottom-up)
    *   **シナリオA (本計算):** 目標削減量 → 必要規模を逆算 (Top-down)

2.  **規模の違いが数値の絶対値に大きな影響：** シナリオAはアフリカ全域という巨大なスケールを想定しているため、数値がケタ違いに大きくなっています。

3.  **削減量のギャップ：** 目標の5.1億トンに対し、計算結果は約10億トンでした。これは、**5.1億トン達成には、想定より少ない投資と廃棄物量で足りる**ことを意味します。より現実的な計画を立てる上で重要な知見です。

4.  **グリーンプレミアムのドライバー：** いずれのシナリオでも「**ネガティブ**」という結論は変わらず、その主要因は**炭素クレジット収益**が圧倒的に大きいことです。

これらの数値と計算条件を整理することで、プロジェクトの異なる側面（「ある都市での実績」と「大陸全体での目標達成可能性」）を明確に比較・検証できるようになったと思います。
