# 2) 科学的裏付けとメカニズム（技術別）

## Pivot Bio（報告書に基づく）

* **作用機序**：土壌由来の窒素固定能を強化した微生物を種子や列処理で供給し、作物根圏に定着して成長期にアンモニアを生産。結果として施肥量を置換し、肥料由来のN₂O発生や硝酸流出を削減する。
* **検証／実績**：大学・フィールド試験データ、N-OVATOR™プログラムによる検証結果（代替N量・CO₂e削減の登録）を保有。
* **利点**：導入が簡単（既存農法に組み込みやすい）、製造時のエネルギー・水使用が合成肥料より小さい（報告値：水利用は大幅に少ない）。

## MBT55（提示された特徴に基づく）

* **多層代謝ネットワーク**：セルラーゼ・リグニン分解酵素群、好熱酵素、乳酸菌群、硝化/脱窒連鎖、金属酸化還元代謝などが時相的・空間的に協働し、分解（高速度）と再固定（腐植化）を同時的に進行させる。
* **電子フロー再固定**：生成された還元性中間体（乳酸・酢酸・フェノール類）を微生物ネットワーク内で再利用し、CO₂の一部を有機Cへ“逆戻し”する反応経路を強化する（数式的には固定率 (C_f=\lambda\frac{E_d}{E_d+K_m}) の形で説明）。
* **腐植質生成と長期安定化**：リグニン分解→フェノール変換→フミン化過程、微生物由来ポリマー（菌体残渣含む）による安定化で中期〜長期（年〜百年）スケールのC固定が見込まれる。
* **多元素統合**：窒素はアミノ酸・アンモニウムとして、リンは溶解化して植物に、K・微量元素は鉱物相変換やキレート供給で可用化されるため、総合的な肥料効果が得られる。
* **汚染浄化**：糸状菌・放線菌・金属代謝菌が有害有機物や重金属の分解・不溶化を担い、汚泥・津波堆積物等の迅速肥料化が可能とする事例がある（ユーザーデータ）。

> 補足：MBT55側の数値（例：SOC +3.7%、根量5–8倍等）は本スレッドでの記述に基づく観察値／主張です。実証化のためには定量的フィールド比較が必要です。

# 5) 総合評価（長所・短所）

## MBT55 の長所（相対的）

* **多機能性**：単一栄養素だけでなく、腐植化・微量元素供給・汚染物質分解までを目標化できる点は大きな優位。
* **SOC直接増加**：炭素隔離を直接「土壌に貯める」ことが可能であれば、炭素クレジットの長期価値が高い。
* **廃棄物資源化との親和性**：現地資源（廃棄物）を原料にするため供給チェーンがローカル化できる。

## MBT55 の注意点（検証ポイント）

* **再現性とスケール**：24時間での完全分解・腐植化等は条件依存。土壌・気候・基質の多様性で効果が変動する可能性がある。
* **MRVの確立**：SOCや腐植の長期安定性を第三者が検証できるプロトコルが必要。
* **生態リスク**：微生物群の導入が現地生態系への副作用を及ぼさないか（特に非自生微生物採用時）は注意深く評価する必要がある。

## Pivot Bio の長所・注意点（短評）

* 長所：フィールドでの実証が進み、N-OVATOR等でクレジット化の道筋が既にある。導入が簡便で農家に受け入れられやすい。
* 注意点：窒素以外の課題（SOC、汚染物質、重金属）には直接対応しないため、システム的に見ると補完技術が必要。

# 2. なぜ MBT55 が「多機能」になり得るか（メカニズムの科学的論拠）

1. **コンソーシアム効果（機能分化）**：複数の機能微生物（セルラーゼ、リグニナーゼ、窒素固定、P溶解、金属代謝等）が時相的かつ空間的に協働することで、単一機能菌では不可能な「分解→中間代謝→再固定→安定化」の連鎖を形成する。
2. **相分離（LLPS）と酵素濃縮**：EPS 等による液-液相分離が起こると局所的酵素濃度が上がり反応速度が増すため、短時間での高度な変換が可能（理論的・観察的裏付けあり）。
3. **電子フロー再固定**：還元的中間体（乳酸、酢酸、フェノール類等）を利用してCO₂を還元的に有機Cへと固定する経路を微生物間ネットワークが支える。数式モデル ( C_f = \lambda \frac{E_d}{E_d+K_m} ) のように電子供与体効率と固定比率が結びつく理論枠組みで説明可能。
4. **金属・硫黄代謝による相変換**：Fe/Mn・硫黄代謝菌は重金属の可溶化／沈殿を制御できる。これにより汚染土壌の不溶化や栄養素の再分配が進む。

これらは生態学・微生物学の既存知見（微生物相互作用、酵素バイオケミストリー、金属微生物学）と整合しますが、**定量化と長期安定性（特に SOC の耐久性）を示すためのフィールドデータが必要**です。

# 3. 科学的検証（推奨 MRV・評価指標）

MBT55 の有効性を Pivot Bio と同等の信頼度で示すため、以下を必須推奨します。

**土壌 / 炭素関連**

* SOC総量（0–30 cm, 30–100 cm）年次測定。
* SOCフラクション（可溶性、有機物フラクション、フミン/フルボ酸、微生物バイオマス炭素）。
* 安定同位体（δ¹³C, ¹⁴C）で「新規固定C」の起源と寿命評価。

**栄養素 / 作物生産性**

* 可給性 N（NH₄⁺, NO₃⁻）、Olsen-P、可給性K、作物吸収量、収量、品質（蛋白質含量等）。
* 根系指標（根量、根の化学組成：スベリン量など）。

**GHG / 廃棄物ライフサイクル**

* 現場フラックス（CO₂, CH₄, N₂O）のリアルタイム測定（自動チャンバ、年積算）。
* 廃棄物→処理→土壌還元のLCA（焼却回避等のGHG回避量を含む）。

**汚染物質・重金属**

* ターゲット有機汚染物の残存と分解産物の測定（GC/MS, LC-MS）。
* 可溶性・不溶性重金属（ICP-MS）および沈殿相解析。

**微生物機能**

* メタゲノム（16S/ITS+機能遺伝子：nif, amoA, laccase, peroxidase 等）と遺伝子発現（mRNA）解析。
* 酵素活性アッセイ（セルラーゼ、ラッカーゼ等）。

これらのデータを第三者（大学・認証機関）で検証し、カーボンクレジット化の基礎データとすることが必須です。

# 4. 実証試験の設計（比較・併用検証案：要旨）

* **実験区**：慣行（Nフル）、Pivot Bio（推奨投与）、MBT55（例：1 t/ha 機能性堆肥）、Pivot＋MBT55、無処理。
* **多地点**：温帯（米国）、温暖湿潤（東南アジア）、乾燥/半乾燥（アフリカ）各 3 サイト。
* **期間**：3 年（SOC安定性は5年推奨）。
* **主要測定**：収量、SOC、GHGフラックス、可給性NPK、根量、土壌微生物メタゲノム、土壌物理（団粒、CEC）。
* **評価**：経済性（コスト/利益）、GHG回避量（IPCC基準併用）、炭素クレジット化可否検討。

# 5. バイオセーフティ／規制上の留意点

* **微生物導入の生態リスク評価**（非標的影響、水平遺伝子移動の可能性）を事前に行うこと。
* MBT55が「自然選抜由来」としても、導入先の現地生態系との相互作用評価と国別の法規制（放出／土壌改良材）に従う必要あり。
* ヒト・動物の健康についての主張（経口摂取で疾患が治る等）は臨床試験・獣医試験の根拠が必要。現時点での断定は避け、検証研究計画を掲示するのが適切。

# 6. 結論（提案的まとめ）

* **Pivot Bio** は「窒素問題」に対する非常に実証的で実装可能なソリューションであり、短期的な N₂O 削減・農家の費用低減に即効性がある。
* **MBT55** は **より広範な機能（N,P,K・微量元素・腐植化・汚染分解・金属無害化・バイオコントロール）** を同一プラットフォームで実現し得る可能性を持つ。これが確認されれば、**炭素隔離（SOC）の直接増加**と廃棄物資源化という面で大きな優位性がある。
* ただし MBT55 の「多機能性」については、**第三者検証済みの長期データ（SOCの耐久性、汚染物質の完全分解・無害化、病害防御の汎用性）**がまだ十分に公開されているわけではないため、**比較優位を確定するには対照試験と MRV の整備が必須**である。
