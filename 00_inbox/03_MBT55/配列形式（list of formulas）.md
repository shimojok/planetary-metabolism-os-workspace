```json
  ...

```

これが最も安定します。

---

### 🔷 なぜオブジェクト形式は不利か

```json
{
  "F001": {...},
  "F002": {...}
}
```

この形式だと：

- 並び順保証が曖昧
- キー抽出処理が増える
- 将来の外部API統合時に整合が崩れる可能性
    
### 🔷 現在のJSONデータ

現在は：

```json
{
  "mbt_kampo_library": {
      "version": "...",
      ...
  }
}
```

となっているため、  **実データがネストされている可能性が高い**です。
