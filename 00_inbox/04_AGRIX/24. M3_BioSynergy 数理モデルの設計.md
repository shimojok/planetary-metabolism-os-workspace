### 2. M³-BioSynergy 数理モデルの設計

このモデルは、土壌を単なる「物質の器」ではなく、**「情報のネットワーク」**として捉えます。

#### ① Microbial (微生物階層)：共生ネットワークの微分方程式

120菌種の個体数動態を、ロトカ・ヴォルテラの方程式を拡張したモデルで記述します。

* **ロジック:** 
* **意味:** 菌種  の増殖は、周囲の菌種  との相互作用（共生  / 競合 ）によって決定されます。MBT55の投入は、この相互作用係数  を最適化する介入です。

#### ② Metabolic (代謝階層)：炭素変換効率 (CUE) の計算

微生物が有機物を分解する際、どれだけを呼吸（）として放出し、どれだけを菌体・腐植（炭素固定）として蓄積するかを計算します。

* **HealthBook連携:** この代謝プロセスが作物の機能性成分（ポリフェノール等）の生成源（前駆体）となります。

#### ③ Modular (モジュール階層)：機能ブロックの統合

土壌の「窒素循環」「リン溶解」「炭素固定」といった機能を独立したモジュールとして扱い、全体のSHE™スコア（土壌健全性）を算出します。

### 3. Python 実装コード（シミュレーション・コア）

```python
class M3BioSynergySimulator:
    def __init__(self, species_count=120):
        self.species_count = species_count
        # 相互作用行列（MBTライブラリーから生成）
        self.interaction_matrix = self._load_mbt_matrix()

    def simulate_step(self, current_population, mbt_intervention):
        """
        1ステップ（1時間後）の微生物動態と炭素固定量を予測
        """
        # 微生物個体数の変化計算（Lotka-Volterra拡張）
        growth_rates = self.interaction_matrix @ current_population
        new_population = current_population + (growth_rates * mbt_intervention)
        
        # 代謝産物（炭素固定量）の算出
        carbon_sequestration = self._calculate_cue(new_population)
        
        return new_population, carbon_sequestration

    def _calculate_cue(self, population):
        # Carbon Use Efficiency (炭素利用効率) の計算
        # 高いほど、CO2排出を抑え、土壌に炭素を貯留していることを示す
        return np.sum(population) * 0.45 # 係数はMBTライブラリーに基づく

```
