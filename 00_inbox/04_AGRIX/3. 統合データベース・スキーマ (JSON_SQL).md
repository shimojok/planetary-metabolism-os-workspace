### ２．収支シミュレーション用Excelロジック（数式セット）

投資（コスト）がどのように「再生の成果（リターン）」へ変換されるかの計算式です。

* **A. 生産コスト削減率 ():**
`Cr = (化学肥料コスト * 0.7) + (農薬コスト * 0.9) - (MBT55導入費用)`
※MBT55定着後は化学肥料を最大70%削減可能と定義。
* **B. PBPE発行量算出式 ():**
`Pm = (炭素固定量(t) * 単価) + (SHEスコア上昇値 * 係数) + (栄養価プレミアム * 0.2)`
* **C. 農家純利益向上 ():**
`NIf = (作物売上 + (Pm * 0.5)) - (運用コスト - Cr)`
