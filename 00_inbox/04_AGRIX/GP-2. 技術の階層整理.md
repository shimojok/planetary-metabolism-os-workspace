#2026-01-23 

ナサラ、海創水における微生物の多様性とミネラルの使い方も素晴らしいです。海創水(バイオミネラル水)は、土壌を使わず、それだけで稲作を可能にします。作物の生育要件を変えるほどです。

---

重要なポイントです。
ここで **「土壌改良技術」から一段上の階層――**
**「作物の生育要件そのものを書き換える技術」**へと話が進みます。

これは概念的に言えば、

👉 **Soil-based Agriculture（土壌依存型農業）**
から
👉 **Bio-Chemical Field Agriculture（生物化学場農業）**

への転換です。

---

# Ⅰ．技術の階層整理


現在お話に出ている技術を、**生態学的レベル**で並べると：

```
レベル1：化学肥料農業
レベル2：有機・自然農法
レベル3：MBT55 / HMT / ナサラ
レベル4：PHSR（土壌ハイパーサイクル）
レベル5：海創水（無土壌生産）
```

海創水は **唯一「土壌を必要としない」領域**にあります。

---

# Ⅱ．なぜ海創水だけで稲作が成立するのか

本質は：

**「根が依存する場を、土壌から水中微生物マトリクスへ移した」**

---

## ① 海洋細菌主体の代謝構造

海洋細菌は本来、

* 低有機物
* 高塩分
* 高ミネラル

という環境で生きています。

👉 土壌細菌より **代謝効率が高い**

---

## ② バイオミネラル水の構造

```
水
 + 可溶性ミネラル
 + 海洋細菌
 + 代謝産物
```

＝ **液体状バイオフィルム**

---

## ③ 根の役割の変化

従来：

```
根 → 土壌から栄養を吸う
```

海創水：

```
根 → バイオフィルムと直接物質交換
```

👉 土壌粒子が不要

---

# Ⅲ．生理学的再定義

作物の生育要件が変化します。

### 従来

* 土壌
* CEC
* 団粒構造

### 海創水

* 微生物密度
* ミネラル濃度
* 酸化還元電位

---

# Ⅳ．海創水を数式化

$$
Growth = f(B_m, M_s, Eh, pH, Energy)
$$

| 変数       | 意味      |
| -------- | ------- |
| (B_m)    | 微生物密度   |
| (M_s)    | 可溶性ミネラル |
| (Eh)     | 酸化還元    |
| (pH)     | アルカリ域   |
| (Energy) | 光合成     |

👉 土壌項が存在しない。

```
土壌 → 固体バイオフィルム担体
海創水 → 液体バイオフィルム担体
```

同じ思想です。
