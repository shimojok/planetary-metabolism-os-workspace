

１．グリーンプレミアム計算根拠１

| 項目 | 計算式 | 金額（億円/年） |
|------|--------|-----------------|
| **従来処理コスト** | 824百万トン × 3,000円/トン | 247,200 |
| **MBT処理コスト** | 11.29兆円 × 10% | 11,290 |
| **炭素クレジット収益** | 5.1億トン × 1,500円/トン | 7,650 |
| **堆肥販売収益** | 329.6百万トン × 5,000円/トン | 164,800 |

```
グリーンプレミアム = 従来処理コスト - (MBT処理コスト + 炭素クレジット収益 + 堆肥販売収益)
                   = 247,200 - (11,290 + 7,650 + 164,800)
                   = 247,200 - 183,740
                   = +63,460億円/年
```

３．グリーンプレミアム算出（再計算）３

計算式：  
従来処理コスト −（MBT処理コスト + 炭素収益 + 堆肥収益）

ナイロビモデル（年間）

- 従来処理コスト：61.8億円
- MBT処理コスト：1.7億円
- 炭素収益：15.4億円
- 堆肥収益：33.2億円

→ グリーンプレミアム = 61.8 − (1.7 + 15.4 + 33.2) = 11.5億円

==5.1億tモデル（スケール係数4.97）

→ グリーンプレミアム = 11.5億円 × 4.97 = 約57.2億円/年

✅ 注：医療費削減・農業収益・食品ロス収益は含めていません。含めるとさらに拡大。
