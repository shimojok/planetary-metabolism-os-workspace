==例をあげます。

下記はイエバエの幼虫が生成する堆肥の利用に関するものです。
https://smartagri-jp.com/smartagri/1187

一般的に行われている微生物による畜産廃棄物の発酵処理、つまり堆肥づくりには、最低2〜3カ月かかります。中には未熟な堆肥も多く、その処理は完熟させるまで1〜3年間くらいかかるといわれています。

一方、イエバエの幼虫が畜産系の廃棄物を分解すると、1週間で販売可能なレベルの肥料ができるので、製造期間を大幅に短縮できます。微生物による発酵処理と、イエバエの消化酵素による分解では、そのプロセスとタイムが大きく違うのです。

としていますが、
イエバエの卵を大量に入手し育て、主役の幼虫を管理し、さらに成長した蛹や成虫を処理する必要もあります。

考えれば分かりますが、堆肥生産量は、イエバエの幼虫の数と比例し、所詮、スケール感が望めないものです。有機物を分解するのは、イエバエではなく、腸内細菌なのですから、小さなイエバエの腸内に存在する細菌数、つまり処理量は小さなものです。

対するMBT55は、広大な農地や自然界で行なうわけですから、スケール感が違います。微生物は培地の面積により処置量が比例するのです。
さらに、イエバエが７日間で有機物を分解するところ、MBT55は24時間で完全に分解・発酵するのです。かつ、MBT55の管理コストは
ゼロであり、培養コストもイエバエの例より少なくてすみます。
もう一つは、イエバエの腸内細菌の分布は、自然界の土壌細菌がベースのMBT55と比較すると狭いはずです。

多くの投資家がこの事業に投資していますが、私の目には、開発者、この会社の社長も投資家も、バイオテクノロジーの本質、地球生態学のメカニズムを理解していない、問題解決の要件と解決すべき技術要件の裏付けに到達していないと映るのです。

---

🧠 ① Template設定項目（Choiceとして登録）

これは「テンプレートファイルを呼び出す操作」を設定する部分です。  
QuickAddでは「Choice」として登録されます。

| 項目 | 内容（Kaz向け） | 例 |
|------|------------------|----|
| Name | このChoiceの名前 | Create MBT55 Tech Note |
| Choice Type | 操作の種類 | Template を選択 |
| Template File Path | 呼び出すテンプレートのパス | templates/mbt55tech.md |
| File Name Format | 生成されるノートの名前 | MBT55_<% tp.date.now("YYYY-MM-DD") %> |
| Folder Path | ノートの保存先 | Projects/MBT55/TechNotes |
| Tags | ノートに付与するタグ | #MBT55 #tech #evaluation |
| Create Folder if Missing | 保存先フォルダがなければ自動作成 | ✅ ONにする |

---

⚡ ② Macro設定項目（複数Choiceをまとめる）

Macroは、複数のChoice（テンプレート呼び出し、入力プロンプト、タグ付けなど）をまとめて実行する機能です。  
Kazの場合は、テンプレート呼び出しだけでもMacroとして登録可能です。

| 項目 | 内容 | 例 |
|------|------|----|
| Macro Name | このMacroの名前 | MBT55技術ノート作成 |
| Choices Included | 実行するChoiceの一覧 | Create MBT55 Tech Note（上記のテンプレートChoice） |
| 実行順序 | 複数Choiceがある場合の順番 | 今回は1つなので不要 |

※スマホでは「Macro」という言葉が明示されないことがありますが、「Choiceを1つだけ登録してCommanderで呼び出す」＝実質的にMacro化です。

---

🎮 ③ Commander登録項目（1タップ化）

CommanderでこのMacro（またはChoice）を呼び出すことで、スマホでも1タップでノート作成が可能になります。

| 項目 | 内容 | 例 |
|------|------|----|
| Command Name | 表示名 | MBT55技術ノート作成 |
| 実行内容 | 呼び出すQuickAdd Choice | Create MBT55 Tech Note |
| ショートカット登録 | お気に入りやコマンドパレットに追加 | ✅ ONにすると即アクセス可能 |
