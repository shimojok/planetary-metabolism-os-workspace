AgriWare Appは、AgriWare API を利用して、農業生産者や農業関係者に圃場環境の遠隔計測や管理を行うアプリケーションです。AI Edge搭載モバイル端末により、さらにインテリジェンス化されます。

AgriWare APIは、AgriWare AI/Coreの機能を外部のアプリケーションやサービスに提供するAPIです。

#### About AgriWare®

The AgriCloud Platform consists of AgriWare®, AgriChain®, and SafetyChain®.

AgriWare consists of a mobile device equipped with an Edge-AI function and an AgriWare App that allows producers to enter data from the production site, and an AgriWare Core that is equipped with an AI system that accumulates agricultural big data and analyzes various parameters.

AgriWare AI/Core is a core system that collects data from drones, sensors, and satellite images, as well as various data collected by mobile devices equipped with Edge-AI/AgriWare App, such as soil environment, growth conditions, nutrient content, and freshness retention of crops, and analyzes them using phenomics methods, and proposes optimal field management.

Phenomics is a field of study that observes phenomena such as plant morphology, physiology, and biochemistry, and analyzes their relationship with genes and environment. Phenomics can evaluate plant productivity, quality, and adaptability, and contribute to the sustainability and safety of agriculture and food systems.

AgriWare App is an application that uses the AgriWare API to provide remote measurement and management of field environment to agricultural producers and stakeholders. It is further intelligentized by the AI Edge-equipped mobile device.

AgriWare API is an API that provides the functionality of AgriWare AI/Core to external applications and services.

#### AgriWare APIによる、金融機関、保険会社、投資会社など機関投資家へのサービス

AgriWare APIは、農業生産者の圃場環境や作物の品質や収量を高精度に評価することができるため、機関投資家にとって有益な情報源になります。

以下に、想定される、AgriWare APIを利用したサービスの事例を説明します。

•  農業ローンサービス：
    農業生産者が資金調達のためにローンを申し込む際に、AgriWare APIを使って圃場の状況や作物の見込み収量を分析し、返済能力やリスクを評価するサービスです。金融機関は、AgriWare APIの結果に基づいて、ローンの金利や条件を決めることができます。また、ローンの期間中にも、AgriWare APIを使って圃場の状況をモニタリングし、返済計画の変更や支援の提供などを行うことができます。

•  農業保険サービス：
    農業生産者が自然災害や病害虫などのリスクに対して保険に加入する際に、AgriWare APIを使って圃場の状況や作物の品質や収量を分析し、保険料や補償額を決めるサービスです。保険会社は、AgriWare APIの結果に基づいて、保険の種類や内容を提案することができます。また、保険の期間中にも、AgriWare APIを使って圃場の状況をモニタリングし、保険金の支払いや支援の提供などを行うことができます。

•  農業投資サービス：
    農業生産者が資金や技術やノウハウなどの支援を求める際に、AgriWare APIを使って圃場の状況や作物の品質や収量を分析し、投資価値やリターンを評価するサービスです。投資会社は、AgriWare APIの結果に基づいて、投資の対象や規模や期間を決めることができます。また、投資の期間中にも、AgriWare APIを使って圃場の状況をモニタリングし、投資の成果や問題点などを把握することができます。

以上のように、AgriWare APIは、機関投資家にとって、農業生産者の圃場環境や作物の品質や収量を高精度に評価することができる強力なツールになります。AgriWare APIを利用したサービスは、農業生産者と機関投資家の双方にメリットをもたらし、農業の発展や持続可能性に貢献することができます。

Investment companies can decide the target, scale, and duration of investment based on the results of AgriWare API. Also, during the investment period, they can use AgriWare API to monitor the field condition and grasp the results and problems of investment.

As shown above, AgriWare API can be a powerful tool for institutional investors, as it can accurately evaluate the field environment, crop quality, and yield of agricultural producers. Services that use AgriWare API can benefit both agricultural producers and institutional investors, and contribute to the development and sustainability of agriculture.

#### About AgriWare API (draft)

AgriWare API is an API for an advanced cloud-based agricultural production management system that is developed as part of the AgriWare Project. AgriWare API processes all the data obtained from agricultural sensors, machines, and service companies, and provides guidance and advice on proper use of fertilizers and herbicides, efficient use of machines, and work management based on plant nutrition expertise. AgriWare API can be a valuable source of information for institutional investors such as financial institutions, insurance companies, and investment companies, as it can accurately evaluate the field environment, crop quality, and yield of agricultural producers.

AgriWare API provides the following functions:

•  Remote measurement of field environment:
	AgriWare API collects data from drones, sensors, satellite images, etc., and measures environmental parameters such as soil, moisture, and temperature of the field in real time. AgriWare API analyzes these data using AI, detects the condition and problems of the field, and proposes optimal measures.

•  Evaluation of crop quality and yield:
	AgriWare API uses AI Edge devices to evaluate the nutrient and freshness retention of crops. AI Edge devices observe phenomena such as plant morphology, physiology, and biochemistry using phenomics methods, and analyze their relationship with genes and environment. Phenomics is a field of study that evaluates plant productivity, quality, and adaptability, and contributes to the sustainability and safety of agriculture and food systems. AgriWare API analyzes these data using AI, and predicts the quality and yield of crops.

•  Optimization of field management:
	AgriWare API optimizes field management based on the results of remote measurement and evaluation of field environment and crop quality and yield. AgriWare API provides instructions and advice on proper use of fertilizers and herbicides, efficient use of machines, and work management. AgriWare API creates customized field management plans according to the needs and goals of agricultural producers.


AgriWare API is provided as a RESTful web service. AgriWare API exchanges data in JSON format. AgriWare API performs authentication and authorization using OAuth 2.0. AgriWare API encrypts communication using HTTPS. AgriWare API generates documentation using Swagger.

AgriWare API provides a service that allows agricultural producers and stakeholders to remotely measure and manage field environment through an application called AgriWare App. AgriWare App can be used on mobile devices such as smartphones and tablets. AgriWare App provides the functionality of AgriWare API in a user-friendly interface. AgriWare App can perform evaluation and management of field environment and crop quality and yield using visual elements such as graphs and maps.

AgriWare API also provides the functionality of evaluating and managing field environment and crop quality and yield to external applications and services. AgriWare API can be a source of information for institutional investors such as financial institutions, insurance companies, and investment companies to evaluate the repayment ability, risk, investment value, and return of agricultural producers. AgriWare API can realize services such as agricultural loan service, agricultural insurance service, and agricultural investment service.
