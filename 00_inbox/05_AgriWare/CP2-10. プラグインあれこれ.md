# 🎨 Obsidian の Canvas：本物はこれだけ

## ✔ **Core Plugin（標準搭載）："Canvas"**
- Obsidian が公式に提供している唯一の Canvas
- 2022年以降のバージョンで標準搭載
- ノート・画像・PDF・リンクを自由に並べて構造図を作れる
- メモリ消費は軽い（画像を大量に貼らない限り問題なし）
- **Kaz が使うべき Canvas はこれだけ**

**→ Core Plugins の一覧にある “Canvas” を ON にすればOK。**

---

# ❌ 似た名前の「偽物」や「不要な Canvas プラグイン」

コミュニティプラグインには、Canvas に似た名前のものが複数ありますが、  
Kaz の用途には **すべて不要**です。

### 例（入れなくていいもの）

- Canvas Plus  
- Canvas Enhancer  
- Excalidraw Canvas

- Canvas Flow

- Canvas Zoom  
- Canvas Manager  

これらは  
- 重い  
- バグが多い  
- メモリを食う  
- スマホで不安定  
- Kaz のワークフローには不要  
です。

**→ Core Canvas だけで十分。**
