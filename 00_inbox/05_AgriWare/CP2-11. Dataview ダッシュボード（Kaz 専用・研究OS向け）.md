# 🎨 Canvas について（Kaz の疑問に明確に回答）

## ✔ 結論：**Canvas はコアプラグインです。**

Obsidian の設定画面では **カタカナで「キャンバス」** と表示されます。  
これが **本物の Canvas（公式）** です。

### 場所  
**設定 → コアプラグイン → キャンバス**

これだけ ON にすればOK。

### 注意  
コミュニティプラグイン側にある  
- Canvas Plus  
- Canvas Mindmap  
- Canvas Enhancer  
などは **全部不要** です。

Kaz の用途（構造図・アーキテクチャ図）には **公式 Canvas が最適で軽量**です。

# 🧩 Kaz のワークフローに最適化した結論

| プラグイン | 必要性 | 理由 |
|-----------|--------|------|
| **Dataview** | 必須 | 毎日数十ノートを自動集約する唯一の方法 |
| **Canvas（コア）** | 推奨 | AgriWare / HealthBook / BioNexus の構造図に最適 |
| **Periodic Notes** | 不要 | Kaz のノートは型にはまらない |
| **Advanced Tables** | 推奨 | 数理モデル・比較表を扱うのに最適 |
| **QuickAdd** | 不要 | テンプレートを多用しないなら不要 |
