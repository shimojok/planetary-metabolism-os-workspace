# Kaz のワークフローに最適化したタグ運用ルール

## ✔ 1. ノートには「階層タグ + 機能タグ + 文脈タグ」を付ける  
例：  
AgriWare の GHG モデルを AI と議論したノートなら：

```
#AgriWare #model #GHG #AI #chat
```

## ✔ 2. タグは“1ノートに複数”つけてよい  
Kaz のノートは多軸なので、1つのタグでは分類できない。

## ✔ 3. タグはフォルダより強力  
フォルダは「物理的分類」  
タグは「意味的分類」  
→ Dataview はタグの方が圧倒的に扱いやすい。

## ✔ 4. タグは後から追加しても問題なし  
Dataview が自動で拾う。
