# 🧠 Dataview 設定（あなたの画面に沿った最適設定）

---

# ① **Date + time format（日時フォーマット）**

### ✔ 推奨設定  
```
MMMM dd, yyyy - HH:mm
```

### ✔ 理由  
- Dataview ダッシュボードで「いつ作成したか」が一目でわかる  
- Daily Notes の自動集約が読みやすくなる  
- 英語表記の方が Dataview の並び順が安定する  

---

# ② **Primary column name（テーブルの最初の列名）**

### ✔ 推奨設定  
```
File
```

### ✔ 理由  
- Dataview の TABLE 表示で、最初の列が「ノート名」になる  
- 研究ログの一覧性が高い  
- GitHub の構造と一致する  

---

# ③ **Grouped column name（グループ化したときの列名）**

### ✔ 推奨設定  
```
Group
```

### ✔ 理由  
- AgriWare / MBT55 / HealthBook などをグループ化したときに見やすい  
- Dataview ダッシュボードで分類が明確になる  

---

# ④ **Automatic task completion tracking（タスク完了の自動記録）**

### ✔ 推奨設定  
**OFF**

### ✔ 理由  
Kaz のノートは  
- 研究ログ  
- AIチャット  
- 数式  
- コード  
- 戦略  
が中心で、タスク管理は主目的ではない。

ON にすると  
- ノートに勝手に `completion:: 2026-03-04` が挿入される  
- ノートが汚れる  
- Dataview の集約が乱れる  

**→ OFF が最適。**

---

# ⑤ **Warn on empty result（結果ゼロのとき警告）**

### ✔ 推奨設定  
**OFF**

### ✔ 理由  
Kaz のように大量ノートを扱うと、  
Dataview のクエリが「0件」になることはよくある。

ON にすると  
- 毎回警告が出て邪魔  
- ダッシュボードが見づらくなる  

---

# ⑥ **Render null as（null の表示）**

### ✔ 推奨設定  
```
-
```

### ✔ 理由  
- Dataview の TABLE が読みやすくなる  
- 空欄が「エラー」に見えない  
- Markdown と相性が良い  

---

# ⑦ **Automatic view refreshing（自動更新）**

### ✔ 推奨設定  
**OFF（強く推奨）**

### ✔ 理由  
Kaz のノート量では、ON にすると  
- ノート保存のたびに Dataview が全再計算  
- スマホで重くなる  
- フリーズする  
- 大規模ノートで落ちる  

**→ OFF が最適。**

---

# ⑧ **Refresh interval（更新間隔）**

### ✔ 推奨設定  
```
2500（デフォルトのまま）
```

### ✔ 理由  
自動更新を OFF にするため、ここは影響なし。  
デフォルトで問題なし。

---

# ⑨ **Date format（日時フォーマット）**

### ✔ 推奨設定  
```
MMMM dd, yyyy
```

### ✔ 理由  
- Dataview の TABLE で並び順が安定

# ⑩ **Enable inline queries（インラインクエリ）**

### ✔ 推奨設定  
**ON**

### ✔ 理由  
ノート内で

のような動的情報を使える。

研究ログに必須。

---

# ⑪ **Enable JavaScript queries（DataviewJS）**

### ✔ 推奨設定  
**ON（必須）**

### ✔ 理由  
Kaz の研究OSでは  
- 高度なダッシュボード  
- AIログ抽出  
- GitHub構造解析  
- MBT55 モジュール一覧  
などに JS クエリが必要。

---

# ⑫ **Enable inline JavaScript queries**

### ✔ 推奨設定  
**ON**

### ✔ 理由  
ノート内で  
`$= dv.current().file.name`  
のような JS を使える。

---

### ✔ 理由  
`key:: value` が見やすくなる。

---

# ⑭ **Enable inline field highlighting（Live Preview）**

### ✔ 推奨設定  
**ON**

### ✔ 理由

---

# ⑮ **DataviewJS keyword / Inline prefix / JS prefix**

### ✔ 推奨設定  
デフォルトのままでOK  
```

=
$=
```

### ✔ 理由  
変更すると混乱するため、標準のままが最適。
