### Core に含まれるべきもの

- AI基盤（エージェント、最適化、デジタルツイン）  
- 共通ユーティリティ（データ構造、可視化、API）  
- GitHub Actions / CI / セキュリティ基盤  

### Core が担う役割
- HealthBook と AGRIX の両方に **同じ科学基盤**を提供  
- SafetyChain と PBPE の **データ整合性**を保証  
- ACAIN（農業AIネットワーク）の **共通AIモデル**を提供

**Core は「BioNexusの脳」であり、他の事業はその“表現型”です。**
