# 🔷 3️⃣ 数理変換アルゴリズム

## Step 1：生薬→15経路マッピング辞書作成

形式：

```json
{
  "桂枝": [0.3,0.1,0.0,...],
  "芍薬": [0.0,0.2,0.4,...]
}
```

※ 各値は 0〜1 の寄与スコア  
※ 初期は文献ベース＋均質化

---

## Step 2：方剤ベクトル生成

方剤 F に含まれる生薬数を n とする。

[  
V_F = \frac{1}{n} \sum_{i=1}^{n} V_{herb_i}  
]

完全等重み平均。

---

## Step 3：正規化

L2正規化：

[  
V_F = \frac{V_F}{||V_F||}  
]

これにより距離計算安定化。
