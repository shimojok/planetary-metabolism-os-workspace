# 1️⃣ 診断視点の構造差

あなたが学ばれた  
長春中医薬大学  
北京中医薬大学

での内治・四診は、次の特徴を持ちます。

### 西洋医学

- 数値中心
    
- 臓器単位
    
- 病理確定
    
- 静的スナップショット
    

### 中医学（四診）

- 全身相関
    
- 動的バランス
    
- 症候群（証）
    
- 機能偏位
    

つまり、

> 西洋医学 = 点の診断  
> 中医学 = 状態空間の位置推定

です。

---

# 2️⃣ フェノタイピングとして再定義

四診は実は：

- 視診 → 形態情報
    
- 聞診 → 音声・呼吸情報
    
- 問診 → 主観症状データ
    
- 切診 → 脈動パターン
    

という**マルチモーダルデータ取得プロトコル**

です。

これはAIに極めて適しています。

---

# 3️⃣ 数理的に翻訳すると

「証」は曖昧概念ではなく、

> 高次元代謝状態のクラスタリング結果

とみなせます。

例：

- 気虚 → 低ATP生成効率＋自律神経副交感低下
    
- 瘀血 → 微小循環低下＋炎症性サイトカイン偏位
    
- 痰湿 → 脂質代謝滞留＋インスリン抵抗性傾向
    

これらは代謝経路解析と結び付け可能です。
