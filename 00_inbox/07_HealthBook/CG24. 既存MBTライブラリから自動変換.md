# ② 成分 → 15経路 自動変換ロジック

MBTライブラリに既に存在する：

- 代謝関連タグ
    
- 炎症関連タグ
    
- 微生物関連タグ
    
- 既存作用記述
    

これを数値化します。

---

## スコア変換関数

例：

```
"AMPK activation" → +0.8 (v5)
"NF-kB inhibition" → +0.9 (v6)
"lipogenesis suppression" → +0.7 (v4)
```

自然言語タグ → 経路IDへのマッピング辞書を作ります。

---

## 変換式

$$  
v_r(H_p)

\frac{  
\sum_m  
C_m \cdot TagScore_{m,r}  
}{  
\sum_m C_m  
}  
$$

正規化範囲：

$$  
v_r \in [-1, 1]  
$$

# ③ 自動生成パイプライン（実装骨格）

```python
class MBTVectorizer:

    def load_library(self, path):
        return library_json

    def compound_to_pathway(self, compound):
        return pathway_score_vector_15d

    def formula_vector(self, formula):
        V = np.zeros(15)
        for herb in formula["herbs"]:
            for compound in herb["compounds"]:
                score = self.compound_to_pathway(compound["compound_name"])
                V += compound["weight"] * score
        return self.normalize(V)

    def normalize(self, V):
        return np.tanh(V)
```
