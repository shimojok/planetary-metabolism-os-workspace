# ② 294処方ベクトル化

## 2.1 ベクトル生成式（タグ主導）

$$  
v_r(H_p)

\tanh!\left(  
\sum_{herb}  
w_{herb}  
\sum_{tag \in herb}  
\mathbf{1}(tag \to r)  
\right)  
$$

- 出力：15次元、範囲 ([-1,1])
    
- タグなし成分は0（中立）
    

## 2.2 実装骨格

```python
import numpy as np

class TagAbstractor:
    def __init__(self, tag_map, dim=15):
        self.tag_map = tag_map
        self.dim = dim

    def formula_vector(self, formula):
        V = np.zeros(self.dim)
        for herb in formula["herbs"]:
            w = herb.get("weight_ratio", 1.0)
            tags = herb.get("action_tags", [])
            for t in tags:
                if t in self.tag_map:
                    V[self.tag_map[t]] += w
        return np.tanh(V)
```

## 2.3 品質チェック

- 各次元の分散が十分あるか（σ > 0.1目安）
    
- 全処方の平均が0付近に過度集中していないか
    
- 極端な外れ値の確認

# ④ 出力と解釈

## 4.1 クラスタ重心

$$  
\bar{V}_c = \frac{1}{|C|}\sum_{H_p \in C} V_p  
$$

→ 各クラスタの代謝戦略プロファイルを数値表示。

## 4.2 仮ラベル付与（例）

- C0：抗炎症主導型（v5, v7高）

- C2：腸内再構築型（v12, v13高）
    
- C3：脂質代謝改善型（v2, v3高）
    
- C4：HPA調整型（v10高）
