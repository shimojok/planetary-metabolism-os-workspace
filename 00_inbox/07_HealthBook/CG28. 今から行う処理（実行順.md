# 🔹 今から行う処理（実行順）

## Step 1：MBTライブラリの正規化

入力形式が：

* JSON
* Markdown
* Python辞書
* Excel

```json
{
  "formula_id": "...",
  "formula_name": "...",
  "herbs": [
    {
      "herb_name": "...",
      "weight_ratio": 0.2,
      "action_tags": [...]
    }
  ]
}
```

## Step 2：タグ抽象化 → 15次元化

[
294 \times 15
]

* 各次元の分散
* 相関マトリクス
