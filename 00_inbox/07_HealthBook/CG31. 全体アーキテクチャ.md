# 設計思想

1処方 = 複数経路寄与の線形結合

[  
V_f = \sum_{i=1}^{n} w_i P_i  
]

- ( V_f )：処方ベクトル
    
- ( P_i )：15経路基底ベクトル
    
- ( w_i )：寄与重み

# 自動生成アルゴリズム（4層ロジック）

---

## Layer 2：疾患 → 経路逆マッピング

既存 `DiseaseMatrix` から

例：

- 胃がん → PATH14 + PATH09
    
- 糖尿病 → PATH05 + PATH15

---

## Layer 3：HealthBook補正係数導入

例：

```json
"HealthBook補正係数": 0.75
```

これを経路寄与のスケーリング因子として適用。

[  
w_i = raw_score_i \times correction  
]

---

## Layer 4：正規化

最終的に：

[  
\sum_{j=1}^{15} V_{f,j} = 1  
]

L1正規化を採用。

# 実装仕様（Python）

```python
import numpy as np

class FormulaVectorizer:

    def __init__(self, tag_dict, disease_map):
        self.tag_dict = tag_dict
        self.disease_map = disease_map

    def vectorize(self, formula):
        vector = np.zeros(15)

        # 1. タグから加算
        for tag in formula["Functional_Tags"]:
            idx = self.tag_dict[tag] - 1
            vector[idx] += 1

        # 2. 疾患から加算
        for disease in formula["ターゲット疾患"]:
            if disease in self.disease_map:
                for path, weight in self.disease_map[disease]:
                    vector[path-1] += weight

        # 3. 補正係数
        correction = formula.get("HealthBook補正係数", 1.0)
        vector *= correction

        # 4. 正規化
        if vector.sum() > 0:
            vector = vector / vector.sum()

        return vector
```
