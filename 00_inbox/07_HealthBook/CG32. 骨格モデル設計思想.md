# Ⅰ. 骨格モデル設計思想

ルールベースは **3種類のスコア合成** で構築します。

[  
V_f = \alpha T + \beta D + \gamma M  
]

- **T（Tagスコア）**：生薬機能タグ由来
    
- **D（Diseaseスコア）**：ターゲット疾患由来
    
- **M（Metaboliteスコア）**：MBT代謝産物由来
    

初期係数：

[  
\alpha=0.5,\quad \beta=0.3,\quad \gamma=0.2  
]

理由：

- 生薬構成が最も本質的
    
- 疾患は臨床バイアス
    
- 代謝産物は補助軸

# Ⅳ. ベクトル生成アルゴリズム（骨格版）

```python
import numpy as np

class MBT15Vectorizer:

    def __init__(self, tag_rule, disease_rule, metabolite_rule):
        self.tag_rule = tag_rule
        self.disease_rule = disease_rule
        self.metabolite_rule = metabolite_rule

        self.alpha = 0.5
        self.beta  = 0.3
        self.gamma = 0.2

    def vectorize(self, formula):

        T = np.zeros(15)
        D = np.zeros(15)
        M = np.zeros(15)

        # --- Tag ---
        for tag in formula.get("Functional_Tags", []):
            if tag in self.tag_rule:
                idx, w = self.tag_rule[tag]
                T[idx-1] += w

        # --- Disease ---
        for disease in formula.get("ターゲット疾患", []):
            if disease in self.disease_rule:
                for idx, w in self.disease_rule[disease]:
                    D[idx-1] += w

        # --- Metabolite ---
        for m in formula.get("MBT代謝産物", []):
            if m in self.metabolite_rule:
                idx, w = self.metabolite_rule[m]
                M[idx-1] += w

        # 合成
        V = self.alpha*T + self.beta*D + self.gamma*M

        # 補正係数
        correction = formula.get("HealthBook補正係数", 1.0)
        V *= correction

        # 正規化
        if V.sum() > 0:
            V = V / V.sum()

        return V
```
