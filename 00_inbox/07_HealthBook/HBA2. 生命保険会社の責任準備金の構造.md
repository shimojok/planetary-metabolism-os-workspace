# Ⅰ. 生命保険会社の責任準備金の構造

生命保険会社の責任準備金 ( L ) は、将来給付の現在価値：

$$  
L = \sum_{t=0}^{T} \frac{E[B_t]}{(1+r)^t}  
$$

ここで：

- ( B_t )：時点 ( t ) における保険給付
    
- ( r )：割引率（予定利率）
    
- ( E[\cdot] )：期待値
    
給付は主に：
$$  
B_t = q_t \cdot S + C_t  
$$

- ( q_t )：死亡率または罹患率
    
- ( S )：保険金額
    
- ( C_t )：医療給付・入院給付など
    

したがって：

$$  
L = \sum_{t} \frac{q_t S + C_t}{(1+r)^t}  
$$

## 本質

責任準備金は：

* 死亡率
* 罹患率
* 医療費水準
* 割引率

に依存する。

HealthBookは、**( q_t ) と ( C_t ) の構造そのものを下げるモデル**である。

# Ⅱ. HealthBookによる責任準備金変革モデル

## 1. 代謝改善による死亡率低下

仮に：
$$
q_t^{new} = q_t (1-\alpha)
$$

ここで：

* ( \alpha )：代謝改善による死亡率改善率

責任準備金は：
$$
L^{new} = \sum_t \frac{q_t(1-\alpha)S + C_t^{new}}{(1+r)^t}

差分：
$$
\Delta L = L - L^{new}
$$

これは構造的な準備金削減。
