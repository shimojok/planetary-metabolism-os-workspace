# ④ Excel用数式テンプレート

以下をそのままExcelに入力可能。

---

## シート構成

### 入力

|セル|内容|
|---|---|
|A1|総責任準備金|
|A2|医療給付比率|
|A3|死亡率改善率|
|A4|医療費削減率|
|A5|割引率|

---

## 数式

死亡部分削減：

```
= A1 * (1 - A2) * A3
```

医療部分削減：

```
= A1 * A2 * A4
```

合計削減：

```
= 上記2つの合計
```

削減率：

```
= 合計削減 / A1
```

---

## NPV計算

```
= Σ (給付削減額 / (1+r)^t)
```
