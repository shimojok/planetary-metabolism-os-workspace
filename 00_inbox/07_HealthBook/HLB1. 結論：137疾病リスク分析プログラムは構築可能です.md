# 💡 **ステップ1：問診質問（200項目）をデータベース化**

アップロードされたPDF（問診表）から項目抽出し…

**例）問1：濃い野菜をよく食べるか？
選択肢：はい／いいえ
→ 食道がん：−
→ 慢性胃炎：＋**

といった形に構造化します。

---

# 💡 **ステップ2：病気と食生活の関係シート（137疾病×200項目の重み）を抽出**

これを以下の形式に変換します：

```json
{
  "濃い野菜を食べない": {
    "食道がん": 1,
    "肺がん": 1,
    "慢性胃炎": -1
  },
  "お酒をよく飲む": {
    "食道がん": -1,
    "慢性胃炎": 1
  }
}
```

この重み行列をモデルの基盤として使用します。

---

# 💡 **ステップ3：ユーザー回答 → 137疾病のスコア算出アルゴリズム**

Pythonなどで以下のようなロジックになります：

```python
for disease in diseases:
    score[disease] = sum(
        weight[question][disease] * answer_value
        for question in answered_questions
    )
```

この**スコアを正規化 → リスクレベル（Low / Mid / High）**に変換します。
