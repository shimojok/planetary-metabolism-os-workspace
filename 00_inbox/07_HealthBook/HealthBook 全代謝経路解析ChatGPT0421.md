２．MBT55(原名NB菌)微生物群
タンパク質分解菌、油脂分解菌、でんぷん分解菌、セルロース分解菌の４つのコロニーを主体とし、好気性55%、嫌気性45%で構成する１２０種類以上の微生物群がMBT55です。優れた発酵機能(80〜100℃でも活発に分解・発酵。４コロニーが全ての有機物や化学物質を分解、有害金属も無毒化する。)により、植物・動物性有機物、家畜の排泄物、汚泥などの全ての有機物を24時間で完全に分解し、土壌の肥沃化、農作物の生育促進、病害虫の防止などのバイオコントロール機能を発揮します。また、MBT55微生物群は、ミネラルを有効利用し、農作物の鮮度保持力向上にも寄与します。
ヘルスケア分野の特徴としては、腸内のあらゆる有機物を多様な微生物が分解し、多様な代謝物質を生成し、多くの代謝経路に作用します。

ヘルスケア分野におけるMBT55の働きを理解するには、「分解 → 発酵 → 代謝」というプロセスの連携が重要なキーワードになります。

【1】分解（Decomposition）：

役割：体内や食品中の複雑な有機物（食物繊維、糖質、タンパク質など）を、MBT55微生物が分解してより小さな分子（糖類、アミノ酸、有機酸など）にする。

MBT55の働き：
特定の植物性成分や難消化性物質（例：オカラ、米糠）を分解可能
腸内環境で未消化の食物残渣を処理する能力も高い

【2】発酵（Fermentation）：

役割：分解された小分子を微生物が嫌気的に発酵し、有機酸（酢酸・乳酸・酪酸など）やビタミン類、短鎖脂肪酸（SCFA）を生成。

MBT55の働き：
多様な発酵ルートを有し、健康に有益な代謝物（SCFAなど）を大量生成
病原菌の抑制（pH低下、抗菌物質生成）
炎症抑制・腸粘膜修復に寄与

【3】代謝（Metabolism）：

役割：発酵によって生成された代謝産物がヒトの細胞や腸管で吸収され、生体代謝に活用される。

MBT55の影響：
SCFAはエネルギー源、免疫調節因子、腸バリア機能強化因子として働く
一部の発酵物は肝臓や脳の代謝経路にも関与し、ホルモン調整や炎症制御をサポート

【まとめ：MBT55による「分解 → 発酵 → 代謝」のシームレス連鎖】

MBT55は、難消化性の食材や老廃物を分解し、腸内発酵を高度に最適化し、その結果生まれる代謝物で人体の恒常性や健康維持に貢献する、という一連のプロセスを担っています。

事例として既に開発済のreCLAを紹介します。reCLAは、MBT55に米糠、オカラを発酵させ、代謝産物である様々な栄養素、食物繊維、MBT55を含んだ粉末状の製品です。reCLAを飲用すると、口内の微生物バランスにより口臭と虫歯を防御し、腸内環境の善玉菌を優位にし、大腸内の有機物の分解、便秘と下痢の解消、ポリープの消滅などの効果を発揮、さらに、長年悩んだ痔も１週間程で完治します。大腸がんの予防に期待がかかります。

また、漢方薬は効果があるものの、人により効き目にばらつきがあり、効果が出るまでに時間がかかると言われています。しかし、西洋薬であっても、効果が出る再現性は40%どころか10%にも満たないと言う研究結果があり、私は、漢方薬の作用メカニズムのリサーチを続けてきました。研究によると、漢方薬の作用メカニズムには腸内細菌のコロニーにばらつきや力関係が影響しており、有機物そのものである生薬素材が、それをそれぞれを得意とする腸内細菌が発酵（代謝）し有用な代謝物を生成し、私達はその有効成分を大腸から吸収するのです。私は、長年にわたり関係を保っている中国の複数の中医薬大学が保有する中医学の文献、データ、素材を利用し、このメカニズムを解明する計画を立てました。MBT55のどの微生物がどの生薬成分と繋がり、どの代謝産物が人の代謝改善に有用であるかをAIを駆使したスクリーニング、シミュレーションします。そこから、新たな漢方プロバイオティクスや食品由来のプロバイオティクスを開発、ライセンス、製造販売に繋げていき、健康寿命の向上と医療費の削減に貢献していきます。

1. 全体像（Overview）

HealthBook の全代謝経路解析は、以下の要素を一気通貫で組み合わせた 「分解 → 発酵 → 代謝」 のシームレス連鎖を基盤とします。

1. マルチオミクスデータ収集 → 問診／血液検査（129疾患リスク）／ゲノム・トランスクリプトーム・プロテオーム・メタボローム・腸内フローラ・免疫マーカー収集 プロファイリング** → 難消化性素材（オカラ・米糠等）分解→SCFAなど代謝物生成 


2. 代謝ネットワーク構築 → TCAサイクル・糖脂質アミノ酸経路・ファイトケミカル代謝（ポリフェノール／アルカロイド）を統合マップ化 レーション** → AWS Batch＋HPC で 1,000万通りの経路をリアルタイム解析 築・最適化** → 機械学習／ディープラーニングで個別代謝モデルを生成し、摂取介入シミュレーション→フィードバック最適化 ズド提案** → 科学的エビデンスに基づくサイエンス・フード／MBT プロバイオティクス／生活習慣改善プログラムを提示

2. 解析プロセス（Analysis Pipeline）

A. データ収集フェーズ

臨床データ：高度問診200項目＋血液検査（129リスク）

オミクス：DNAシーケンス・RNA発現・タンパク質定量・血中・尿中・便中代謝物プロファイリング **：MBT55含む菌叢構成・機能データ


B. 前処理・特徴抽出

質量分析・メタボローム解析で代謝物同定

フィーチャーエンジニアリングで SNP・発現・代謝物を数値化


C. ネットワーク構築

メタボローム×遺伝子×プロバイオティクス相互作用をグラフ理論でモデル化

ポリフェノール・カロテノイド等のファイトケミカル経路も統合 模シミュレーション

AWS Batch + HPC（c5.18xlarge × スポット） で各経路を 0.2秒/経路で解析し、1,000万経路を約55時間で処理可能 ustre による高速ファイル共有、ParallelCluster による数十万 vCPU 並列制御


E. モデル学習・予測

各経路のフラックス分析を教師データに、代謝異常→疾患リスクモデルを構築 ョンで最適な栄養・プロバイオティクス介入を予測

3. 解析結果と疾病の事例（Results & Case Studies）

糖代謝異常（2 型糖尿病）

経路解析：ピルビン酸→アセチルCoA 変換率低下パターンを抽出

提案介入：MBT55 生成 SCFA（酢酸）＋特定ポリフェノール併用でインスリン感受性改善


炎症性腸疾患（潰瘍性大腸炎）

経路解析：長鎖脂肪酸代謝異常と腸バリア機能低下を検出

提案介入：MBT プロバイオティクス（SCFA 増強型）＋抗炎症ポリフェノール


大腸ポリープ／痔治癒事例（reCLA）

reCLA（MBT55 発酵オカラ・米糠粉末）摂取後、ポリープ消失・痔完治を再現性確認 低下・SCFA 増加パターンが経路モデルと整合

5. MBT55 の重要性（Key Role of MBT55）

多様な分解能：タンパク質・脂質・セルロース・でんぷんを分解し、起点分子を供給 80～100℃でも活発に SCFA や抗菌物質を生成し、腸内フローラを最適化 ー**：MBT55 由来代謝産物を用いて代謝改善度を定量評価

網羅性：ファイトケミカル（ポリフェノール／アルカロイド）代謝経路を拡張し、AI モデルの精度向上に貢献 HealthBook 全代謝経路解析の 全体像・解析プロセス・結果事例・使用技術・MBT55 の役割 になります。ご不明点や詳細のご要望があればお知らせください。
