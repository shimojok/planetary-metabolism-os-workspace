## 一、プロジェクトの背景とビジョン

HealthBook は、AIとビッグデータを駆使した「予防・解析・改善」を統合した健康管理プラットフォームです。医療費の削減と健康寿命の延伸を目的としており、以下の核心機能を備えています：

- **200項目のAI問診システム**
- **129種類（実質137種類）の疾病リスク評価モデル**
- **MBT55腸内細菌代謝解析**
- **全代謝経路AIシミュレーション**
- **中医学に基づくパーソナライズ健康提案**

現在の Amazon AWS 環境から **Microsoft Azure** への移行を計画しており、Azure のグローバルインフラ、AIサービス、データコンプライアンスを活用し、より安全で拡張性の高い健康プラットフォームを構築します。

## 二、疾病リスク分析プログラミングの仕組み

### 1. データソースとモデル基盤  
**30年間・30万人以上の患者データ**に基づき、137種類の疾病と生活習慣の関連性をマトリクス化（「病気と食生活の関係シート」）しています。

### 2. 問診表とリスクマッピング機構  
- **4種類の問診カテゴリ**：
  - 食習慣
  - 生活習慣
  - 環境・遺伝要因
  - 現在の症状・健診データ
- 各回答は**プラス（+）／マイナス（-）の重み**を持ち、アルゴリズムにより疾病リスクスコアを算出。

### 3. リスク計算の例（大腸がんの場合）  
| リスク要因 | 重み | ユーザー回答 | スコア |
|------------|------|--------------|--------|
| 野菜摂取不足 | +2 | はい | 2 |
| 10年以上の飲酒歴 | +3 | はい | 3 |
| 運動頻度が高い | -1 | はい | -1 |
| **累計リスクスコア** | | | **4（高リスク）** |

### 4. AIによる継続的学習機構  
ユーザーの健康データと疾病アウトカムを収集し、**Azure Machine Learning** を用いてリスクモデルを継続的に最適化します。

```
   - パーソナライズ健康レポート（Power BI + Azure Logic Apps）
   - 栄養・運動プログラム配信（Azure Bot Service）

```

## 四、MBT55代謝解析との統合

HealthBook はリスク予測だけでなく、**代謝改善に基づく介入プログラム**を提供します：

1. **腸内細菌解析** → 栄養吸収障害の特定  
2. **代謝産物測定** → SCFA・ビタミン等の評価  
3. **パーソナライズドプロバイオティクス提案** → MBT55発酵産物に基づく腸内環境最適化  

この一連の流れをAIで自動化し、「リスク検知」から「精密介入」までの閉ループを実現します。
