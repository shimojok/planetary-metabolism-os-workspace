このシステムは、問診回答を「リスク因子」として捉えるだけでなく、下條先生の**「作物の質（ファイトケミカルの機能）」をリスク相殺の変数**として組み込んでいる点が最大の武器です。

#### ロジックの構成：

1. **ベースリスク算出**: 浜田先生の「病気と食生活の関係シート」に基づき、137疾病の●（重み）を合算。
2. **ファイトケミカル・ブースト（相殺係数）**:
* 例：「野菜が少ない」にチェックがあっても、摂取している野菜が**AGRIX産（高鮮度・高ミネラル）**であれば、リスク加算を50%軽減。
* 例：脂溶性成分（カロテノイド）の摂取がある場合、細胞膜に関連する疾患リスクを低減。

### 2. プロトタイプ・プログラム（Python）

このコードは、アップロードされたCSVデータを読み込み、ユーザーの回答に応じて疾病リスクを算出するエンジンの核となります。

```python
import pandas as pd
import numpy as np

class HealthBookEngineV2:
    def __init__(self, matrix_path):
        # 浜田先生の137疾病マトリックスをロード
        self.matrix = pd.read_csv(matrix_path)
        self.diseases = self.matrix['病名'].tolist()
        
    def calculate_risk(self, user_answers, phytochemical_quality_score=1.0):
        """
        user_answers: { '早食い型': 1, '野菜少ない': 1, ... } (1: Yes, 0: No)
        phytochemical_quality_score: 1.0 (標準) ~ 0.5 (AGRIX高品質によるリスク低減)
        """
        results = {}
        
        for index, row in self.matrix.iterrows():
            disease_name = row['病名']
            base_score = 0
            
            # 各リスク因子をスキャン
            for factor, answer in user_answers.items():
                if factor in row and row[factor] == '●':
                    # 下條式ブースト：特定の改善因子があればリスク重みを軽減
                    weight = 1.0
                    if factor == '野菜少ない' and phytochemical_quality_score < 1.0:
                        weight = phytochemical_quality_score  # 質が高ければリスク上昇を抑える
                    
                    base_score += (answer * weight)
            
            results[disease_name] = base_score
            
        return pd.DataFrame(results.items(), columns=['Disease', 'RiskScore']).sort_values(by='RiskScore', ascending=False)

# システムの実行例
# 137疾病マトリックスを読み込み
engine = HealthBookEngineV2('disease_matrix.csv')

# ユーザーの問診回答サンプル（200項目のうちの一部）
user_data = {
    '早食い型': 1,
    '野菜少ない': 1,
    '塩分': 1,
    '飲酒': 0,
    '夜食方(寝る前の食事）': 1
}

# AGRIX高品質野菜（ファイトケミカル高含有）を摂取している場合 (0.6倍のリスク低減)
final_report = engine.calculate_risk(user_data, phytochemical_quality_score=0.6)

print(final_report.head(10)) # リスクが高い上位10疾患を出力

```
