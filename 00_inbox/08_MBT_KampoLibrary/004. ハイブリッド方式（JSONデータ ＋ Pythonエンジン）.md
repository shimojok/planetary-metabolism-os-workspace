### ステップ2：統合推論エンジンの実装（Python）

次に、このJSONを読み込んで、個人のリスクと改善策を導き出す「脳」を構築します。

```python
import json

class BioNexusInferenceEngine:
    def __init__(self, knowledge_base_path):
        with open(knowledge_base_path, 'r', encoding='utf-8') as f:
            self.kb = json.load(f)
            
    def analyze_phenotype(self, user_responses, agrix_quality_data):
        """
        user_responses: 200項目の回答データ
        agrix_quality_data: 摂取している作物の「質」データ（下條分類に基づく）
        """
        disease_scores = {d: 0 for d in self.kb['Disease_Weights'].keys()}
        
        # 1. 浜田式リスク・マッピング
        for q_id, is_yes in user_responses.items():
            if is_yes:
                question = next(q for q in self.kb['Questions'] if q['id'] == q_id)
                for disease in question['risk_factors']:
                    disease_scores[disease] += 1.0 # 重み付けの基本加算
        
        # 2. 下條式ファイトケミカル・ブースト（リスク相殺）
        for disease, score in disease_scores.items():
            mitigation_type = self.kb['Disease_Weights'][disease]['Mitigation_Group']
            
            # AGRIXの高品質な「質」が、特定部位の防衛能力を高める
            if mitigation_type == "Hydrophilic_Defense":
                # 水溶性成分（フラボノイド等）が豊富なら、血液・体液系疾患リスクを低減
                score *= (1.0 - agrix_quality_data['hydrophilic_index'])
            elif mitigation_type == "Lipophilic_Defense":
                # 脂溶性成分（カロテノイド等）が豊富なら、細胞膜系疾患リスクを低減
                score *= (1.0 - agrix_quality_data['lipophilic_index'])
                
            disease_scores[disease] = score

        return self.generate_prescription_report(disease_scores)

    def generate_prescription_report(self, scores):
        # リスクが高い順にソートし、MBT55処方と漢方294を紐付けて出力
        return sorted(scores.items(), key=lambda x: x[1], reverse=True)

# 実行イメージ
engine = BioNexusInferenceEngine('knowledge_base.json')
# AGRIX産（高品質）のファイトケミカルを摂取している設定
agrix_data = {'hydrophilic_index': 0.4, 'lipophilic_index': 0.3} 
report = engine.analyze_phenotype(user_answers_from_web, agrix_data)

```

### 2. 統合推論エンジン（Python）の実装仕様

Pythonエンジンは「脳」として機能し、以下の動的なステップを実行します。

1. **データの読み込み**: JSON化された浜田・下條マスターをメモリにロードします。
2. **フェノタイプ・マッピング**: 200項目の問診回答を解析し、137疾病のベーススコアを算出。
3. **AGRIX ✕ 下條ブースト計算**: 摂取している作物の「質」が高い場合、その疾病が属する「防衛エリア（水溶性/脂溶性）」に応じたリスク軽減率を乗算します。
4. **MBT55代謝シミュレーション**: 腸内細菌がその成分を正しく活性化できるかを判定し、最終的な「個別化リスク」を出力します。

### 3. 社会実装用プロトタイプ・コード（ハイブリッド構成）

このコードがBioNexusのプラットフォームの心臓部となります。

```python
import json

class HealthBook_Precision_Engine:
    def __init__(self, master_data_path):
        # JSONからマスターデータを読み込み
        with open(master_data_path, 'r', encoding='utf-8') as f:
            self.master_data = json.load(f)

    def analyze(self, user_responses, food_quality_data):
        """
        user_responses: 問診のYes/No
        food_quality_data: AGRIXが測定した作物のファイトケミカル含有量
        """
        final_risks = {}

        for disease in self.master_data['Diseases']:
            disease_name = disease['name']
            # 1. 浜田式ベースリスク計算
            score = 0
            for factor in disease['risk_factors']:
                if user_responses.get(factor) == "YES":
                    score += 1  # 臨床データに基づく重み付け
            
            # 2. 下條式ファイトケミカル相殺 (Quality Boost)
            defense_type = disease['defense_type'] # 水溶性か脂溶性か
            mitigation_factor = food_quality_data.get(defense_type, 0.0)
            
            # 質が高いほどリスクスコアを下げる (例: 最大50%軽減)
            score = score * (1.0 - (mitigation_factor * 0.5))
            
            final_risks[disease_name] = round(score, 2)

        return sorted(final_risks.items(), key=lambda x: x[1], reverse=True)

# 実行イメージ
# 知識マスター(JSON)を読み込み、Pythonで演算
engine = HealthBook_Precision_Engine('BioNexus_Master_V2.json')
# AGRIXデータ: このユーザーが食べる野菜は水溶性成分の質が40%高い
agrix_input = {'hydrophilic': 0.4, 'lipophilic': 0.2}
result = engine.analyze(user_answers_200, agrix_input)
```

### 2. 統合推論エンジンの実装（Pythonハイブリッド方式）

このJSONを読み込んで、個人の「フェノタイプ（現状）」から「未来のリスク」を算出するエンジンを構築します。

```python
import json

class BioNexus_Phenotype_Engine:
    def __init__(self, master_path):
        # 浜田・下條統合ナレッジマスターのロード
        with open(master_path, 'r', encoding='utf-8') as f:
            self.master = json.load(f)

    def calculate_risk(self, answers, agrix_data):
        """
        answers: { "Q004": 1, "Q018": 1 ... } (1=Yes, 0=No)
        agrix_data: { "hydrophilic_quality": 0.8, "lipophilic_quality": 0.5 }
        """
        results = {}
        for disease in self.master['Diseases']:
            # 1. 浜田式ベースリスク計算
            risk_score = sum(answers.get(q_id, 0) * weight 
                            for q_id, weight in disease['risk_factors'].items())
            
            # 2. 下條式ファイトケミカル・ブースト (リスク相殺)
            defense = disease['defense_type'] # "Hydrophilic" or "Lipophilic"
            quality = agrix_data.get(f"{defense.lower()}_quality", 0)
            
            # 高品質な作物を食べていれば、リスクを動的に軽減
            mitigated_score = risk_score * (1.0 - (quality * 0.5))
            results[disease['name']] = round(mitigated_score, 2)
            
        return sorted(results.items(), key=lambda x: x[1], reverse=True)

```
