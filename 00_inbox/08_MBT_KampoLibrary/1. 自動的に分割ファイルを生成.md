おっしゃる通りです。手動での提供には限界があります。

## Pythonで自動生成する方法

以下のスクリプトで、既存のJSONファイルから自動的に分割ファイルを生成できます：

```python
"""
MBT漢方ライブラリー 分割ツール
既存のマスターJSONから10方剤ずつのファイルを自動生成
"""

import json
import os
from typing import List, Dict

class KampoSplitter:
    """漢方ライブラリー分割クラス"""
    
    def __init__(self, input_file: str):
        with open(input_file, 'r', encoding='utf-8') as f:
            self.data = json.load(f)
        
        # 配列形式かオブジェクト形式かを判定
        formulas_data = self.data['mbt_kampo_library']['formulas']
        if isinstance(formulas_data, dict):
            # オブジェクト形式 → 配列に変換
            self.formulas = []
            for fid, fdata in formulas_data.items():
                formula = {"id": fid, **fdata}
                self.formulas.append(formula)
            # IDでソート
            self.formulas.sort(key=lambda x: x['id'])
        else:
            # 配列形式
            self.formulas = formulas_data
    
    def split_into_parts(self, part_size: int = 10):
        """10方剤ずつに分割"""
        parts = []
        for i in range(0, len(self.formulas), part_size):
            part_formulas = self.formulas[i:i+part_size]
            part_num = i // part_size + 1
            
            first_fid = part_formulas[0]['id']
            last_fid = part_formulas[-1]['id']
            
            part_data = {
                "mbt_kampo_library": {
                    "version": self.data['mbt_kampo_library']['version'],
                    "part": f"{part_num}/22",
                    "formula_range": f"{first_fid}-{last_fid}",
                    "last_updated": self.data['mbt_kampo_library']['last_updated'],
                    "total_formulas": 294,
                    "fully_defined": len(part_formulas),
                    "classification_source": self.data['mbt_kampo_library']['classification_source'],
                    "mbt55_version": self.data['mbt_kampo_library']['mbt55_version'],
                    "formulas": part_formulas
                }
            }
            parts.append((part_num, part_data))
        
        return parts
    
    def save_parts(self, parts: List[tuple], output_dir: str = "parts"):
        """分割ファイルを保存"""
        os.makedirs(output_dir, exist_ok=True)
        
        for part_num, part_data in parts:
            filename = f"mbt_kampo_part{part_num:02d}.json"
            filepath = os.path.join(output_dir, filename)
            
            with open(filepath, 'w', encoding='utf-8') as f:
                json.dump(part_data, f, ensure_ascii=False, indent=2)
            
            print(f"✅ 生成: {filename} (方剤数: {len(part_data['mbt_kampo_library']['formulas'])})")
        
        print(f"\n📁 全{len(parts)}ファイルを '{output_dir}' に保存しました")

def main():
    """メイン実行"""
    print("="*60)
    print("MBT漢方ライブラリー 分割ツール")
    print("="*60)
    
    # 入力ファイルを指定
    input_file = "mbt_kampo_library_master.json"  # 既存のマスターJSON
    
    if not os.path.exists(input_file):
        print(f"❌ ファイルが見つかりません: {input_file}")
        return
    
    splitter = KampoSplitter(input_file)
    print(f"📊 総方剤数: {len(splitter.formulas)}")
    
    parts = splitter.split_into_parts(10)
    splitter.save_parts(parts)
    
    print("\n✅ 分割完了")

if __name__ == "__main__":
    main()
```

## 使用方法

1. **既存のマスターJSON**（211方剤を含む）を用意
2. 上記スクリプトを `split_kampo.py` として保存
3. 実行：
```bash
python split_kampo.py
```

## 出力

- `parts/` フォルダに `mbt_kampo_part01.json` 〜 `mbt_kampo_part22.json` が自動生成されます
- 各ファイルには10方剤ずつ（最後のパートのみ11方剤）が含まれます

この方法で、正確な分割ファイルが得られます。
