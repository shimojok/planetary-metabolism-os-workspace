## ✅ はい、その理解で正しいです！

### ① 入力ファイルについて
```python
input_file = "mbt_kampo_library_master_v2.44.json"
```
で**正しい**です。

**理由**:
- このスクリプトは **元のオブジェクト形式** のJSONに新しい方剤を追加するためのものです
- 配列形式（`_array.json`）は表示用/開発チーム渡し用であり、**編集用のマスターは元の形式** のまま維持するのがベストプラクティスです

### ② スクリプトのファイル名
**推奨ファイル名**: `add_missing_kampo.py`

```bash
add_missing_kampo.py
```

### ③ 処理の流れ
1. 元のJSON（オブジェクト形式）を読み込む
2. 103方剂（実際は102）を追加
3. 同じく**オブジェクト形式**で保存（`mbt_kampo_library_complete.json`）
4. 必要に応じて、後で配列形式に変換する

### ④ 補足：欠番について
- F030: 欠番（スキップ）
- 実際に追加するのは **102方剂** ですが、最終的な総数は **294方剂** になります
