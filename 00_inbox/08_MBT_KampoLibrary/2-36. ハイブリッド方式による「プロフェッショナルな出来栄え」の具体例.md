### 3. ハイブリッド方式による「プロフェッショナルな出来栄え」の具体例

先生のシステムをこの方式で構築した場合、以下のような構造になります。

#### **【データ層：JSON】（知恵のマスターデータ）**

```json
{
  "Question_ID": "Q018",
  "Text": "色の濃い野菜は嫌いなほうである",
  "Shimojo_Category": "Flavonoid_Deficiency_Risk",
  "Base_Weights": { "Gastric_Cancer": 1.2, "Immune_Disorder": 1.5 },
  "AGRIX_Mitigation_Factor": 0.5

```

> ※「もし野菜の質が高ければ、リスクを0.5倍に軽減する」という**下條理論**がここに定義されます。

#### **【エンジン層：Python】（推論の脳）**

```python
def calculate_metabolic_risk(user_answer, json_data):
    # JSONからルールを読み込み、Pythonで動的に計算
    risk = json_data['Base_Weights'] * user_answer
    if user_is_using_agrix:
        risk *= json_data['AGRIX_Mitigation_Factor'] # 質によるリスク相殺

```
