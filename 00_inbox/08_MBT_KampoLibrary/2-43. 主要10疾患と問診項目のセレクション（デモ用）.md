### 2. インタラクティブ・ダッシュボードの構成

デモ画面では、以下の3つのセクションが連動して動きます。

#### ① リスク・インプット・パネル（左側）

ユーザーが「早食い：Yes」「野菜不足：Yes」とチェックを入れると、即座に計算が走ります。

#### ② 躍動するリスク・レーダー（中央）

計算結果が137疾患（デモでは10疾患強調）のレーダーチャートとして現れます。

* **演出**: リスクが高い項目は、赤く脈打つようなアニメーション（Pulsing Effect）を加え、危機感を視覚化します。

#### ③ 下條式「質（AGRIX）」トグル（右側）

ここがこのデモの**「決め手」**です。「AGRIX Grade Food」というスイッチをONにすると、以下の変化が起こります。

* **挙動**: グラフがグンと縮小し、色が「警告の赤」から「安全の緑」へ変化。
* **表示**: 「下條分類：水溶性シールドが血液の炎症を30%抑制しました」というエビデンス・メッセージがポップアップします。

### 3. デモ用ロジックの実行コード（Python）

背後で動く簡潔かつ強力なエンジンです。

```python
# BioNexus First-Demo Logic
def calculate_demo_risk(answers, agrix_mode=False):
    # 浜田マトリックス（デモ用抜粋）
    matrix = {
        "胃がん": {"weights": {"Q04": 1.5, "Q11": 1.0, "Q18": 2.0}, "defense": "hydrophilic"},
        "狭心症": {"weights": {"Q02": 1.2, "Q09": 1.8, "Q21": 1.5}, "defense": "lipophilic"},
        "眼精疲労": {"weights": {"Q15": 2.5, "Q18": 1.5}, "defense": "hydrophilic"}
    }
    
    results = {}
    for disease, data in matrix.items():
        score = sum(answers.get(q, 0) * w for q, w in data['weights'].items())
        
        # 下條式「質」の相殺ロジック
        if agrix_mode:
            # 高品質ファイトケミカルは、その防衛ラインのリスクを40%カットする
            score *= 0.6 
            
        results[disease] = round(score, 2)
    return results

```
