GitHub Pagesで言語切替UIを実現する手順を説明します。

---

## 全体の流れ

```
1. GitHubリポジトリ作成
2. ファイル構成を準備
3. GitHub Pagesを有効化
4. 公開URLで確認
```

---

## ステップ1：リポジトリ作成

1. [github.com](https://github.com/) にログイン
2. 右上「**+**」→「**New repository**」
3. 設定：
    - Repository name: 例 `mbt55-climate-overview`
    - **Public** を選択（GitHub Pagesは無料プランではPublicのみ）
    - 「Add a README file」にチェック
4. 「**Create repository**」

---

## ステップ2：ファイル構成

以下の構成でファイルを用意します：

```
mbt55-climate-overview/
├── index.html        ← メインページ（言語切替UI込み）
├── slides_en.pdf     ← 英語版スライド
└── slides_ja.pdf     ← 日本語版スライド
```

MDファイルは`index.html`の中に**直接埋め込む**形が最もシンプルです（外部fetch不要）。

---

## ステップ3：index.html の構成イメージ

ポイントは3つです：

- 英語・日本語のコンテンツを`<div>`で分けて持つ
- ボタンクリックでCSSの`display`を切り替える
- PDFリンクも言語に連動して切り替える

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>MBT55 Climate Overview</title>
  <style>
    .lang-en, .lang-ja { display: none; }
    .active { display: block; }
    /* ボタンのスタイル等 */
  </style>
</head>
<body>

  <!-- 言語切替ボタン -->
  <button onclick="switchLang('en')">English</button>
  <button onclick="switchLang('ja')">日本語</button>

  <!-- 英語コンテンツ -->
  <div class="lang-en" id="content-en">
    <h1>Comprehensive Overview of Climate Change and MBT55</h1>
    <!-- 英語本文をここに -->
    <a href="slides_en.pdf">📄 Download English Slides (PDF)</a>
  </div>

  <!-- 日本語コンテンツ -->
  <div class="lang-ja" id="content-ja">
    <h1>気候変動とMBT55の科学的・応用的ポテンシャル総覧</h1>
    <!-- 日本語本文をここに -->
    <a href="slides_ja.pdf">📄 日本語スライドをダウンロード (PDF)</a>
  </div>

  <script>
    function switchLang(lang) {
      document.querySelectorAll('[class^="lang-"]').forEach(el => {
        el.classList.remove('active');
      });
      document.getElementById('content-' + lang).classList.add('active');
    }
    // デフォルトは英語
    switchLang('en');
  </script>

</body>
</html>
```

## ステップ4：ファイルをGitHubにアップロード

### 方法A：ブラウザから（コマンド不要）

1. リポジトリページで「**Add file**」→「**Upload files**」
2. `index.html`、PDFをドラッグ&ドロップ
3. 「**Commit changes**」

### 方法B：Git CLIから

```bash
git clone https://github.com/あなたのID/mbt55-climate-overview.git
cd mbt55-climate-overview
# ファイルをフォルダに配置後
git add .
git commit -m "Add multilingual page"
git push origin main
```

---

## ステップ5：GitHub Pagesを有効化

1. リポジトリの「**Settings**」タブ
2. 左メニュー「**Pages**」
3. Source：「**Deploy from a branch**」
4. Branch：`main` / `/ (root)` を選択
5. 「**Save**」

数分後、以下のURLで公開されます：

```
https://あなたのGitHubID.github.io/mbt55-climate-overview/
```
