## ✅ 寄与率計算の信頼性

### 現在のデータでの計算は**可能だが、限界あり**

**良い点**：
- 191方剤については十分なデータがある
- 主要な植物化学物質は網羅されている

**問題点**：
- 103方剂のデータが不足 → これらの処方は寄与率が過小評価される

- 濃度情報が範囲指定（例：`"3.0-6.0"`）であり、正確な数値ではない

```python
# 特徴ベクトル生成時の重み付け
def calculate_pathway_score(formula):
    scores = {f"PATH_{i:02d}": 0.0 for i in range(1, 6)}
    
    # 1. phytochemicals からの寄与（高信頼）
    for phytochem in get_phytochemicals(formula):
        pathway = phytochem.get('metabolic_pathway')
        if pathway and pathway in scores:
            conc = parse_concentration(phytochem.get('concentration_mg_per_g', '0-0'))
            scores[pathway] += conc * 1.0  # 信頼度1.0
    
    # 2. 多糖類などからの寄与（中信頼）
    for herb_data in formula['components'].values():
        if 'polysaccharides' in herb_data:
            ps = herb_data['polysaccharides']
            if ps.get('metabolic_pathway') == 'PATH_05':
                scores['PATH_05'] += 0.5  # 多糖類の寄与（低めに）
    
    # 3. データ不足の処方への補完（低信頼）
    if is_data_sparse(formula):
        # 主要生薬からパスウェイを推測
        for herb in formula.get('main_herbs', []):
            if herb in HERB_TO_PATHWAY:
                scores[HERB_TO_PATHWAY[herb]] += 0.3  # 補完値
    
    return scores
```
