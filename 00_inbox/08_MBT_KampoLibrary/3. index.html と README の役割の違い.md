## index.html と README の役割の違い

|ファイル|誰が見るか|目的|
|---|---|---|
|`index.html`|**一般ユーザー**（公開URL経由）|言語切替UI付きの本文を表示するWebページ|
|`README.md`|**GitHubを見る開発者・共同作業者**|リポジトリの説明、使い方、公開URLへの誘導|

つまり**2つは別物で、両方必要**です。

---

## index.html の中身

はい、アップロードいただいた**英語版と日本語版の本文を両方埋め込んだ**HTMLファイルです。ボタンで表示を切り替えます。

```
index.html = 英語版本文 + 日本語版本文 + 言語切替ボタン + PDFリンク
```

## README.md の中身

GitHubのリポジトリトップに自動表示される説明ファイルです。こう書くのがおすすめです：

```markdown
# MBT55 Climate Change Overview

Multilingual overview of climate change and the scientific/applied 
potential of MBT55.

## 🌐 Public Page (GitHub Pages)
👉 https://あなたのID.github.io/mbt55-climate-overview/

English and Japanese versions available with language toggle.

## 📄 Files
- `index.html` - Main multilingual web page
- `slides_en.pdf` - English presentation slides
- `slides_ja.pdf` - Japanese presentation slides

## 📝 Source Documents
- English: https://gist.github.com/shimojok/b046b4105956ad2289d7f31efd8088d2
```
