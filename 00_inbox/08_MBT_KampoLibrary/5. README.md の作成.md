mbt55-climate-overview 直下にIndex.htmlをアップロードするのですね？

---

はい、その通りです。

リポジトリの**ルート直下**（一番上の階層）に置くのがポイントです。

```
mbt55-climate-overview/          ← リポジトリのルート
├── index.html                   ← ここに置く ✅
├── slides_en.pdf                ← 後で追加
└── slides_ja.pdf                ← 後で追加
```

GitHub Pagesは**ルートにある `index.html` を自動的にトップページとして認識**します。フォルダの中に入れてしまうとURLが変わってしまうので、必ずルート直下に置いてください。

---

## アップロード手順（ブラウザから）

1. `https://github.com/あなたのID/mbt55-climate-overview` を開く
2. 「**Add file**」→「**Upload files**」をクリック
3. ダウンロードした `index.html` をドラッグ＆ドロップ
4. 下の「**Commit changes**」ボタンを押す

これだけで完了です。その後 Settings → Pages で公開設定をすれば、数分で URL が発行されます。

## 推奨手順

### ① まず index.html をアップロード（済んだら）

### ② GitHub Pages を有効化

Settings → Pages → Branch: main / root → Save

### ③ URLが確定してから README.md を作成

URLが確定したら（例：`https://shimojok.github.io/mbt55-climate-overview/`）、以下の内容でREADMEを作成します。

---

## README.md の作成方法

リポジトリトップの「**Add file**」→「**Create new file**」をクリックし、ファイル名に `README.md` と入力して、以下をペーストします：

```markdown
# MBT55 Climate Change Overview

Comprehensive overview of climate change and the scientific & applied 
potential of MBT55 (Microbial BioTransformation 55).

## 🌐 Public Page
👉 https://shimojok.github.io/mbt55-climate-overview/

English and Japanese versions available with language toggle.

## 📄 Presentation Slides
- [English PDF](./slides_en.pdf)
- [日本語 PDF](./slides_ja.pdf)

## 📝 Source Documents
- English (full): https://gist.github.com/shimojok/b046b4105956ad2289d7f31efd8088d2
- 日本語版：本リポジトリの index.html に収録

## 📁 Files
- `index.html` — Multilingual web page (EN / JA toggle)
- `slides_en.pdf` — English presentation slides
- `slides_ja.pdf` — Japanese presentation slides
```
