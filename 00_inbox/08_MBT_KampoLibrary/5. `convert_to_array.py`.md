## ✅ やるべきこと：変換スクリプトを実行する

あなたの元ファイル  
`mbt_kampo_library_master_v2.44.json`  
を、**配列形式のJSON** に変換します。

変換後のファイル名は（たとえば）  
`mbt_kampo_library_array.json`  
とします。

---

## 🐍 Pythonスクリプト（完成版）

以下のスクリプトを `convert_to_array.py` などの名前で保存し、  
**元のJSONファイルと同じフォルダに置いて実行**してください。

```python
import json

# 入力ファイル（あなたの元ファイル）
input_file = "mbt_kampo_library_master_v2.44.json"

# 出力ファイル（配列形式）
output_file = "mbt_kampo_library_array.json"

# JSONを読み込む
with open(input_file, "r", encoding="utf-8") as f:
    data = json.load(f)

# ルートオブジェクトを取得
root = data["mbt_kampo_library"]

# メタ情報（必要に応じて調整）
meta = {
    "version": root["version"],
    "last_updated": root["last_updated"],
    "total_formulas": root["total_formulas"],
    "fully_defined": root["fully_defined"],
    "classification_source": root["classification_source"],
    "mbt55_version": root["mbt55_version"]
}

# 処方データをオブジェクト → 配列に変換
formulas_list = []
for formula_id, formula_data in root["formulas"].items():
    # IDを各オブジェクトのプロパティとして追加（任意）
    formula_data["id"] = formula_id
    formulas_list.append(formula_data)

# 新しい構造
output_data = {
    **meta,
    "formulas": formulas_list
}

# JSONファイルとして保存
with open(output_file, "w", encoding="utf-8") as f:
    json.dump(output_data, f, ensure_ascii=False, indent=2)

print(f"✅ 変換完了: {output_file}")
print(f"   → 処方数: {len(formulas_list)}")
```

---

## ▶️ 実行方法（ターミナル / コマンドプロンプト）

```bash
python convert_to_array.py
```

実行すると、  
`mbt_kampo_library_array.json` が作成されます。
