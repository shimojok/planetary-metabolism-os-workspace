## ✅ ② 生成済み行列（再現可能定義）

```text
【PATH寄与行列 定義】

■ 基本情報
　・対象：294方剤（F001〜F294、F030は欠番のため除外）
　・次元：5次元（PATH_01〜PATH_05）
　・出力形式：293行 × 5列の数値行列（L2正規化済み）

■ 特徴量計算アルゴリズム

Step 1: 初期化
　各方剤について、5次元のスコア配列をゼロ初期化
　score[PATH_01〜PATH_05] = 0.0

Step 2: phytochemicals の走査
　各生薬の components 内の phytochemicals 配列をループ：
　　・metabolic_pathway が PATH_01〜05 のいずれかの場合：
　　　　concentration_mg_per_g をパース（例：「3.0-6.0」→ 平均値 4.5）
　　　　該当する PATH のスコアに加算
　　・metabolic_pathway がない場合はスキップ

Step 3: polysaccharides の処理
　各生薬に polysaccharides オブジェクトがある場合：
　　・metabolic_pathway が PATH_05 の場合：
　　　　content_percent をパース（例：「70-80%」→ 平均値 75）
　　　　スコアに加算（phytochemicals と同等の重み付け）

Step 4: minerals の処理
　各生薬に minerals オブジェクトがある場合：
　　・すべて PATH_02 に加算（ミネラル還元代謝）
　　・含有量をパース（例：「200-250 mg/g」→ 平均値 225）
　　・mg/g 単位をそのまま使用

Step 5: 正規化
　各方剤の5次元ベクトルに対して L2正規化を適用：
　　norm = sqrt(Σ(score_i²))
　　if norm > 0:
　　　　normalized_score_i = score_i / norm
　　else:
　　　　normalized_score_i = 0（全成分ゼロの場合）

■ 濃度文字列のパースルール
　・範囲形式「X-Y」→ (X + Y) / 2
　・単一値「Z」→ Z
　・パーセント「X%」→ X（数値化）
　・未記載 → デフォルト値 1.0

■ 補足事項
　・データ不足の方剤（fully_defined = false）は、phytochemicals がないため
　　主に polysaccharides や minerals のみで構成される
　・F030は欠番のため、行列から除外
　・最終的な行数 = 293（294 - 1欠番）
```
