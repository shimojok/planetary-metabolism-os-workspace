## 🧭 このビジョンをどうライブラリー構築に反映するか

1. **症状・疾病クラスターを2軸で整理**  
   - 地域軸：途上国（感染・栄養・母子）／先進国（代謝・ストレス・慢性疾患）  
   - 年齢・性別軸：乳幼児／妊産婦／成人／高齢者／女性特有疾患

2. **方剤を“社会的適応”でタグ付け**  
   - 例：麻黄湯 → #感染症 #呼吸器 #急性期 #途上国対応  
   - 加味逍遙散 → #ストレス #女性ホルモン #生活習慣病 #先進国対応

3. **MBT55代謝物の“社会的インパクト”評価**  
   - 例：SCFA → 腸内環境改善 → 乳幼児の下痢予防／メタボ改善  
   - グリチルリチン誘導体 → 粘膜修復 → 呼吸器感染症対策
