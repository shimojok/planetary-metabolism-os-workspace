    → 生理痛・月経不順（女性健康）
    → 生活習慣病（高血圧・血栓傾向）

  → 清熱 / 収斂
    → 花粉症
    → 寒冷蕁麻疹

粘膜修復代謝物（ムチン誘導・上皮修復）

    → むくみ
    → 腎・泌尿器のうっ滞
    → 心不全・循環負荷
