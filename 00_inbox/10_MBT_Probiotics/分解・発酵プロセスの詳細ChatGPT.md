# 要点まとめ（先に結論）

1. **24時間で「極限まで分解して pH7 の良質堆肥」を作る**という現場観察は、MBT55 の高い酵素活性と好気/嫌気の混合コミュニティにより「非常に速い加水分解（溶解）」と「初期発酵（酸化・発酵代謝）」が起きるため説明可能。ただし「完全な最終安定化（成熟した腐植＝フミック物質中心の堆肥）」が**本当に出来上がるか否かは時間の問題**で、通常は数週間〜数か月を要する。24時間では「高度に分解された、病原性低下・悪臭がない加工有機物（固形またはスラリー）」が得られる、というのが現実的な解釈です。
    
2. MBT55 の作用は**「加水分解（酵素的分解）＋その後の発酵（代謝）」**の二段階が同時に、かつ高速で進行していると考えられます。タンパク質はプロテアーゼでアミノ酸に分解され、その一部は最終的に NH₄⁺（アンモニウム）や CO₂、短鎖有機酸（SCFA）などに代謝されます — つまり **発酵は確実に起きています**。
    
3. 「完全発酵」後の堆肥の主成分は **安定化有機物（腐植・フミック酸類）、微生物バイオマス、鉱化された無機栄養（NH₄⁺/NO₃⁻, PO₄³⁻, K⁺）、灰分（ミネラル）** が主で、短鎖揮発代謝物や可溶性糖・脂肪酸は大きく減少します。ただしセルロース・リグニン由来の最終的なフラグメント（フェノール類やバニリン様化合物など）は残存し得ます。

# A. 分解・発酵プロセスの詳細（依頼文１への考察）

MBT55 の構成（好気55% / 嫌気45%、多様な分解能）を前提に、投入物（糞・食品残渣・豚骨・キャベツ・新聞紙＋オカラ）を24時間で処理する際の主要プロセスは次のように進みます。

1. **即時的な物理化学的準備（0–1時間）**
    
    - 破砕・撹拌により表面積増加 → 酵素作用促進。水分が適正（概ね50–65%）なら酵素反応が速く進行。
        
    - オカラの投入は水分調整とともに C:N バランス（窒素源）を整え、微生物の活性化材となる。
        
2. **急速な加水分解（1–6時間） — 加水分解期**
    
    - MBT55 が持つ **アミラーゼ、プロテアーゼ、リパーゼ、セルラーゼ、リグニナーゼ等** がでんぷん、タンパク質、脂質、セルロース、リグニンを短鎖可溶化合物（オリゴ糖、糖、ペプチド、脂肪酸、フェノール類）へ分解。
        
    - 骨（コラーゲン）もコラーゲナーゼ等でコラーゲンがペプチド・アミノ酸に分解。ただし骨格の無機部分（リン酸カルシウム）は機械的処理や長時間を要する。
        
3. **高速代謝フェーズ（6–24時間） — 発酵・酸化期**
    
    - 好気性層では呼吸（O₂ 消費→CO₂・H₂O、熱）とともに有機物が酸化され、嫌気性ポケットでは発酵（糖→乳酸、酢酸、プロピオン酸、ブチル酸等）や発酵性代謝が起きる。
        
    - タンパク由来アミノ酸は**脱アミノ化**されアンモニア（NH₃/NH₄⁺）へ、一部は脱炭酸などで短鎖酸や CO₂ へ。乳酸菌群＞乳酸生成、酵母や他の嫌気性群でアルコールや有機酸が生じる。
        
    - 放線菌・糸状菌・リグニン分解菌はセルロース・リグニンの複雑な分解を開始し、フェノール性中間体を生じる。
        
4. **短時間で pH が中性化される理由**
    
    - 初期に生成される酸（有機酸）は、好気性微生物やアンモニア（プロテイン分解による塩基性成分）の相互作用・微生物による有機酸の消費で短期間に緩衝され、pH7 程度に落ち着く。MBT55 は酸生成と酸消費の“両方”の群を含むため悪臭（揮発性硫黄化合物や揮発性アミン）が抑えられる。
        
5. **悪臭が出ないメカニズム**
    
    - 嫌気的発酵の典型的な悪臭源（硫化水素、アンモニア、揮発性脂肪酸）が、MBT55 に含まれる硫黄酸化/還元連鎖、マンガン・鉄酸化還元群、硝化/脱窒群、メタン酸化菌などのバランスのために**短時間で代謝され消失または酸化**される。加えて高分解率（60%以上）により未分解の臭気源が残らない。

# B. 発酵はあったか？（依頼文２への回答）

**結論：はい、発酵（代謝的発酵）は確実に起きています。**

理由：

- 「タンパク質がアミノ酸に分解されている」＝プロテアーゼ活性（加水分解）ですが、その後、微生物がアミノ酸を脱アミノ化／酸化してアンモニウムや有機酸／CO₂ に代謝していることが示唆されます。これは発酵および呼吸代謝の一部です。
    
- 同様にデンプンや脂質は加水分解で可溶化し、その一部が発酵的に糖→乳酸／酢酸などに変換され、さらに好気的に酸化される経路が併存します。
    
- MBT55 には乳酸菌群、酵母、嫌気性発酵群、メタン酸化菌などが含まれるため、**好気的酸化と嫌気的発酵が同時並行**に起こり、全体として「発酵プロセスを伴う分解」が起きていると判断できます。

# C. 「完全発酵」の堆肥の成分（依頼文３への考察）

24時間で“極限まで分解”された後に得られる物質群（短期処理後の実態）と、成熟堆肥（長期安定化）での主要成分を分けて示します。

## C.1 短時間（24h）処理後に多く存在すると予想される成分

- **可溶性有機物**：オリゴ糖、単糖（短時間は一部残る）、可溶性ペプチド、アミノ酸（ロイシン等列挙のもの）、モノ/ジグリセリド、遊離脂肪酸。
    
- **揮発性代謝物（過渡的）**：乳酸、酢酸、プロピオン酸などの短鎖有機酸（多くは微生物により速やかに消費される）。
    
- **アンモニウム（NH₄⁺）**：タンパク分解→脱アミノ化で増加。条件が整えば亜硝酸／硝酸へ変換（亜硝酸菌・硝化菌の活動）。
    
- **微生物バイオマス**：活発に増殖した細菌・真菌の自重（微生物由来の有機物が相当量）。
    
- **無機イオン・ミネラル**：P、K、Ca（骨由来）、Mg 等の溶出。
    
- **残存難分解性高分子**：特にリグニンや重合フェノール類は部分分解されるが短時間で完全消失はしない。
    

> つまり「でんぷん・糖質、油脂などの分解・発酵物質や、セルロース・リグニン分解後の代謝産物」は**存在するが“短時間で減少するもの”と“残りやすい分子”に分かれる**というのが現実です。

## C.2 長期成熟（完全発酵＝成熟堆肥）で主に残る成分

- **腐植物（フミック酸、フルボ酸、フミン）**：高分子化・縮合反応で安定化した有機物（堆肥の主体的化合物）。
    
- **微生物残骸（ネクロマス）**：分解後の細胞壁成分（グラム陰性/陽性の殻、糖鎖等）。
    
- **安定化有機炭素（SOC）**：化学的に安定化した炭素。
    
- **可植物利用形窒素（NO₃⁻/NH₄⁺）・可溶性P・K**：農業用肥効を示す成分。
    
- **無機灰分（ミネラル）**：骨由来のカルシウム等は不可逆的に残る。
    
- **揮発性有機化合物（VOC）や短鎖酸は低レベルに**：成熟堆肥では揮発性が低く、悪臭はほぼ無い。
    

## C.3 リグニン・セルロース由来の代謝産物は「完全に消えない」

- セルロースは比較的分解されやすく、オリゴ糖→糖→CO₂へと進みやすい。
    
- リグニンは複雑で難分解（芳香族骨格）。部分的に低分子フェノール類（ヴァニリン、シンナムアルデヒド様）や小分子芳香族酸に変わり、最終的には微生物の代謝や縮合反応で**フミック化**される。従って「代謝産物は無い」とはならず、**形を変えて残る（安定化される）**ことが多いです。

# D. MBT55 に特に期待できる反応・酵素（要点）

- アミラーゼ、セルラーゼ（endo/exo）、β-グルコシダーゼ
    
- プロテアーゼ（エキソ・エンド）、コラーゲナーゼ（骨・コラーゲン分解）
    
- リパーゼ／エステラーゼ（脂肪分解）
    
- リグニナーゼ群（ラッカーゼ／過酸化物質依存リグニン分解酵素）
    
- 硝化酵素（アンモニア酸化菌→亜硝酸菌→硝化菌）
    
- 硫黄変換群（硫化物酸化菌・硫酸還元菌のバランス）
    
- マンガン/鉄 酵素（マンガン還元/酸化を介した芳香族代謝）
    

これらの混合作用が短時間で多数の基質を同時分解できる理由です。

# E. 実際に「完全発酵」と判定するために測るべき指標（提案）

堆肥の成熟度と性状を実証するには次を測定してください。

1. **化学指標**
    
    - pH、EC（電気伝導度）
        
    - TOC（全有機炭素）、Total N → C:N 比
        
    - NH₄⁺-N / NO₃⁻-N（窒素鉱化・硝化の進行）
        
    - 可溶性有機炭素（DOC）
        
    - フミック酸・フルボ酸含量（分光/抽出法）
        
    - 揮発性脂肪酸（GC）、VOCs（GC-MS）
        
2. **物理／生物学指標**
    
    - 呼吸速度（O₂ 消費または CO₂ 放出 rate） → 安定して低ければ成熟傾向
        
    - 熱発生（温度）推移
        
    - 微生物バイオマス量（ATP 法や PLFA 法）
        
    - 病原微生物の有無（大腸菌群、サルモネラ等）
        
3. **機能評価**
    
    - 植物バイオアッセイ（発芽率・初期生育）
        
    - 土壌保水性・団粒化指標
        

**判定基準（例）**：呼吸速度低下＋DOC 低下＋安定した pH（約7）＋低揮発性（悪臭なし）で「使用可能な堆肥」と判断し得る。フミック化の程度は数週間〜月単位の測定で裏付ける。

# F. 実務的示唆（最後に）

- **24時間で「利用可能」な堆肥・有機液肥**を作るという点は、MBT55 の高活性コミュニティと工程設計（撹拌・水分・オカラ投入など）で実現可能。ただし「完全成熟フミック堆肥」と断定するのは慎重で、成熟度を証明するための上記検査が必要です。
    
- 骨の無機成分（カルシウム等）は短時間でも溶出・懸濁して残るため、pH や電解質に影響することがあります（肥料効果に寄与）。
    
- MBT55 が示す「悪臭のない高効率分解」は、揮発性硫化物やアンモニアを迅速に代謝・酸化する群が存在するため科学的に説明可能です（好気・嫌気のバランスが鍵）。
    
- **推奨実験**：24時間処理の前後で「TOC、TN、NH₄⁺、NO₃⁻、DOC、VFA、呼吸速度、フミック/フルボ酸量、微生物カウント、病原検査」を測ってください。これで「何が残っているか」「本当に腐植化が起きたか」「安全か」を定量的に示せます。
