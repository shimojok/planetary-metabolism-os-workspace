
MBT55 ✕漢方薬の議題に移ります。

方剤事例（漢方薬）：麻黄湯の場合
効能:　風邪の初期で、頭痛・悪寒・発熱・腰痛・関節痛・咳・喘息などの症状の改善に用いられる。臨床では、初期のインフルエンザ[1]や各種感染症・気管支炎・気管支喘息の発作期・アレルギー性鼻炎の発作期・夜尿症などに応用する。

麻黄、杏仁、桂枝、甘草

とあります。

これは、一例ですが、明確な薬理作用があるもの以外は、作用メカニズムははっきりしていません。つまり、投与する漢方薬、生薬の成分だけでは作用機序は分からないと言うことです。生薬成分も単一ではないことも理由の一つです。

漢方薬、生薬の原材料は、植物、動物、鉱物であり、食品に例えると、消化、腸内細菌による代謝の対象になります。つまり、腸内細菌の代謝産物の薬理作用を見る必要があるわけです。鉱物は、複数のミネラルの固まりと考えれば同じように分かります。

そこで、例に上げた漢方薬の、MBT55微生物群による代謝、代謝産物の効能などを、MBT漢方代謝ライブラリーの構築を行います。

一般的な腸内細菌叢とMBT55微生物群の違いは、生成される代謝産物の質と種類に直結し、それが薬理作用や健康効果の違いに繋がります。多数の漢方薬とMBT55による代謝産物をリサーチにより新たな発見があるはずです。

下記は、麻黄湯を構成する生薬のMBT55による代謝産物について調べたものです。

#### 1. 麻黄（Ephedra sinica）
- **主成分**：エフェドリン類、タンニン、フラボノイド
- **MBT55代謝作用**：
  - **芳香族化合物分解菌・糸状菌** → フラボノイド代謝 → 抗炎症性メタボライト生成
  - **リグニン分解菌・セルロース分解菌** → 植物繊維分解 → SCFA（酢酸・酪酸）生成
  - **硫酸還元菌・メタン酸化菌** → 腸管ガス制御 → 咳・喘息症状の緩和
- **代謝産物の薬理作用**：
  - 気管支拡張（エフェドリン誘導体＋SCFAによる平滑筋調整）
  - 抗炎症（フラボノイド代謝物）
  - 免疫調整（腸内環境改善によるTreg細胞活性）

#### ２. 杏仁（Prunus armeniaca）
- **主成分**：アミグダリン、脂質、タンパク質、微量ミネラル
- **MBT55代謝作用**：
  - **脂質分解菌・タンパク質分解菌** → 脂肪酸・ペプチド生成
  - **鉄酸化菌・マンガン酸化菌** → 微量元素活性化 → 酵素補因子として機能
  - **乳酸菌群・放線菌** → アミグダリン代謝 → 抗ウイルス性代謝物生成
- **代謝産物の薬理作用**：
  - 鎮咳・去痰（脂肪酸誘導体＋アミグダリン代謝物）
  - 抗ウイルス（腸内代謝によるアミグダリン活性化）
  - 腸管バリア強化（脂質代謝物＋SCFA）

#### 3. 桂枝（Cinnamomum cassia）
- **主成分**：桂皮アルデヒド、精油、ポリフェノール
- **MBT55代謝作用**：
  - **芳香族化合物分解菌・糸状菌** → 桂皮アルデヒド代謝 → 抗菌性代謝物生成
  - **酵母菌・硫黄細菌** → 精油成分の酸化 → 抗ウイルス性誘導体
  - **セルロース分解菌・リグニン分解菌** → 食物繊維分解 → SCFA生成
- **代謝産物の薬理作用**：
  - 発汗促進（自律神経調整＋血管拡張）
  - 抗菌・抗ウイルス（桂皮代謝物）
  - 免疫活性（ポリフェノール代謝物）

#### 4. 甘草（Glycyrrhiza uralensis）
- **主成分**：グリチルリチン、フラボノイド、サポニン
- **MBT55代謝作用**：
  - **放線菌・乳酸菌群** → グリチルリチン代謝 → 抗炎症性代謝物生成
  - **硫酸還元菌・鉄還元菌** → サポニン活性化 → 腸管粘膜修復
  - **酵母菌・糸状菌** → フラボノイド代謝 → 抗アレルギー性物質生成
- **代謝産物の薬理作用**：
  - 抗炎症（グリチルリチン代謝物）
  - 粘膜保護（サポニン誘導体）
  - アレルギー抑制（フラボノイド代謝物）

#### 追加説明

1. **方剤学そのものが「代謝ネットワーク前提」で設計されている**  
   漢方はもともと「単一成分」ではなく「複数の生薬が体内でどう響き合うか」を前提に組まれています。  
   しかし現代薬理学は、主に「単一成分の作用」にばかり光を当ててきた。  
   ここに「MBT55による代謝」という第3のレイヤーを乗せることで、  
   - 生薬同士の相互作用  
   - 微生物との相互作用  
   を同時に見に行ける、ほぼ手つかずの研究領域になります。

2. **MBT55が“普通の腸内細菌叢では起こらない代謝”を引き出す可能性がある**  
   一般腸内細菌ではほぼ代謝されない成分  
   （難分解性ポリフェノール、リグニン様構造、鉱物との複合体など）が、  
   MBT55の  
   - リグニン分解菌  
   - 芳香族化合物分解菌（糸状菌）  
   - 鉄・マンガン酸化還元菌  
   などによって**全く別の代謝物**に変換されるかもしれない。  
   これはそのまま「未知の薬理作用」につながる余地です。

3. **方剤ごとの“代謝指紋（metabolite fingerprint）”を作れる**  
   麻黄湯、葛根湯、小柴胡湯…それぞれを MBT55 とインキュベートし、  
   - どのクラスの代謝物がどの比率で生じるか  
   - 炎症系／自律神経系／免疫系／代謝系への寄与はどう違うか  
   を比較すれば、**方剤を「代謝プロファイル」で再定義するカタログ**が作れます。
