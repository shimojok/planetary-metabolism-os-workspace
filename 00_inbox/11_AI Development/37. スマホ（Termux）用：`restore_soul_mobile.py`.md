### 📱 スマホ（Termux）用：`restore_soul_mobile.py`

以下のコードは、下條さんのスマホのパスに合わせて調整してあります。これをTermux上の適当な場所で作成して実行すれば、スマホのObsidian内のファイルが直接書き換わります。

Python

```
import os
import urllib.parse

# --- スマホ用のパス設定 ---
# Termuxから共有ストレージにアクセスするためのパス
BASE_DIR = "/storage/emulated/0/Documents/Obsidian vault"

contents = {
    "10. 状態推論（診断）": """# 状態推論（診断）：今の土壌は何を求めているか？
## 1. 概要
MBT55-HyperCycleを用いた土壌フェノタイピングによる「状態推論」アルゴリズム。
## 2. 推論ロジック
- 微生物叢(Microbiome)と代謝物(Metabolome)の相関解析。
- N2O排出を最小化する炭素源の最適投入タイミングの算出。
""",

    "GP2-1. Green Premium Reverser": """# Green Premium Reverser 設計案
## 1. 概念
脱炭素肥料導入に伴う追加コスト（グリーンプレミアム）を、土壌炭素貯留の価値と医療費削減の価値で相殺・逆転させる仕組み。
## 2. 数理モデル
$$Value_{Total} = \Delta Carbon + \Delta Health - Cost_{Green}$$
""",
    
    # 他のファイルの中身も、ここに順次追加していきます
}

def restore_soul_mobile():
    print(f"--- スマホ版：魂の注入を開始 ---")
    print(f"対象フォルダ: {BASE_DIR}")
    
    # フォルダが存在するか確認
    if not os.path.exists(BASE_DIR):
        print("エラー: Obsidianのフォルダが見つかりません。termux-setup-storage を実行済みですか？")
        return

    # カレントディレクトリにあるURLリストを読み込む
    # ※もしURLリストが別の場所にあるなら、ここを修正してください
    url_list_path = "workspace_all_urls_encoded.txt"
    
    if not os.path.exists(url_list_path):
        print(f"エラー: {url_list_path} が見つかりません。")
        return

    with open(url_list_path, "r", encoding="utf-8") as f:
        urls = f.readlines()

    success_count = 0
    for url in urls:
        url = url.strip()
        filename = urllib.parse.unquote(url.split("/")[-1])
        
        # Obsidian内を再帰的に探索
        for root, dirs, files in os.walk(BASE_DIR):
            if filename in files:
                filepath = os.path.join(root, filename)
                
                content_to_write = ""
                for key, content in contents.items():
                    if key in filename:
                        content_to_write = content
                        break
                
                if content_to_write:
                    with open(filepath, "w", encoding="utf-8") as wf:
                        wf.write(content_to_write)
                    print(f"注入完了: {filename}")
                    success_count += 1

    print(f"\n完了: {success_count} 個のファイルに魂を込めました。")

if __name__ == "__main__":
    restore_soul_mobile()
```

---

### 🛠 スマホでの実行手順

1. **Termuxを起動**し、`termux-setup-storage` を実行してストレージ権限を許可します。
    
2. `workspace_all_urls_encoded.txt` を Termux の今の作業ディレクトリに配置します。
    
3. 上記のコードを `nano restore_soul_mobile.py` などで保存します。
    
4. `python restore_soul_mobile.py` を実行します。
