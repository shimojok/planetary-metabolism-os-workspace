### 🛠️ スマホ専用リポジトリの作成・構成案

以下の手順で進めるのが最も安全でスムーズです。

1. **新しいリポジトリを作成**（GitHubのWebサイトまたはアプリから）
    
    - 名前案：`planetary-metabolism-os-mobile-lab`
        
    - Public設定、READMEあり。
        
2. **スマホ（Termux）にクローン**
    
    Bash
    
    ```
    git clone https://github.com/shimojok/planetary-metabolism-os-mobile-lab.git
    ```
    
3. **Obsidianのファイルをコピー**
    
    スマホのObsidian内にある「ガワ（空のファイル群）」を、この新しい `mobile-lab` フォルダへコピーします。
    

---

### 🔄 なぜこの方法が良いのか？

- **コンフリクトの完全回避**: PC側のメインリポジトリを汚さずに、スマホで「魂の注入（コンテンツ生成）」のテストができます。
    
- **同期のハブ**: スマホで生成した中身が完璧だったら、それをGitHubにPushし、PC側で「必要な部分だけ」をメインに取り込む（Cherry-pickやコピペ）ことができます。
    
- **バックアップ**: スマホを紛失したりObsidianのデータが消えても、GitHubにさえあればいつでも復元可能です。
