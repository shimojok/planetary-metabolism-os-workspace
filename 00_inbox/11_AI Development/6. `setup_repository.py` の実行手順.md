## 1. `setup_repository.py` の実行手順

このスクリプトは、あなたのPC（ローカル）やAzure上の環境で、**「一瞬で複雑なフォルダとファイル構造を自動生成する」**ためのものです。

### ステップ1：Pythonファイルの作成

1. PCのデスクトップなどに、新しいフォルダ（例：`PBPE_Project`）を作ります。
    
2. メモ帳やVS Codeを開き、先ほどのスクリプトを貼り付けます。
    
3. 名前を **`setup_repository.py`** として、そのフォルダ内に保存します。
    

### ステップ2：実行

1. コマンドプロンプト（Windows）またはターミナル（Mac）を開きます。
    
2. 作成したフォルダに移動します。
    
    Bash
    
    ```
    cd Desktop/PBPE_Project
    ```
    
3. スクリプトを実行します。
    
    Bash
    
    ```
    python setup_repository.py
    ```
    

### ステップ3：結果の確認

実行が終わると、フォルダ内に `Planetary-Metabolism-OS`、`AGRIX_Regen_Engine`、`HealthBook_MI_Core` の3つのリポジトリ用ディレクトリが生成されます。
