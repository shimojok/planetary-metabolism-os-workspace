無料で使えますか？

---

はい、**無料で使うことは可能です。**
ただし重要なのは「**どこまで無料か**」です。AIエージェントは通常、次の **3つのコスト要素**で構成されています。

---

# 1. AIエージェントフレームワーク

これは **無料**です。

例

* LangGraph
* CrewAI
* AutoGen
* LangChain

例えば **LangGraph** は **MITライセンスのオープンソースで完全無料**です。([langchain.com][1])

つまり

```text
pip install langgraph
```

で無料で使えます。

さらに

* 商用利用 OK
* GitHub公開 OK
* 改変 OK

です。([Articsledge][2])

---

# 2. AIモデル（ここが主なコスト）

AIエージェントは **LLM API**を使うので、ここが有料になることが多いです。

例

* GPT
* Claude
* Gemini

つまり

```text
AIエージェント
   ↓
LLM API
   ↓
料金
```

です。

ただし

無料枠もあります。

---

# 3. サーバー / クラウド

AIエージェントを動かすには

* サーバー
* データベース
* API

が必要です。

これも

* 自分のPC
* 自分のサーバー

なら **無料**です。

クラウドにすると課金されます。

---

# 4. 完全無料構成（実際に可能）

以下の構成なら **ほぼ無料でAIエージェント開発できます。**

```text
VS Code
   ↓
Python
   ↓
LangGraph
   ↓
Local PC
```

費用

```text
0円
```

です。

---
