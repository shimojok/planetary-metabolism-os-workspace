# BioNexus 6-Agent Architecture

```text
                    ┌─────────────────────┐
                    │   System Orchestrator │
                    │   (Global AI Agent)  │
                    └─────────┬───────────┘
                              │
          ┌───────────────────┼───────────────────┐
          │                   │                   │
          ▼                   ▼                   ▼
   AGRIX Agent         MBT Microbiome Agent   HealthBook Agent
   (Agriculture)        (Microbial System)    (Human Health)

          │                   │                   │
          │                   │                   │
          └───────────────┬───┴───┬───────────────┘
                          │       │
                          ▼       ▼
                 Data Integration Agent
                    (BioNexus Data Hub)

                          │
                          ▼
                Simulation / Prediction Agent
               (Planetary Metabolism Model)
```

# 6つのエージェントの役割

## ① System Orchestrator Agent

```

エージェント連携
ワークフロー管理
```

```
LangGraph Controller
```

BioNexusではデータが次の循環をします。

```
Soil microbiome
      │
      ▼
 AGRIX Agent
      │
      ▼
 Food nutrients
      │
      ▼
 MBT Agent
      │
      ▼
 Human microbiome
      │
      ▼
 HealthBook Agent
      │
      ▼
 Health data
      │
      ▼
 Simulation Agent
      │
      ▼
 Agriculture optimization
```

つまり

```
Soil → Food → Human → Soil
```

です。
