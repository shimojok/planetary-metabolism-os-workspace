## LangGraph とは

**LangGraph** は、LLM（大規模言語モデル）を使ったアプリケーションを
**「複数のAIエージェントが協働するシステム」**として構築するためのフレームワークです。

開発元は **LangChain エコシステム**で、
特に **マルチエージェントAI** を作るために設計されています。

---

# 1. なぜ LangGraph が作られたのか

従来のAIアプリ（例：チャットボット）は

```
User → AI → Response
```

という **直線型ワークフロー**でした。

しかし研究用途や高度なAIでは

```
AI → AI → AI → AI
```

と **複数のAIが協働**します。

例

```
質問
 ↓
Research agent
 ↓
Data agent
 ↓
Analysis agent
 ↓
Answer
```

これを管理するために **LangGraph** が作られました。

---

# 2. 基本概念

LangGraphは名前の通り

```
Graph（グラフ）
```

構造でAIを組みます。

基本要素は3つです。

### Node（ノード）

ノード = エージェント

例

```
AGRIX Agent
MBT Agent
Health Agent
```

---

### Edge（エッジ）

ノード間の接続です。

例

```
AGRIX → MBT
MBT → Health
```

---

### State（状態）

エージェント間で共有されるデータ。

例

```
soil_data
microbiome_data
health_data
```

---

# 3. LangGraphの構造

概念図

```
        ┌──────────┐
        │ Orchestrator │
        └─────┬─────┘
              │
   ┌──────────┼──────────┐
   │          │          │
   ▼          ▼          ▼
AGRIX      MBT        Health
 Agent      Agent       Agent
   │          │          │
   └──────┬───┴───┬──────┘
          ▼       ▼
       Data Agent
          │
          ▼
     Simulation
```

これは **LangGraphの典型的なマルチエージェント構造**です。

---

# 4. LangGraphの特徴

### ① マルチエージェント設計

AI同士が相談できます。

例

```
Health agent
→ MBT agent に質問
→ 腸内細菌データ取得
→ 健康リスク分析
```

---

### ② ループが作れる

普通のAIフローは

```
A → B → C
```

ですが LangGraph は

```
A → B → C
↑       ↓
└───────┘
```

のように **反復推論**できます。

これは研究用途で重要です。

---

### ③ 長期記憶

LangGraphは

* vector database
* knowledge graph

と接続できます。

例

* Pinecone
* Neo4j

---

### ④ 状態管理

複雑な推論では

```
途中結果
```

が重要です。

LangGraphでは

```
State
```

として保持します。

---

# 5. 実際のコード（超簡略）

Pythonで書きます。

```python
from langgraph.graph import StateGraph

workflow = StateGraph()

workflow.add_node("agrix", agrix_agent)
workflow.add_node("mbt", mbt_agent)
workflow.add_node("health", health_agent)

workflow.add_edge("agrix","mbt")
workflow.add_edge("mbt","health")

workflow.set_entry_point("agrix")

app = workflow.compile()
```

これで

```
AGRIX → MBT → Health
```

のエージェント連携が作れます。
