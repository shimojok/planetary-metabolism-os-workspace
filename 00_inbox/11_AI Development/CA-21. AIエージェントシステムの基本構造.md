# AIエージェントシステムの基本構造

```
開発ツール
↓
フレームワーク
↓
AIエージェント
↓
AIモデル
↓
データ・ツール
```

それぞれの役割です。

---

## ① 開発ツール（作る場所）

ここでコードを書きます。

代表例

* Visual Studio Code
* GitHub

これは **研究室の机**のようなものです。

---

## ② フレームワーク（仕組み）

AIエージェントを組み立てる仕組みです。

代表例

* LangGraph
* LangChain

ここで

```
AI同士の連携
ワークフロー
```

を作ります。

---

## ③ AIエージェント（役割AI）

それぞれ **役割を持つAI**です。

例

```
Research Agent
Data Agent
Analysis Agent
Decision Agent
```

Kazさんの構想なら

```
AGRIX Agent
Microbiome Agent
Health Agent
Simulation Agent
```

になります。

---

## ④ AIモデル（知能）

ここが **推論能力そのもの**です。

例

* GPT-4
* Claude
* Gemini

エージェントは **このモデルを使って考えます**。

---

## ⑤ データ・ツール

AIが現実と接続する部分です。

例

```
データベース
論文
センサー
API
```

例

* PostgreSQL
* Neo4j
