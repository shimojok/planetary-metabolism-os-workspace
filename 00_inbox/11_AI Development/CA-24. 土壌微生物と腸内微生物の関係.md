# 1. 土壌微生物と腸内微生物の関係

まず微生物生態学では次の概念があります。

* Soil Microbiome
* Gut Microbiome

これらは別の生態系ですが、**進化的・機能的に非常に近い系統群**を共有しています。

代表的な例：

| 微生物門           | 土壌 | 腸  |
| -------------- | -- | -- |
| Firmicutes     | 多い | 多い |
| Bacteroidetes  | 多い | 多い |
| Actinobacteria | 多い | 多い |

つまり

```
Soil microbiome
≈ evolutionary reservoir
```

になっています。

---

# 2. 代謝系が共通している理由

Kazさんの洞察の核心はここです。

土壌微生物の主要な仕事は

```
有機物分解
↓
発酵
↓
代謝産物生成
```

です。

腸内微生物も同じです。

```
食物繊維分解
↓
発酵
↓
短鎖脂肪酸生成
```

この代謝は

* Fermentation

に基づきます。

---

# 3. 土壌有機物と農産物

Kazさんが言われた

> 土壌有機物と農産物は同じ構成

これはかなり重要なポイントです。

土壌有機物の主要構成：

```
セルロース
ヘミセルロース
リグニン
アミノ酸
有機酸
```

これらは植物組織の構成と同じです。

つまり

```
Plant biomass
↓
Soil organic matter
↓
Food substrate
```

は同じ炭素化学系です。

関連概念

* Soil Organic Matter

---

# 4. 微生物代謝の共通生成物

土壌と腸の両方で生成される重要物質があります。

代表例

| 物質     | 機能       |
| ------ | -------- |
| 酪酸     | 腸上皮エネルギー |
| 酢酸     | 免疫調整     |
| プロピオン酸 | 代謝制御     |

これは

* Short-Chain Fatty Acids

と呼ばれます。
