# 3 数理モデル

微生物群 i の動態

```
dXi/dt = Xi (ri + Σ aij Xj)
```

意味

```
Xi = 微生物量
ri = 自己増殖率
aij = 相互作用係数
```

相互作用

```
共生
競合
代謝供給
```

# 5 AIモデル構造

AIのコアは3つです。

### ① Microbial Interaction Graph

```
node = 微生物
edge = 相互作用
```

例

```
55 nodes
~300 interactions
```

---

### ② Metabolic Network

```
node = 代謝物
edge = 代謝反応
```

例

```
SCFA
CLA
アミノ酸
ポリフェノール代謝
```

---

### ③ Nutrient Cascade

```
soil → plant → food → gut
```

ここがKazさんの理論の核心です。
