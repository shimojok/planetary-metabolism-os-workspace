# 2 微生物ダイナミクス

微生物はロトカ＝ヴォルテラ型で表現します。

土壌微生物

$$
\frac{dS_i}{dt} =
r_i S_i
+
\sum_j a_{ij}S_iS_j
-

d_i S_i
$$

意味

| 項        | 意味    |
| -------- | ----- |
| (r_i)    | 増殖率   |
| (a_{ij}) | 共生/競合 |
| (d_i)    | 減衰    |

MBT55は

**共生係数のネットワーク**

です。

# 3 代謝生成

微生物は代謝物を生成します。

$$
\frac{dM_l}{dt}
=

\sum_i b_{il}S_i
+
\sum_k c_{kl}H_k
-

\lambda_l M_l
$$

意味

| 項           | 意味     |
| ----------- | ------ |
| (b_{il})    | 土壌代謝生成 |
