# 4 LangGraph実装構造

LangGraphでは

**State Graph**

を作ります。

```python
from langgraph.graph import StateGraph

graph = StateGraph()

graph.add_node("soil_microbiome", soil_agent)
graph.add_node("fermentation", fermentation_agent)
graph.add_node("crop_metabolism", crop_agent)
graph.add_node("food_system", food_agent)
graph.add_node("human_microbiome", human_agent)
graph.add_node("health_risk", health_agent)

graph.add_edge("soil_microbiome", "fermentation")
graph.add_edge("fermentation", "crop_metabolism")
graph.add_edge("crop_metabolism", "food_system")
graph.add_edge("food_system", "human_microbiome")
graph.add_edge("human_microbiome", "health_risk")
graph.add_edge("health_risk", "soil_microbiome")
```

ここで

**Hypercycle loop**

が完成します。
