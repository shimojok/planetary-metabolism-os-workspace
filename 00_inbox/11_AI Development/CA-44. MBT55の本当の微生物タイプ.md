# 5 Hypercycle理論

ハイパーサイクルは

* Manfred Eigen

が提唱した理論です。

基本式

[
\frac{dx_i}{dt} =
x_i (k_i x_{i-1} - \phi)
]

意味

| 項   | 意味   |
| --- | ---- |
| x_i | 分子/種 |
| k_i | 触媒係数 |
| φ   | 希釈   |
