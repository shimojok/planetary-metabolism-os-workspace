# HealthBook_MI_Core

**Integrated Metabolic Intelligence Core Engine**  
**食生活・生活習慣・症状・健診データ・MBT55・ファイトケミカル・漢方代謝を統合した次世代ヘルスインテリジェンス基盤**

---

## 🚀 プロジェクト概要

**HealthBook_MI_Core** は、以下の複数の科学領域を統合し、  
個人の健康状態を多面的に解析する **Metabolic Intelligence（MI）エンジン** のコア実装です。

- **浜田式 200 問診システム**
- **137 疾病マトリックス（食生活 × 疾病リスク）**
- **下條式ファイトケミカル分類（2008）**
- **MBT55（Food & Herbal）代謝モデル**
- **漢方方剤ライブラリ（MBT 漢方代謝）**
- **腸内細菌（MBT55 菌数データ）**
- **健康診断データ（血液・画像・代謝指標）**
- **症状チェック（身体症状・精神症状）**

これらを統合し、  
**食生活 → 代謝 → 病態 → 介入（Food / Herbal / Probiotics）**  
という一連の因果モデルを構築するための基盤です。

## 🎯 目的

HealthBook_MI_Core の目的は、以下の 3 層構造の **完全統合モデル** を実装することです。

### 1. **物理モデル（Physical Model）**

- 食事・生活習慣・腸内細菌・代謝経路
- MBT55（タンパク・糖・脂質・セルロース分解菌）
- ファイトケミカルの化学的作用（抗酸化・抗炎症・代謝調整）

### 2. **科学モデル（Scientific Model）**

- 浜田式 30 万人データに基づく問診 → 疾病リスク
- 137 疾病マトリックス
- 健康診断データ（血糖・脂質・肝機能・腎機能・炎症）
- 漢方方剤の代謝作用（MBT 漢方代謝）

### 3. **哲学モデル（Philosophical Model）**

- 「食・代謝・環境・心身」の統合
- Teleodynamic（目的論的）な生命モデル
- 個体の恒常性（Homeostasis）と適応（Adaptation）
- MBT（Metabolic Balance Theory）としての統合理論

## 🧠 コアデータ構造（完全構造）

すべての問診項目は以下の **7 要素** を持つ JSON として実装します。

```json
{
  "id": 2,
  "category": "食習慣",
  "question": "朝食を抜くことがよくある",
  "risk_factors": ["欠食型（朝食抜き）"],
  "metabolic_impact": "糖新生リズムの乱れ → 午前中の低血糖傾向",
  "related_diseases": ["糖尿病", "肥満", "慢性疲労"],
  "weight": 0.85
}
```

この構造により、  
**問診 → 代謝 → 疾病 → 介入**  
が一貫したデータモデルで扱えるようになります。
