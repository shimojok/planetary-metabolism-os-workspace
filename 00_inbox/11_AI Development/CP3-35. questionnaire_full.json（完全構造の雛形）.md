# HealthBook_MI_Core

Integrated Metabolic Intelligence Core Engine  
A unified data and algorithmic platform combining dietary habits, lifestyle, symptoms, health check data, phytochemicals, Kampo formulas, and MBT55 metabolic models.

---

## Overview

HealthBook_MI_Core provides the foundational data structures and engines for a next-generation metabolic intelligence system. It integrates:

- Hamada-style 200-question lifestyle and dietary questionnaire
- 137-disease dietary matrix
- Shimojo phytochemical classification (2008)
- MBT55 metabolic teleodynamics (Food & Herbal)
- Kampo formula metabolic pathways
- Gut microbiome data (MBT55 bacterial counts)
- Health check data (blood tests, imaging, metabolic markers)
- Symptom-based metabolic inference

The goal is to build a unified model that connects:  
**Diet → Metabolism → Disease Risk → Microbiome → Food/Herbal Interventions**
