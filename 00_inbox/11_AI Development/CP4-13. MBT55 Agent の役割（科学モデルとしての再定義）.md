### 1. MBT55 Agent の役割（科学モデルとしての再定義）

あなたの定義を、あえて「エージェントの責務」として言い換えると：

> **MBT55 は、微生物コンソーシアムが物質・エネルギー・電子・情報・元素の流れを変換し、生態系の創発秩序を再構成する生体変換エンジンである。**（原文より）

MBT55 Agent は、この定義を**数式とシミュレーション API に落とし込んだもの**です。

- 入力：  
  - 土壌条件（SOC, C/N, pH, 水分, 温度）  
  - 有機物投入量・組成  
  - MBT55 投入量・条件  

- 出力：  
  - 炭素固定量（SOC 増加）  
  - GHG 削減量（CO₂, CH₄, N₂O）  
  - 栄養カスケード（C/N/P/微量元素の再配分）  
  - ハイパーサイクル指標（自己触媒性・安定性）

### 2. 数学モデルの最小コア（ここから実装する）

#### 2-1. 炭素ダイナミクス（SOC モデル）

AGRIX_Theory で既に提示されている式を、MBT55 Agent の中核に据えます。

土壌有機炭素 \( C_s \) の時間変化：

\[
\frac{dC_s}{dt} = I - k_d \cdot C_s + f_m(\text{MBT55}) \cdot \eta
\]

- \( C_s \)：SOC（t-C/ha）  
- \( I \)：有機物供給量  
- \( k_d \)：分解定数  
- \( f_m(\text{MBT55}) \)：MBT55 による炭素安定化関数（0–1）  
- \( \eta \)：電子再固定効率（0.2–0.8）

**MBT55 Agent の責務：**  
- \( f_m \) と \( \eta \) を、**微生物ギルド構成・環境条件から計算する関数**として実装する。

---

#### 2-2. 電子フロー（Electron Flow）モデル

電子フロー \( J_{e^-} \)：

\[
J_{e^-} = \sum_i \left( k_i \cdot [S_i] \cdot \phi_i \right)
\]

- \( k_i \)：微生物群 i の代謝速度  
- \( [S_i] \)：基質濃度  
- \( \phi_i \)：経路効率係数  

**MBT55 Agent の責務：**

- ギルド別（でんぷん分解菌・タンパク分解菌・脂質分解菌・セルロース分解菌）の  
  \( k_i, [S_i], \phi_i \) をパラメータ化し、  
  **電子フローを数値として返す関数**を持つ。

---

#### 2-3. 炭素固定効率（\( C_f \)）モデル

\[
C_f = \lambda \cdot \frac{E_d}{E_d + K_m}
\]

- \( E_d \)：電子供与体生成効率  
- \( \lambda \)：最大固定率  
- \( K_m \)：電子飽和定数  

**MBT55 Agent の責務：**

- 電子供与体生成効率 \( E_d \) を、  
  - 有機物組成  
  - 微生物ギルド構成  
  - 環境条件  
 から計算し、**炭素固定効率 \( C_f \) を返す関数**として実装。

---

#### 2-4. ハイパーサイクル指標（自己触媒性）

シンプルな最初の指標として：

\[
H = \alpha \cdot C_f + \beta \cdot D_m + \gamma \cdot R_b
\]

- \( H \)：ハイパーサイクル強度指標  
- \( C_f \)：炭素固定効率  
- \( D_m \)：微生物多様性指標（Shannon など）  
- \( R_b \)：Redox バランス指標（Eh の安定度）  

**MBT55 Agent の責務：**

- 入力条件から \( H \) を計算し、  
  **「この条件で MBT55 を入れると、どれだけ自己触媒的な循環が立ち上がるか」**を数値で返す。

### 3. MBT55 Agent の API 設計（関数一覧）

VS Code / Python で実装することを前提に、**最初のバージョンの API** をこう置きます。

#### 3-1. データ構造

```python
from dataclasses import dataclass
from typing import Dict

@dataclass
class SoilState:
    soc: float          # t-C/ha
    cn_ratio: float
    ph: float
    moisture: float
    temperature: float

@dataclass
class OrganicInput:
    carbon: float       # t-C/ha/year
    nitrogen: float
    lignin_ratio: float
    lipid_ratio: float
    protein_ratio: float
    starch_ratio: float

@dataclass
class MBT55Params:
    kd: float           # 分解定数
    lambda_cf: float    # 最大炭素固定率
    km_ed: float        # 電子飽和定数
    guild_activity: Dict[str, float]  # {'starch':..., 'protein':..., ...}
```
