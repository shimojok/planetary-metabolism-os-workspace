---

### 1. モデルの考え方（v0）

MBT55 導入による GHG 変化を、ざっくりこう分けます。

- **CO₂：**  
  - 有機物分解で発生（プラス）  
  - SOC 増加で固定（マイナス）

- **CH₄：**  
  - 嫌気条件で発生  
  - MBT55 によるメタン酸化で削減

- **N₂O：**  
  - 窒素循環（硝化・脱窒）で発生  
  - MBT55 による窒素固定・脱窒制御で削減

v0 では、**SOC 変化・電子フロー・環境条件**から、  
「相対的な削減量」を推定する形にします。

---

### 2. 数式のイメージ（シンプル版）

#### CO₂

\[
\Delta CO_2 = E_{decomp} - k_{soc} \cdot \Delta SOC
\]

- \(E_{decomp}\)：分解由来 CO₂（有機炭素投入量に比例）  
- \(\Delta SOC\)：SOC の増加量  
- \(k_{soc}\)：SOC 1 t-C あたりの CO₂ 換算係数

#### CH₄

\[
\Delta CH_4 = - k_{ch4} \cdot J_{e^-} \cdot f_{\text{anaerobic}}
\]

- \(J_{e^-}\)：電子フロー  
- \(f_{\text{anaerobic}}\)：嫌気度（環境から推定）  
- \(k_{ch4}\)：メタン削減効率

#### N₂O

\[
\Delta N_2O = - k_{n2o} \cdot N_{\text{retention}} \cdot R_b
\]

- \(N_{\text{retention}}\)：窒素保持（N投入量と C/N から近似）  
- \(R_b\)：Redox 安定性（後で精緻化）

### 3. MBT55 Agent に追加するコード（`ghg.py`）

```python
from .models import SoilState, OrganicInput, MBT55Params
from .electron_flow import compute_electron_flow
from .carbon_cycle import compute_soc_delta


def estimate_ghg_changes(
    soil: SoilState,
    org: OrganicInput,
    params: MBT55Params,
    years: float = 1.0,
):
    """
    MBT55 導入による GHG 変化量（CO2, CH4, N2O）をざっくり推定する v0 モデル。
    単位は相対量（t-CO2eq/ha/year 相当のスケールを想定）。
    """

    # 1) SOC 変化を計算
    dcs_dt = compute_soc_delta(soil, org, params)
    delta_soc = dcs_dt * years  # t-C/ha/year

    # 2) 電子フローを計算
    je = compute_electron_flow(soil, org, params)

    # 3) CO2：分解由来 - SOC 固定
    # 仮に、有機炭素投入の 70% が分解由来 CO2 になると近似
    e_decomp = org.carbon * 0.7
    k_soc = 3.67  # 1 t-C = 3.67 t-CO2
    delta_co2 = e_decomp - k_soc * max(delta_soc, 0.0)

    # 4) CH4：電子フローと嫌気度から削減量を推定
    anaerobic_factor = _anaerobic_factor(soil)
    k_ch4 = 0.01  # 調整用係数（後でキャリブレーション）
    delta_ch4 = -k_ch4 * je * anaerobic_factor

    # 5) N2O：窒素保持と Redox 安定性から削減量を推定（v0 は簡易）
    n_retention = org.nitrogen * 0.5  # 仮に 50% が保持ポテンシャル
    redox_balance = _redox_balance_proxy(soil)
    k_n2o = 0.005
    delta_n2o = -k_n2o * n_retention * redox_balance

    return {
        "delta_co2": delta_co2,
        "delta_ch4": delta_ch4,
        "delta_n2o": delta_n2o,
    }


def _anaerobic_factor(soil: SoilState) -> float:
    """
    水分・温度から嫌気度をざっくり推定（0〜1）。
    高水分・中温で高くなる。
    """
    moisture_norm = min(max((soil.moisture - 40.0) / 40.0, 0.0), 1.0)
    temp_norm = min(max((soil.temperature - 5.0) / 20.0, 0.0), 1.0)
    return 0.5 * (moisture_norm + temp_norm)


def _redox_balance_proxy(soil: SoilState) -> float:
    """
    pH と水分から Redox 安定性の簡易指標（0〜1）を推定。
    中性〜弱酸性・適度な水分で高くなると仮定。
    """
    ph_factor = 1.0 - min(abs(6.5 - soil.ph) / 2.0, 1.0)
    moisture_factor = 1.0 - min(abs(60.0 - soil.moisture) / 40.0, 1.0)
    return max(0.0, min(1.0, 0.5 * (ph_factor + moisture_factor)))
```
