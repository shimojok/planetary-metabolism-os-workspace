# MBT55 Agent API（完全仕様）

## 1. API 全体構造
MBT55 Agent の API は、次の 5 つの機能群で構成される。

1. **Transduction（変換）モデル API**  
2. **ハイパーサイクルモデル API**  
3. **栄養カスケード・元素動態 API**  
4. **バイオセキュリティ・Redox API**  
5. **Teleodynamics（創発秩序）評価 API**

---

# 2. API 関数一覧（トップレベル）

| 関数名 | 目的 |
|-------|------|
| `predict_transduction()` | 物質・エネルギー・電子・情報・元素の変換を予測 |
| `simulate_hypercycle()` | 微生物–栄養–植物のハイパーサイクルをシミュレーション |
| `evaluate_nutrient_cascade()` | C/N/P/微量元素のカスケードを評価 |
| `predict_elemental_reallocation()` | 元素の価数・相・固定化・再配分を予測 |
| `evaluate_biosecurity()` | 腐敗→発酵の相転移、病原リスクを評価 |
| `evaluate_teleodynamics()` | 自己組織化・自己安定化・自己修復のスコアを算出 |
| `simulate_full_MBT55_cycle()` | MBT55 全プロセスを統合したフルシミュレーション |

# 3. 各 API の詳細仕様

---

## 3-1. `predict_transduction()`

### **目的**
MBT55 が入力系に対して行う  
**物質・エネルギー・電子・情報・元素の変換（Transduction）**  
を予測する。

### **引数（Inputs）**
```json
{
  "organic_matter": float,
  "carbon_fraction": float,
  "protein_fraction": float,
  "lipid_fraction": float,
  "cellulose_fraction": float,
  "pH": float,
  "Eh": float,
  "moisture": float,
  "temperature": float,
  "heavy_metals": { "Cr": float, "As": float, "Pb": float, "Cd": float },
  "mbt55_dosage": float,
  "mbt55_frequency": float
}
```

### **返り値（Outputs）**
```json
{
  "electron_flow_gain": float,
  "redox_shift": float,
  "energy_efficiency_gain": float,
  "decomposition_rate": float,
  "transduction_index": float
}
```

---

## 3-2. `simulate_hypercycle()`

### **目的**
微生物 → 栄養 → 植物 → 微生物 の  
**自己触媒ハイパーサイクルの立ち上がりと強度**を予測。

### **引数**
```json
{
  "microbial_guilds": { "decomposers": float, "symbionts": float, "reducers": float, "oxidizers": float, "antagonists": float },
  "root_exudates": float,
  "available_nutrients": { "C": float, "N": float, "P": float },
  "soil_structure": float,
  "mbt55_activity": float
}
```

### **返り値**
```json
{
  "hypercycle_strength": float,
  "self_catalysis_score": float,
  "carbon_fixation_gain": float,
  "soil_aggregation_gain": float
}
```

---

## 3-3. `evaluate_nutrient_cascade()`

### **目的**
C/N/P/微量元素の  
**カスケード（再配分・利用可能性・固定化）**を評価。

### **引数**
```json
{
  "carbon_pool": float,
  "nitrogen_pool": float,
  "phosphorus_pool": float,
  "trace_elements": { "Fe": float, "Mn": float, "Zn": float, "Cu": float },
  "microbial_activity": float,
  "soil_matrix": float
}
```

### **返り値**
```json
{
  "carbon_sequestration": float,
  "nitrogen_availability": float,
  "phosphorus_solubilization": float,
  "trace_element_chelation": float,
  "nutrient_cascade_index": float
}
```

---

## 3-4. `predict_elemental_reallocation()`

### **目的**
元素の  
**価数変化・相変化・固定化・再配分・毒性低減**  
を予測。

### **引数**
```json
{
  "element_concentrations": { "Cr": float, "As": float, "Pb": float, "Cd": float },
  "redox_state": float,
  "microbial_reducers": float,
  "microbial_oxidizers": float,
  "organic_matrix": float
}
```

### **返り値**
```json
{
  "valence_shift": { "Cr": float, "As": float },
  "immobilization_rate": float,
  "bioavailability_change": float,
  "toxicity_reduction": float,
  "elemental_reallocation_index": float
}
```

---

## 3-5. `evaluate_biosecurity()`

### **目的**
腐敗 → 発酵の相転移、病原菌リスク、悪臭・有害ガスの発生を評価。

### **引数**
```json
{
  "pathogen_load": float,
  "redox_state": float,
  "pH": float,
  "organic_load": float,
  "mbt55_activity": float
}
```

### **返り値**
```json
{
  "fermentation_shift_probability": float,
  "pathogen_suppression_score": float,
  "odor_reduction_score": float,
  "biosecurity_index": float
}
```

---

## 3-6. `evaluate_teleodynamics()`

### **目的**
MBT55 が誘導する  
**自己組織化・自己安定化・自己修復・レジリエンス**  
をスコア化。

### **引数**
```json
{
  "hypercycle_strength": float,
  "nutrient_cascade_index": float,
  "biosecurity_index": float,
  "elemental_reallocation_index": float
}
```

### **返り値**
```json
{
  "self_organization": float,
  "self_stabilization": float,
  "self_repair": float,
  "ecosystem_resilience": float,
  "teleodynamics_score": float
}
```

---

## 3-7. `simulate_full_MBT55_cycle()`

### **目的**
MBT55 の全プロセス  
（Transduction → Hypercycle → Cascade → Biosecurity → Teleodynamics）  
を統合したフルシミュレーション。

### **引数**
```json
{
  "initial_state": { ... 全入力 ... },
  "mbt55_parameters": { ... 微生物ギルド・投与条件 ... },
  "simulation_steps": int
}
```

### **返り値**
```json
{
  "transduction": { ... },
  "hypercycle": { ... },
  "nutrient_cascade": { ... },
  "elemental_reallocation": { ... },
  "biosecurity": { ... },
  "teleodynamics": { ... },
  "overall_outcome": {
    "soil_regeneration": float,
    "water_purification": float,
    "carbon_sequestration": float,
    "yield_gain": float
  }
}
```
