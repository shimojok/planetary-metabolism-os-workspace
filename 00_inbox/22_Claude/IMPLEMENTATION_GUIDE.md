## 📝 File Creation Guidelines

### For Each Case Study Directory:

**1. README.md** (Main file, follows template)
- Executive Summary
- Problem Statement
- MBT55 Solution
- Results (quantified)
- Video Evidence
- Data Files
- Economic Analysis
- Scientific Validation

**2. results-data.json** (Structured data)
- Use data-template.json
- Fill in all relevant fields
- Include measurement methods
- Document statistical significance

**3. video-links.md** (Video documentation)
```markdown
# Video Evidence for [Case Study Name]

## Overview Video
- **Title**: [Title]
- **Platform**: [YouTube/LinkedIn]
- **URL**: [Full URL]
- **Duration**: [MM:SS]
- **Date Recorded**: [YYYY-MM-DD]
- **Description**: [What viewers will see]

## Key Timestamps
- 00:00 - [Description]
- 01:30 - [Description]
- ...

## Additional Videos
[List other related videos]
```

**4. images/** (Visual evidence)
- Naming convention: `before-001.jpg`, `after-001.jpg`, `process-001.jpg`
- Include captions file: `image-captions.txt`
- Charts/graphs: descriptive names (e.g., `yield-comparison-2023.png`)

## 💻 Code for Automation (Optional)

### Script to Generate New Case Study

```python
#!/usr/bin/env python3
"""
Script to scaffold a new video-proof case study directory
Usage: python create_case_study.py "Case Study Name" "02-directory-name"
"""

import os
import sys
import shutil
from pathlib import Path

def create_case_study(name: str, dir_name: str):
    """Create a new case study directory with template files"""
    
    base_path = Path("evidence/video-proof")
    case_path = base_path / dir_name
    
    # Create directory structure
    case_path.mkdir(parents=True, exist_ok=True)
    (case_path / "images").mkdir(exist_ok=True)
    
    # Copy templates
    template_md = base_path / "_templates" / "application-template.md"
    template_json = base_path / "_templates" / "data-template.json"
    
    shutil.copy(template_md, case_path / "README.md")
    shutil.copy(template_json, case_path / "results-data.json")
    
    # Create video-links.md
    video_template = f"""# Video Evidence for {name}

## Overview Video
- **Title**: 
- **Platform**: 
- **URL**: 
- **Duration**: 
- **Date Recorded**: 
- **Description**: 

## Key Timestamps
- 00:00 - 
- 01:30 - 

## Additional Videos
[List other related videos]
"""
    
    with open(case_path / "video-links.md", "w") as f:
        f.write(video_template)
    
    print(f"✅ Created case study directory: {case_path}")
    print(f"📝 Files created:")
    print(f"   - README.md (from template)")
    print(f"   - results-data.json (from template)")
    print(f"   - video-links.md")
    print(f"   - images/ (empty directory)")
    print(f"\nNext steps:")
    print(f"1. Edit README.md with case-specific information")
    print(f"2. Fill in results-data.json with actual data")
    print(f"3. Add video URLs to video-links.md")
    print(f"4. Add images to images/ directory")

if __name__ == "__main__":
    if len(sys.argv) != 3:
        print("Usage: python create_case_study.py 'Case Study Name' '02-directory-name'")
        sys.exit(1)
    
    create_case_study(sys.argv[1], sys.argv[2])
```
