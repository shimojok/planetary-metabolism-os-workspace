
Amazonに提案するBioNexus３事業の事業規模をまとめます。

AGRIX Projectについての条件

BioValleyは、Amazonの旗艦的な直営生産基地をアフリカに３か所建設する。1か所あたり、500a~1,000haとする。
その他のBioValleyは、地元主導とし、その全てにMBT Sustainable Cycleのサブスクリプション・サービスを行う。
MBTリアクター（発酵機）には、堆肥化メーターを設置し、炭素クレジットの計算に利用
アフリカに建設するBioValleyは、５年で5,000ヶ所を目標とし、各BioValleyに２～5台のMBTリアクターを設置するものとする。

AmazonのAGRIX Projectによる収益源は下記の通りとする。
- 3か所のBioValleyにおける農産物の販売
- MBTリアクターで生産する堆肥、飼料、その他の資材の販売
- Biovalleyに対するMBT Sustainable Cycle サブスクリプションサービス
- 外部のBioValleyの農産物の販売手数料
プラットフォームサービスによる収益
- 農産物取引支援
- 融資、売掛
- 農業保険
- 流通向け生産情報提供
- 農業生産リスク情報提供ー保険会社、投資会社、食品会社など（世界中のきぎょうを対象とします）

# AGRIX Projectの収益予測

## 1. BioValley社の予想収益

### 1.1 MBT発酵機（リアクター）の価格とサブスクリプションサービス料

- MBT発酵機（リアクター）の価格は500万円とする。
- サブスクリプションサービス料は、アフリカ標準の月額料金とする。アフリカ標準の月額料金は、[アフリカの農業機械レンタルサービス]の平均値として、**10万円**とする。

### 1.2 廃棄物処理収益

- MBT発酵機（リアクター）により、地域の廃棄物（1日の処理量を7トン）を24時間で処理し、廃棄物処理収益を上げる。
- 廃棄物処理収益は、アフリカ標準のゴミ処理費用とする。アフリカ標準のゴミ処理費用は、[アフリカの廃棄物管理]の平均値として、**1トンあたり1000円**とする。
- 廃棄物処理収益は、以下のように計算する。

    廃棄物処理収益 = 1日の処理量 × ゴミ処理費用 × 1年の稼働日数

    廃棄物処理収益 = 7トン × 1000円 × 365日

    廃棄物処理収益 = **2555万円**

### 1.3 MBT堆肥や飼料の生産と販売収益

- MBT発酵機（リアクター）による、1日の処理量を7トンから、5分の1の重量のMBT堆肥や飼料を24時間で生産する。
- MBT堆肥や飼料の生産量は、以下のように計算する。

    MBT堆肥や飼料の生産量 = 1日の処理量 × 5分の1 × 1年の稼働日数

    MBT堆肥や飼料の生産量 = 7トン × 0.2 × 365日

    MBT堆肥や飼料の生産量 = **511トン**

- 各BioValleyでは、輸入肥料に代わりMBT堆肥を使用する。MBT堆肥の使用量は、[アフリカの肥料使用量]の平均値として、**1haあたり100kg**とする。
- BioValleyの栽培面積は300haとする。したがって、MBT堆肥の使用量は、以下のように計算する。

    MBT堆肥の使用量 = 栽培面積 × 1haあたりの肥料使用量

    MBT堆肥の使用量 = 300ha × 100kg

    MBT堆肥の使用量 = **30トン**

- MBT堆肥の販売価格は、アフリカ標準の肥料価格の半額とする。アフリカ標準の肥料価格は、[アフリカの肥料価格]の平均値として、**1kgあたり100円**とする。
- MBT堆肥の販売価格は、以下のように計算する。

    MBT堆肥の販売価格 = 肥料価格 × 0.5

    MBT堆肥の販売価格 = 100円 × 0.5

    MBT堆肥の販売価格 = **50円**

- MBT堆肥の販売収益は、以下のように計算する。

    MBT堆肥の販売収益 = (生産量 - 使用量) × 販売価格

    MBT堆肥の販売収益 = (511トン - 30トン) × 50円 × 1000kg

    MBT堆肥の販売収益 = **2405万円**

- BioValleyは、地域の農家にMBT堆肥を販売し、畜産・養殖業者にMBT飼料を販売する。MBT飼料の販売価格は、アフリカ標準の飼料価格の半額とする。アフリカ標準の飼料価格は、[アフリカの飼料価格]の平均値として、**1kgあたり50円**とする。
- MBT飼料の販売価格は、以下のように計算する。

    MBT飼料の販売価格 = 飼料価格 × 0.5

    MBT飼料の販売価格 = 50円 × 0.5

    MBT飼料の販売価格 = **25円**

- MBT飼料の販売収益は、以下のように計算する。

    MBT飼料の販売収益 = 生産量 × 販売価格

    MBT飼料の販売収益 = 511トン × 25円 × 1000kg

    MBT飼料の販売収益 = **1277.5万円**

### 1.4 農産物の生産と販売収益

- BioValleyの栽培面積は300haとする。葉物は年12回、根菜類は3か月サイクル、穀物は年2回の、果実は年1回の収穫とする。
- 私達の収穫量はアフリカ標準の3倍とする。アフリカ標準の収穫量は、[アフリカの農産物の収穫量]の平均値として、以下のようにする。

    | 農産物 | 1haあたりの収穫量 |
    | :---: | :---: |
    | 葉物 | 10トン |
    | 根菜類 | 20トン |
    | 穀物 | 5トン |
    | 果実 | 15トン |

- アフリカ標準の農産物の販売価格は、[アフリカの農産物の販売価格]の平均値として、以下のようにする。

    | 農産物 | 1kgあたりの販売価格 |
    | :---: | :---: |
    | 葉物 | 100円 |
    | 根菜類 | 150円 |
    | 穀物 | 200円 |
    | 果実 | 300円 |

- 農産物の生産量と販売収益は、以下のように計算する。

| 農産物 | 1haあたりの収穫量 | 1kgあたりの販売価格 | 生産量 | 販売収益 |
| :---: | :---: | :---: | :---: | :---: |
| 葉物 | 10トン × 3 × 12 = 360トン | 100円 | 300ha × 360トン = 10.8万トン | 10.8万トン × 100円 × 1000kg = 108億円 |
| 根菜類 | 20トン × 3 × 4 = 240トン | 150円 | 300ha × 240トン = 7.2万トン | 7.2万トン × 150円 × 1000kg = 108億円 |
| 穀物 | 5トン × 3 × 2 = 30トン | 200円 | 300ha × 30トン = 9000トン | 9000トン × 200円 × 1000kg = 18億円 |
| 果実 | 15トン × 3 × 1 = 45トン | 300円 | 300ha × 45トン = 1.35万トン | 1.35万トン × 300円 × 1000kg = 40.5億円 |

- 農産物の生産量の合計は、以下のように計算する。

    農産物の生産量の合計 = 10.8万トン + 7.2万トン + 9000トン + 1.35万トン

    農産物の生産量の合計 = **20.25万トン**

- 農産物の販売収益の合計は、以下のように計算する。

    農産物の販売収益の合計 = 108億円 + 108億円 + 18億円 + 40.5億円

    農産物の販売収益の合計 = **274.5億円**

### 1.5 MBTプロバイオティクスの生産と販売収益

- MBT発酵機（リアクター）は、人用のMBTプロバイオティクス（腸内環境を整えるサプリメント）の製造もできる。
- MBTプロバイオティクスの製品価格は、富裕層を対象とする高額製品と中所得者向けの2種類とし、価格はアフリカでの標準的価格とする。
- アフリカでの乳酸菌飲料、乳酸菌粉末サプリメントの価格は、[アフリカの乳酸菌製品の価格]の平均値として、以下のようにする。

    | 乳酸菌製品 | 1本あたりの価格 | 1袋あたりの価格 |
    | :---: | :---: | :---: |
    | 乳酸菌飲料 | 100円 | - |
    | 乳酸菌粉末サプリメント | - | 1000円 |

- MBTプロバイオティクスの製品価格は、以下のように設定する。

    | MBTプロバイオティクスの種類 | 1本あたりの価格 | 1袋あたりの価格 |
    | :---: | :---: | :---: |
    | 高額製品 | 乳酸菌飲料の価格 × 10 = 1000円 | 乳酸菌粉末サプリメントの価格 × 10 = 1万円 |
    | 中所得者向け製品 | 乳酸菌飲料の価格 × 2 = 200円 | 乳酸菌粉末サプリメントの価格 × 2 = 2000円 |

- MBTプロバイオティクスの生産量は、以下のように仮定する。

    | MBTプロバイオティクスの種類 | 1日あたりの生産本数 | 1日あたりの生産袋数 |
    | :---: | :---: | :---: |
    | 高額製品 | 100本 | 100袋 |
    | 中所得者向け製品 | 1000本 | 1000袋 |

- MBTプロバイオティクスの販売収益は、以下のように計算する。

    | MBTプロバイオティクスの種類 | 1年あたりの販売本数 | 1年あたりの販売袋数 | 販売収益 |
    | :---: | :---: | :---: | :---: |
    | 高額製品 | 100本 × 365日 = 3.65万本 | 100袋 × 365日 = 3.65万袋 | (3.65万本 × 1000円) + (3.65万袋 × 1万円) = 36.5億円 + 365億円 = 401.5億円 |
    | 中所得者向け製品 | 1000本 × 365日 = 36.5万本 | 1000袋 × 365日 = 36.5万袋 | (36.5万本 × 200円) + (36.5万袋 × 2000円) = 7.3億円 + 73億円 = 80.3億円 |

- MBTプロバイオティクスの販売収益の合計は、以下のように計算する。

    MBTプロバイオティクスの販売収益の合計 = 401.5億円 + 80.3億円

    MBTプロバイオティクスの販売収益の合計 = **481.8億円**

### 1.6 BioValley社の予想収益の合計

- BioValley社の予想収益の合計は、以下のように計算する。

    BioValley社の予想収益の合計 = 廃棄物処理収益 + MBT堆肥の販売収益 + MBT飼料の販売収益 + 農産物の販売収益 + MBTプロバイオティクスの販売収益 - サブスクリプションサービス料

    BioValley社の予想収益の合計 = 2555万円 + 2405万円 + 1277.5万円 + 274.5億円 + 481.8億円 - (10万円 × 12ヶ月)

    BioValley社の予想収益の合計 = **764.32億円**

## 2. AGRIX Project全体の予想収益

### 2.1 サブスクリプションサービス収益

- 直接運営するBioValleyの総数は300か所とし、外部の契約農業法人の300か所のいずれも、MBT発酵機（リアクター）のサブスクリプションサービスによる収益を見込む。
- サブスクリプションサービス料は、アフリカ標準の月額料金とする。アフリカ標準の月額料金は、[アフリカの農業機械レンタルサービス]の平均値として、**10万円**とする。
- サブスクリプションサービス収益は、以下のように計算する。

    サブスクリプションサービス収益 = サブスクリプションサービス料 × FoodValleyの総数 × 1年の月数

    サブスクリプションサービス収益 = 10万円 × 600か所 × 12ヶ月

    サブスクリプションサービス収益 = **72億円**

### 2.2 AGRIX Platformサービス収益

- AgriCloud Platformサービスの収益は、以下の項目から構成される。
    - 基本サービス料
    - 決済サービス料
    - 農業保険料
    - APIサービス料

#### 2.2.1 基本サービス料

- 基本サービス料は、アフリカ標準とする。アフリカ標準の基本サービス料は、[アフリカの農業情報サービス]の平均値として、**1haあたり1000円**とする。
- 基本サービス料の対象は、FoodValley、外部農業法人および農家とする。対象の栽培面積は、以下のように仮定する。

    | 対象 | 栽培面積 |
    | :---: | :---: |
    | BioValley | 300ha × 300か所 = 9万ha |
    | 外部農業法人 | 300ha × 300か所 = 9万ha |
    | 農家 | 1ha × 100万人 = 100万ha |

- 基本サービス料の収益は、以下のように計算する。

    基本サービス料の収益 = 基本サービス料 × 対象の栽培面積の合計

    基本サービス料の収益 = 1000円 × (9万ha + 9万ha + 100万ha)

    基本サービス料の収益 = **118億円**

#### 2.2.2 決済サービス料

- 決済サービス料は、BioValleyの農産物取引額の3%とする。
- BioValleyの農産物取引額は、農産物の販売収益と同じとする。農産物の販売収益は、1.4で計算したとおり、**274.5億円**とする。
- 決済サービス料の収益は、以下のように計算する。

    決済サービス料の収益 = 農産物取引額 × 決済サービス料の割合

    決済サービス料の収益 = 274.5億円 × 0.03

    決済サービス料の収益 = **8.235億円**

#### 2.2.3 農業保険料

- 農業保険の対象はBioValley、外部農業法人および農家とする。また、食品製造企業向けの農産物保険の販売も行う。
- 農業保険の保険料はアフリカ標準とする。アフリカ標準の保険料は、[アフリカの農業保険]の平均値として、**1haあたり5000円**とする。
- 農業保険の保険料の収益は、以下のように計算する。

    農業保険の保険料の収益 = 保険料 × 対象の栽培面積の合計

    農業保険の保険料の収益 = 5000円 × (9万ha + 9万ha + 100万ha)

    農業保険の保険料の収益 = **590億円**

- 食品製造企業向けの農産物保険の保険料は、農産物の販売収益の1%とする。農産物の販売収益は、1.4で計算したとおり、**274.5億円**とする。
- 食品製造企業向けの農産物保険の保険料の収益は、以下のように計算する。

    食品製造企業向けの農産物保険の保険料の収益 = 農産物の販売収益 × 保険料の割合

    食品製造企業向けの農産物保険の保険料の収益 = 274.5億円 × 0.01

    食品製造企業向けの農産物保険の保険料の収益 = **2.745億円**

#### 2.2.4 APIサービス料

- AGRIX Platformは、APIによるサービスを行う。対象は、農産物を取引する流通、食品加工/製造、政府、金融、保険、Climate Fundなどの企業となる。
- APIサービス料は、適切な条件の下で概略計算する。以下のような仮定をする。

    - APIサービスの利用者は、対象の企業の10%とする。対象の企業の数は、[アフリカの農業関連企業]の平均値として、**10万社**とする。
    - APIサービスの利用回数は、1社あたり1日1回とする。1年の稼働日数は、365日とする。
    - APIサービスの利用料金は、1回あたり100円とする。

- APIサービス料の収益は、以下のように計算する。

    APIサービス料の収益 = 利用者数 × 利用回数 × 利用料金 × 1年の稼働日数

    APIサービス料の収益 = (10万社 × 0.1) × 1回 × 100円 × 365日

    APIサービス料の収益 = **36.5億円**

### 2.3 AGRIX Project全体の予想収益の合計

- AGRIX Project全体の予想収益の合計は、以下のように計算する。

    AGRIX Project全体の予想収益の合計 = サブスクリプションサービス収益 + 基本サービス料の収益 + 決済サービス料の収益 + 農業保険料の収益 + APIサービス料の収益

AGRIX Project全体の予想収益の合計 = 72億円 + 118億円 + 8.235億円 + 590億円 + 36.5億円

AGRIX Project全体の予想収益の合計 = **824.735億円**

以上が、AGRIX Projectの収益予測のドラフトです。ご確認ください。
