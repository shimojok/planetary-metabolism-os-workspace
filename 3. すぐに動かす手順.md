## すぐに動かす手順

```bash
# 1. プロジェクトフォルダへ移動
cd PMOS_v2

# 2. 仮想環境作成
python -m venv venv && source venv/bin/activate

# 3. パッケージインストール
pip install -r requirements.txt

# 4. まずデモモードで動作確認（API不要）
python agents/orchestrator.py --demo

# 5. .env を設定して本番実行
cp .env.example .env
# → .env を開いて OPENAI_API_KEY と VAULT_PATH を入力

python agents/orchestrator.py --vault /your/vault/path --limit 100
```
