## 具体的なセットアップ手順

```bash
# 現在の状況を確認（Vaultの場所を特定）
# Mac の場合、たいてい以下のいずれか：
ls ~/Documents/ObsidianVault
ls ~/ObsidianVault
ls ~/Desktop/MyVault

# ① Vaultの隣にPMOS_v2を配置（推奨）
cd ~/Documents          # Vaultと同じ階層へ移動
mkdir PMOS_v2           # コードフォルダを作成
cd PMOS_v2

# ② ダウンロードしたファイルをここに置く
# （今回ダウンロードした .py ファイルをすべてここへ）

# ③ .env のVaultパスを「相対パス」で指定
echo 'VAULT_PATH=../MyVault' >> .env
# または絶対パスで
echo 'VAULT_PATH=/Users/あなた/Documents/MyVault' >> .env
```

---

## Vaultの実際の場所の調べ方

Obsidianを開いて、左下の「保管庫を開く」または設定から確認できます。Macなら `⌘ + ,` → 「保管庫」タブで絶対パスが表示されます。

Vaultの正確なパスが分かったら教えていただければ、`.env` の `VAULT_PATH=` に入れる値を具体的にお伝えします！
