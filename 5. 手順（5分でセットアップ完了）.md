現在、VS Codeは、Bionexus_vaultを見ており。PMOS_12dayにpythonコードを入れ、j拮抗していますが、隣に新たに作るPMOS_v2も読み込めるようにできるのですか？

---

できます！VS Codeの「マルチルートワークスペース」機能を使えば、複数のフォルダを同時に開けます。

![[Pasted image 20260321190407.png]]

---

## 手順（5分でセットアップ完了）

**Step 1 — PMOS_v2フォルダを作成**

Bionexus_vaultやPMOS_12dayと**同じ階層**に置きます。

```bash
# PMOS_12dayと同じ場所へ移動（例）
cd ~/Documents

# フォルダ作成
mkdir PMOS_v2
```

ダウンロードした `.py` ファイルをすべて `PMOS_v2/` の中に配置します。

---

**Step 2 — ワークスペースファイルを作成**

VS Codeで `ファイル` → `名前を付けてワークスペースを保存` から `PMOS.code-workspace` を作り、中身をこれに書き換えます（パスはご自身の環境に合わせてください）：

```json
{
  "folders": [
    {
      "name": "🧬 Bionexus Vault",
      "path": "../Bionexus_vault"
    },
    {
      "name": "🔧 PMOS_12day",
      "path": "../PMOS_12day"
    },
    {
      "name": "🚀 PMOS_v2",
      "path": "../PMOS_v2"
    }
  ],
  "settings": {
    "python.defaultInterpreterPath": "${workspaceFolder:🚀 PMOS_v2}/venv/bin/python"
  }
}
```

---

**Step 3 — `.env` のVaultパスを設定**

`PMOS_v2/.env` を開いて：

```
VAULT_PATH=../Bionexus_vault
```

これで `PMOS_v2` のコードが `Bionexus_vault` を自動的に読みに行きます。

---

**Step 4 — VS Codeで開く**

```bash
# ワークスペースファイルをダブルクリック、またはターミナルから
code PMOS.code-workspace
```

左サイドバーに3つのフォルダが並んで表示され、それぞれ独立して動作します。

---

Vaultの実際の場所（絶対パス）が分かれば、`.env` に入れる正確な値をお伝えできます。VS CodeのエクスプローラーでBionexus_vaultを右クリック →「パスのコピー」で確認できます。
