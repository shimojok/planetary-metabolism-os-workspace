## 実際の操作手順（Windows）

**Step 1 — PowerShellでフォルダ作成**

```powershell
cd "E:\Documents\BioNexus Holdings\BioNexus_vault"
mkdir PMOS_v2
mkdir PMOS_v2\agents
mkdir PMOS_v2\data
mkdir PMOS_v2\graph
mkdir PMOS_v2\targets\gates_foundation
mkdir PMOS_v2\targets\microsoft
mkdir PMOS_v2\memory
```

**Step 2 — ダウンロードしたファイルを配置**

```
PMOS_v2\config.py          ← ここへ
PMOS_v2\setup.py           ← ここへ
PMOS_v2\requirements.txt   ← ここへ
PMOS_v2\.env               ← ここへ（.env.exampleをコピーしてリネーム）
PMOS_v2\agents\explorer.py
PMOS_v2\agents\classifier.py
PMOS_v2\agents\critic.py
PMOS_v2\agents\orchestrator.py
```

**Step 3 — `.env` のパスを設定**

`PMOS_v2\.env` を開いて：

```
VAULT_PATH=..
OPENAI_API_KEY=sk-あなたのキー
```

`..` だけでOKです。PMOS_v2から1つ上 = BioNexus_vaultそのものを指します。

---

**Step 4 — ワークスペースファイルを作成**

`E:\Documents\BioNexus Holdings\BioNexus_vault\PMOS.code-workspace` を新規作成：

```json
{
  "folders": [
    {
      "name": "🧬 BioNexus Vault",
      "path": "."
    },
    {
      "name": "🔧 PMOS_12day",
      "path": "PMOS_12day"
    },
    {
      "name": "🚀 PMOS_v2",
      "path": "PMOS_v2"
    }
  ],
  "settings": {
    "python.defaultInterpreterPath": "${workspaceFolder:🚀 PMOS_v2}/../venv/Scripts/python.exe"
  }
}
```

**Step 5 — venvを共用（既存のものをそのまま使う）**

```powershell
# 既存のvenvを有効化
E:\Documents\BioNexus Holdings\BioNexus_vault\venv\Scripts\Activate.ps1

# PMOS_v2の追加パッケージをインストール
pip install openai tqdm rich python-dotenv
```

---

**Step 6 — VS Codeで開く**

`PMOS.code-workspace` をダブルクリック、または：

```powershell
code "E:\Documents\BioNexus Holdings\BioNexus_vault\PMOS.code-workspace"
```

これでVault・PMOS_12day・PMOS_v2の3つが左サイドバーに並び、既存のvenvも共用できます。`VAULT_PATH=..` と設定しているので、PMOS_v2のコードはVault内の全4800ノートを自動的に読みに行きます。
