# 2. モデルの基本構造

PMOS Hypercycleは **7ノード系**として定義します。

状態変数

```text
S(t)  Soil Microbiome
P(t)  Plant Metabolome
F(t)  Food System
M(t)  Human Microbiome
H(t)  Human Metabolic Health
W(t)  Organic Waste
R(t)  Fermentation Regeneration
```

循環

```
S → P → F → M → H → W → R → S
```

---

# 3. 基本ダイナミクス

各状態は

**自己増殖 + 上流ノード依存**

で変化します。

例

Soil 微生物
$$
\frac{dS}{dt}=\alpha_S S + \beta_R R S - \delta_S S
$$
意味

```
αS   = soil微生物の自然増殖
βR   = 発酵による再生効果
δS   = 劣化
```

---

# 4. Plant Metabolome
$$
\frac{dP}{dt}=\alpha_P P + \beta_S S P - \delta_P P
$$
意味

```
土壌微生物 → 植物代謝促進
```

---

# 5. Food System
$$
\frac{dF}{dt}=\alpha_F F + \beta_P P F - \delta_F F
$$

# 6. Human Microbiome

$$

# 17. Pythonシミュレーション（完全版）

例

```python
import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

def hypercycle(X,t):

    S,P,F,M,H,W,R = X

    alpha=[0.2]*7
    delta=[0.1]*7
    beta=0.15

    dS=alpha[0]*S + beta*R*S - delta[0]*S
    dP=alpha[1]*P + beta*S*P - delta[1]*P
    dF=alpha[2]*F + beta*P*F - delta[2]*F
    dM=alpha[3]*M + beta*F*M - delta[3]*M
    dH=alpha[4]*H + beta*M*H - delta[4]*H
    dW=beta*H - delta[5]*W
    dR=alpha[6]*R + beta*W*R - delta[6]*R

    return [dS,dP,dF,dM,dH,dW,dR]


t=np.linspace(0,200,2000)

X0=[1,1,1,1,1,1,1]

sol=odeint(hypercycle,X0,t)

plt.plot(t,sol)
plt.show()
```
