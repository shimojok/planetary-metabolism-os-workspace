ここでは **PMOS を GitHub 上で実装するための実行設計**を示します。目的は、**Vault → Knowledge Graph → AIエージェント → シミュレーション**を一貫して動かすリポジトリ構造を作ることです。GitHub自体は GitHub、クラウド実行基盤は Microsoft の Microsoft Azure を前提にします。

---

# 1. PMOS GitHub の基本思想

PMOSのGitHubは**単なるコード管理ではなく「Planetary Metabolic OS のモジュール群」**です。  
すべてのリポジトリは以下の3層構造に分類されます。

```
Science Layer
AI Layer
Infrastructure Layer
```

これにより、Kazさんの

- AGRIX
    
- HealthBook
    
- MBT55
    

がそれぞれ **独立モジュール**として統合されます。
