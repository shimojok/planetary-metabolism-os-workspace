# PMOS Knowledge Graph

## 基本構造

Knowledge Graph は

```text
Node（概念）
Edge（関係）
Weight（影響度）
Evidence（科学根拠）
```

で構成します。

JSON構造

```json
{
 "node": "Soil Microbiome",
 "relation": "increases",
 "target": "Nutrient Availability",
 "weight": 0.82,
 "evidence": "soil microbiology studies"
}
```

# PMOS Knowledge Graph

## 主要ドメイン（6領域）

PMOSは **6つの代謝領域**で構成します。

```
SOIL
FOOD
MICROBIOME
HEALTH
CLIMATE
ECONOMY
```

```

```

# Planetary Metabolic Hypercycle

PMOSの核心は **この循環**です。

```
Food Waste
↓
Microbial Fermentation
↓
Soil Microbiome
↓
Soil Fertility
↓
Crop Production
↓
Human Nutrition
↓
Metabolic Health
↓
Waste
```

このループが

```
Carbon Cycle
Nutrient Cycle
Health Cycle
```

を同時に回します。
