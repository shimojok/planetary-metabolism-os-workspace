# 1

# Planetary Metabolic Hypercycle の基本構造

地球代謝は **4つの状態変数**で表現できます。

```
S(t) = Soil metabolic capacity
F(t) = Food production & nutrient density
H(t) = Human metabolic health
C(t) = Atmospheric carbon level
```

補助変数

```
M(t) = Microbial biomass
W(t) = Organic waste flow
```

この6変数で **Planetary Metabolism**を記述します。

# 3

# Soil dynamics

土壌代謝能力

```
dS/dt = αM − βS − γC
```

意味

```
α = microbial regeneration coefficient
β = soil degradation rate
γ = climate stress factor
```

解釈

- 微生物が増えると土壌機能が回復
    
- 劣化で土壌能力が減少
    
- 気候ストレスで低下
    

---

# 4

# Microbial dynamics

微生物量

```
dM/dt = λW + μS − δM
```

意味

```
λ = fermentation efficiency
μ = soil microbial support
δ = microbial decay
```

ポイント

```
Waste → microbes → soil regeneration
```

これは **MBT55循環の数式化**です。

---

# 5

# Food production

食料生産

```
dF/dt = ηS + θM − κF
```

意味

```
η = soil productivity factor
θ = microbiome nutrient enhancement
κ = food system inefficiency
```

つまり

```
Soil + Microbiome → Food
```

---

# 6

# Human metabolic health

人間の健康状態

```
dH/dt = ρF + σM − ωH
```

意味

```
ρ = nutritional quality effect
σ = microbiome health effect
ω = disease burden
```

つまり

```
Food + Microbiome → Health
```

これは **HealthBookの中心モデル**になります。

---

# 7

# Carbon dynamics

炭素

```
dC/dt = E − ξS − ψF
```

意味

```
E = anthropogenic emissions
ξ = soil carbon sequestration
ψ = regenerative agriculture impact
```

解釈

```
Healthy soil → carbon storage
```

---

# 8

# Waste generation

廃棄物

```
dW/dt = aF + bH − cW
```

意味

```
a = food waste coefficient
b = metabolic waste
c = recycling efficiency
```

これが

```
Circular Bioeconomy loop
```

を作ります。

# 9

# Hypercycle 完成形

6方程式の連立系

```
dS/dt = αM − βS − γC
dM/dt = λW + μS − δM
dF/dt = ηS + θM − κF
dH/dt = ρF + σM − ωH
dC/dt = E − ξS − ψF
dW/dt = aF + bH − cW
```

これは

```
Planetary Metabolic System
```

です。
