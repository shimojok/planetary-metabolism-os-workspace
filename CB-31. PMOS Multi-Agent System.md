# Agent Communication

エージェント同士は **メッセージで通信**します。

例

```
Soil Agent
→ Agriculture Agent
soil fertility update
```

```
Food Agent
→ Health Agent
nutrition profile
```

---

# Communication Protocol

定義ファイル

```
agent_protocol.yaml
```

例

```yaml
message:
 sender: SoilAgent
 receiver: AgricultureAgent
 type: soil_update
 payload:
   soil_carbon: 3.2
   microbial_index: 0.71
```
