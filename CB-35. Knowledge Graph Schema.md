# 1

# Knowledge Graph Schema

まず **AIが理解するデータ構造**です。

```json
{
 "node_id": "string",
 "domain": "soil | food | microbiome | health | climate | economy",
 "name": "string",
 "description": "string",
 "relations": [
   {
     "type": "increases | decreases | regulates | causes | depends_on",
     "target": "node_id",
     "weight": 0.0
   }
 ]
}
```

意味

|項目|意味|
|---|---|
|node_id|ノード識別子|
|domain|分野|
|relations|関係|
|weight|影響度|
