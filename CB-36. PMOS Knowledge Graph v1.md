# Relation Rules（1000関係生成）

Knowledge Graphでは **関係はパターン生成**します。

主要関係タイプ

```
increases
decreases
regulates
depends_on
causes
improves
degrades
supports
connects_to
influences
```

例

```
soil_microbiome → increases → soil_fertility
soil_fertility → increases → crop_yield
crop_yield → increases → food_security
dietary_fiber → increases → microbiome_diversity
microbiome_diversity → improves → metabolic_health
metabolic_health → reduces → healthcare_cost
regenerative_agriculture → increases → soil_carbon
soil_carbon → reduces → atmospheric_co2
```
