# なぜ Obsidian Vault が必要なのか

理由は **3つ**あります。

### 1

## 知識グラフを人間側でも可視化できる

PMOSは本質的に

```
soil
food
microbiome
health
climate
economy
```

の **相互関係ネットワーク**です。

Obsidianは

```
ノート → ノート
```

のリンクを作ると

```
Knowledge Graph
```

として可視化できます。

つまり

**AI Knowledge Graph  
＋  
Human Knowledge Graph**

を **二重構造**で持てます。

---

### 2

## ChatGPTとの議論を構造化できる

今までの議論は

- Hypercycle
    
- MBT Sustainable Cycle
    
- AGRIX architecture
    
- HealthBook disease model
    
- PMOS
    

など **論文レベルの概念**です。

しかしチャットは

```
時系列
```

なので **構造が崩れやすい**です。

Obsidianなら

```
Concept
Architecture
Model
Algorithm
```

として整理できます。

---

### 3

## 将来のAI連携

重要なのはここです。

```
Markdown
```

つまり

```

↓
Vector DB

AI Agents
```

という構造にできます。
