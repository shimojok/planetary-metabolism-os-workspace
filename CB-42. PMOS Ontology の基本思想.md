# 1. PMOS Ontology の基本思想

Kazさんの研究は次の **3階層代謝系**です。

```
Planetary System
      ↓
Ecological Metabolism
      ↓
Human Metabolism
```

具体的には

```
Soil metabolism
↓
Plant metabolism
↓
Microbial metabolism
↓
Human metabolism
↓
Societal metabolism
```

これを **PMOS Ontology**として定義します。

# 4. Ontology Relation Types

Knowledge Graphでは  
**関係の種類**が重要です。

PMOSでは **7種類**を定義します。

```
increases
decreases
regulates
produces
consumes
depends_on
correlates_with
```

