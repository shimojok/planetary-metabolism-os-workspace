# 6 Python自動処理

VS Codeから実行できます。

処理フロー

```
Vault
↓
Markdown Parser
↓
Concept extraction
↓
Embedding
↓
Graph generation
```

Python例

```python
import os
import networkx as nx

G = nx.Graph()

for file in vault_files:

    title = extract_title(file)
    links = extract_links(file)

    G.add_node(title)

    for link in links:
        G.add_edge(title, link)
```

出力

