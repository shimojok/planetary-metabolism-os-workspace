ObsidianのVault（4500ノート）を一括取得します。

```python
import os

def load_markdown_files(vault_path):
    docs = []
    for root, _, files in os.walk(vault_path):
        for file in files:
            if file.endswith(".md"):
                path = os.path.join(root, file)
                with open(path, "r", encoding="utf-8") as f:
                    docs.append({
                        "path": path,
                        "content": f.read()
                    })
    return docs
```

# 【STEP 4】Graph構築

```python
import networkx as nx

G = nx.Graph()

def build_graph(nodes, edges):
    for n in nodes:
        G.add_node(n)
    for e in edges:
        G.add_edge(e[0], e[1])
```
