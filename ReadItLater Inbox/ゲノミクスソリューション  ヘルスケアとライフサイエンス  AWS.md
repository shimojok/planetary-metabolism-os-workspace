## AWS for Genomics でゲノミクスイノベーションを促進する

ゲノミクス向け AWS は、ゲノミクス業界のニーズと革新的なテクノロジーをマッチングさせ、ゲノムの発見を加速させるスケーラブルで安全、かつコスト効率の高いツールを提供します。ゲノムデータを AWS に移行して安全に保管するソリューションから、二次解析や三次解析を加速するツール、ゲノムデータをマルチモーダルデータセットに統合するサービスまで、ゲノミクス向け AWS は、ゲノミクスワークフロー全体で検証済みの AWS および AWS パートナーソリューションのキュレートされたポートフォリオを提供します。ほぼ 10 年にわたり、[Ancestry](https://aws.amazon.com/jp/solutions/case-studies/ancestry-case-study/)、[DNAnexus](https://aws.amazon.com/jp/solutions/case-studies/dnanexus/)、[Illumina](https://aws.amazon.com/jp/solutions/case-studies/illumina-case-study/)、[Genomics England](https://aws.amazon.com/jp/solutions/case-studies/genomics-england/)、[GRAIL](https://www.youtube.com/watch?v=eexsAyOJML4&t=786s) などの業界リーダーは、AWS を活用して、コスト削減、セキュリティ強化、データの有効活用を実現しています。

## データ転送と保存のソリューション

安全なデータ転送、グローバルなコラボレーション、ゲノムデータセットへのアクセスにより、ストレージコストを削減し、ゲノミクスイノベーションを加速します。
