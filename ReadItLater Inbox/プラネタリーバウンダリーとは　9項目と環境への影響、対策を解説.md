## 1.プラネタリーバウンダリーとは

プラネタリーバウンダリーとは、地球の環境に変化（とくに人間の影響）が加わってももとの状態に戻り、地球環境が安定した状態を保てる限界の範囲を示したものです。

スウェーデンのヨハン・ロックストローム博士らが、限界の範囲となる九つの要素を分析し、2009年に「プラネタリーバウンダリー」と呼ぶことを提唱しました。

### （1）プラネタリーバウンダリーの概要

地球の限界を知ることは、持続可能な社会を実現するうえで極めて重要なことです。自然には、もとの状態に戻ろうとする回復力が備わっています。そのため、人間が自然の回復力ぎりぎり（限界）まで影響を与えても、時間が経てば再び健全な状態に戻ると考えられています。

ところが、その限界を完全に超えてしまうと、地球に激しい変化が起こり、もとの状態には戻れない可能性があります。

人間は、農業や工業・商業などの活動（総称して「人間活動」といいます）によって自然の資源を利用し、森や海を破壊するなど、環境に影響を与えてきました。18世紀後半にイギリスで起こった産業革命までは、人間が利用する資源は少なく、ゴミなどの廃棄物も大半が自然にかえって浄化されていたため、地球の環境を大きく悪化させることはありませんでした。

しかし、産業革命以降は、石炭や石油などの化石燃料を使った工業活動が盛んになり、生産と消費のスピードが急速に早まっています。人口も飛躍的に増え、その食料を支えるために農地や放牧地も拡大しました。

その結果、環境への影響は、都市や農地だけでなく、大気や水に運ばれて世界の隅々にまで及ぶようになったため、地球そのものの限界が近づいていることが危惧されるようになったのです。

### （2）注目されている理由

プラネタリーバウンダリーは、地球の限界を脅かすさまざまな要因を科学的に分析し、一つの図に集約してわかりやすく示したことで、注目を集めました。

地球環境の悪化と言っても、その原因や状況はさまざまあるため、人によって捉え方が異なるでしょう。そのため、地球環境の全体像を理解することは専門家でも容易ではありませんでした。しかし、「持続可能な限界を超えたかどうか」に注目することで、多様な問題を同じ軸にのせて、俯瞰（ふかん）的に判断できるようになりました。

プラネタリーバウンダリーは、2015年に国連によって提唱された「持続可能な開発目標（SDGs）」とも深い関係があります。SDGsでは17の目標が設定されており、そのなかには「海の生態系」「陸の生態系」「気候変動」のように科学的な評価が必要なものが含まれています。水や食料など、資源や環境に関連する目標もプラネタリーバウンダリーの考え方と共通しています。

### （3）地球にかける四つの圧力

プラネタリーバウンダリーの限界を超えるほど、地球に変化が起きている根本的な原因は人間活動による地球への圧力です。

1950年ごろからは「人新世（じんしんせい）」と言われるほど人間活動が拡大し、地球規模で圧力を強めた時期と考えられています。ここでは、代表的な四つの圧力をご紹介します。

| 四つの圧力 | 地球に与える影響 |
| --- | --- |
| 人口の増加と豊かさの追求 | 世界の人口が80億人を突破している。その大部分を占める発展途上国の人々も、先進国と同様に大量のエネルギーや資源に依存する豊かな生活を求めている |
| 気候変動 | 人間活動によって大気中に放出された温室効果ガスが、豪雨や干ばつなどの極端気象の激甚化（げきじんか）を招いている |
| 生態系の急速な破壊 | 食料生産や都市化のために広大な自然が失われた。その結果、多くの野生生物が絶滅の危機にさらされており、生態系のバランスが崩れる恐れがある |
| 地球環境の突発的、不可逆的、広範な変化のリスク | 地球環境には、あるレベルを超えると急激な変化と影響を引き起こす仕組みがある。代表的なものとして、極域の永久凍土の融解や大西洋の深層水の循環、アマゾンの熱帯雨林の消失などが挙げられる |

## 2.プラネタリーバウンダリーの9項目

![プラネタリーバウンダリーの9項目](https://p0.potaufeu.asahi.com/b7c6-p/picture/28058796/bf4c7f7b911f0cd83ef484a0def26f96.png)

[Current status of control variables for all nine planetary boundaries](https://www.science.org/doi/10.1126/sciadv.adh2458)をもとに筆者作成

プラネタリーバウンダリーに設定されている、地球の限界の範囲を判断する9項目は以下のとおりです。なお、「海洋酸性化」「成層圏オゾン層の破壊」「大気エアロゾルによる負荷」以外の6項目は、現在すでに限界を超えている状態です。

それぞれ詳しく解説します。

| 9項目 | 内容 |
| --- | --- |
| 気候変動 | 大気中の二酸化炭素（CO2）濃度の増加によって地球温暖化が進む |
| 成層圏オゾン層の破壊 | 生物を紫外線から守る成層圏オゾンが、フロンなど化学物質により破壊される |
| 海洋酸性化 | 大気中のCO2が海洋に溶解して海水が酸性化し、生物に悪影響を与える |
| 生物圏の健全さ | 原生林の破壊などによって、生物多様性や生態系のバランスが失われる |
| 生物地球化学的循環 | 農地での肥料の過剰使用などによって、窒素やリンが環境中に多量に流出する |
| 淡水利用 | 地下水や湖沼などの淡水資源が農業・工業活動のため多量に使用されて枯渇する |
| 土地利用変化 | 農地や都市の拡大のため自然の生態系とその回復力が失われる |
| 新規化学物質 | プラスチック・農薬などの化学物質や放射性物質が環境中に広がり、悪影響を及ぼす |
| 大気エアロゾルによる負荷 | 工業活動や火災から放出されたエアロゾルが健康被害などを引き起こす |

### （1）地球システム全体に影響を与える

これから紹介する三つの項目は、地球全体に深刻な影響が及ぶため「ビッグ・スリー」と呼ばれる項目です。

#### ①気候変動

気候変動は、地球温暖化を防ぐために、大気中のCO2濃度の限界値を定めた項目です。

化石燃料の燃焼や森林破壊によって、CO2やメタンなどが大気に放出されています。これらがもたらす温室効果によって、地球のエネルギーバランスが変化し、気候（温度や降水）の変化が急激な規模で進んでいます。

限界値は大気中のCO2濃度が350ppm（ppmは気体に占める割合が100万分の1）でしたが、現状は420ppm近くと限界値を超えている状況です。

#### ②成層圏オゾン層の破壊

成層圏オゾン層の破壊は、成層圏オゾンを回復させるために、オゾン量の限界値を定めたものです。

冷媒として用いられるフロンなどの化学物質が大気中に放出され、それが紫外線を防ぐ成層圏オゾン層を破壊することで、生物に悪影響が生じます。

モントリオール議定書によって使用が規制されており、現在ではほぼ限界値（290ドブソンユニット〈オゾン量の単位〉）付近まで成層圏オゾンが回復していると考えられています。

#### ③海洋酸性化

海洋の酸性化は、CO2が海水に溶け込んで酸性化することを防ぐために、海水の炭酸化の限界値を定めた項目です。

大気中に放出されたCO2の一部は海水に溶け込むことで炭酸となり、貝類や甲殻類など、炭酸カルシウムを利用する生物に深刻な影響を与える恐れがあります。

限界値は炭酸カルシウムの利用しやすさを示す「アラゴナイト飽和度（Ωara）」で設定されています。現在閾値（いきち）に達していないものの、徐々に近づきつつあるとされています。

### （2）局所的に影響を与える

これから紹介する4項目は、農業などの人間活動の周辺で徐々に進行しており、地球環境に強い影響を与える項目です。

#### ①生物圏の健全さ

生物圏の健全性は、生物圏の多様性や働きを健全に保つために、絶滅速度と生産物の利用率を定めた項目です。

環境の調節や遺伝子資源などの便益（生態系サービス）をもたらす生物圏の健全性は、人間活動による破壊や劣化によって徐々に失われています（参照：[生物多様性と生態系サービス｜環境省](https://www.biodic.go.jp/biodiversity/activity/policy/valuation/service.html)）。

その限界は生物多様性の喪失を示す種の絶滅速度（100万年・種あたり1種が絶滅）と、植物による生産物の人間による利用率（産業革命前の10％未満）で設定されていますが、現状はどちらも大きく上回っています。

#### ②生物地球化学的循環

生物地球化学的循環は大気や水質、土壌の清浄さを守るために、窒素やリンの限界値を定めた項目です。

人間活動で使用する窒素やリンが、さまざまな経路（生物地球化学的循環）を通じて流れ出ることで、大気や水質、土壌に深刻な汚染をもたらしています。そのため農地や都市など人間社会に近いところでより激しい影響が見られています。

とくに、過剰な肥料が農地で使用されていることが問題視されています。リンの限界値は、農地から河川を通って海に流れ出る量（年間1100万t）、窒素については人工的に固定される大気中窒素の量（6200万t）と設定されています。

しかし、現状は食料生産のため大きく上回る量の肥料が使われています。

#### ③淡水利用

淡水利用は、枯渇の危機に瀕（ひん）している地下水や湖沼などを守るために、淡水資源の変化幅に限界値を定めた項目です。

農地や耕地への給水や生活用水、工業用水として大量の淡水が利用されていることで、水資源の枯渇が深刻化しています。

人間活動による淡水資源の変化幅に限界が設定されており、現状では、安全な範囲を超えてリスクが高まっていると考えられています。なお、淡水資源には作物などの植物に利用される「グリーンウォーター」と、地下に染み込む「ブルーウォーター」があります。

#### ④土地利用変化

土地利用変化は、自然の生態系と回復力を保護するために、森林面積の限界値を定めた項目です。

森林などの自然環境を切り開いて農地や都市として利用したため、広大な自然の生態系が失われ回復できない状態となっています。

産業革命前の森林面積を基準とし、どれだけ残っているかで限界が設定されています。産業革命前を基準に75％までが安全な範囲となっていますが、現状は世界全体で60％程度まで減ったとされています。

### （3）人間の健康と地球システムに影響を与える

これから紹介する2項目は、主に工業活動によって新たに生じた項目です。知見が少なく判断が困難でしたが、2023年には評価が示されました。

#### ①新規化学物質

人工的に作り出された新しい化学物質は、大気や陸・海での動きや生物への作用が不明なものが多く、正しく評価するのは難しいとされていました。

しかし、マイクロプラスチックのように環境へ悪影響を与える新規化学物質が広く認識され始めたため、現在はリスクが高い項目と捉えられています。

#### ②大気エアロゾルによる負荷

大気エアロゾルによる負荷は、人間の健康被害を防ぐために、大気へのエアロゾル放出量の限界値を定めた項目です。

工業活動などによって大気に放出される微粒子（エアロゾル）は、大気汚染物質として人間の健康だけでなく、地球の気候にも影響を与えています（参照：[エアロゾルとは｜日本エアロゾル学会](https://www.jaast.jp/new/about_aerosol.html)）。

人間活動が盛んな北半球でエアロゾルが多いことから、北半球と南半球の平均的なエアエロゾル量（単位は光学的厚さ）の差により限界が設定されています。現在は、限界とされる0.1を下回っています。

## 3.プラネタリーバウンダリーが限界値を超える原因

人間活動によって地球の資源を消費し、環境に過剰な負荷をかけるようになった原因は多くありますが、ここでは主な三つをご紹介します。

| 限界値を超える原因 | 概要 |
| --- | --- |
| 人口の増加 | 急激な人口増加によって、生物圏の健全さや生物地球化学的循環に関する限界を超えている |
| 経済活動の拡大 | 経済活動が拡大したことで、地球への負荷が加速している |
| 生活習慣の変容 | 生活の欧米化が進んだことで、飼料や化石燃料の消費が増加している |

### （1）人口の増加

地球環境が悪化している根本的な原因は、急激な人口増加です。

産業革命前の世界の人口は10億人以下でしたが、2022年11月には80億人を超えています。人口を支えるために居住地や農地を拡大してきたことで、広大な自然が失われて生物圏の健全さや生物地球化学的循環に関する限界を超えることになりました。

### （2）経済活動の拡大

交通と流通が発達し、国や大陸を超えて経済活動が拡大する、いわゆるグローバリゼーションが進んだことが地球への負荷を加速しています。

欧米諸国による開拓と植民地化は19世紀以前からおこなわれ、広大な熱帯林がプランテーション（大規模農園）に変わり、大自然が失われています。その結果、土地利用変化に関する限界を超えることにつながりました。

現在では欧米を中心に多数のグローバル企業が存在しています。世界にまたがる経済活動を促し、消費を拡大させてきたため、化石燃料だけでなく、プラスチックなど新規化学物質の消費を加速させています。

### （3）生活習慣の変容

世界中で生活の欧米化が進んだことが、環境への負荷を増大させています。

発展途上国を含む世界中で、肉を多く消費する食生活や自動車の普及が進んできました。

そのため、生物地球化学的循環を乱す肥料や、気候変動を招く化石燃料の消費が大きく増加しています。また、生活の変化を求めて人口が都市に集中し、緑が少なく生活用水などの莫大（ばくだい）な資源を消費する大都市が出現しました。

## 4.プラネタリーバウンダリーの範囲内で活動するための取り組み

プラネタリーバウンダリーを超えないようにすることは、人類が将来にわたり持続的に生活していくうえで不可欠です。さまざまな対策や取り組みが考えられますが、ここでは代表的な取り組みを三つご紹介します。

| 取り組み | 概要 |
| --- | --- |
| 気候変動の緩和 | 脱炭素社会を実現し、大気中の温室効果ガス濃度を安全なレベルに維持する |
| 環境と調和した農業 | 飼料や淡水の使用を適正なレベルに保ち、環境と農業を調和させる |
| 生物多様性の保全 | 生物種の絶滅を抑え、生物圏の健全性を保つ |

### （1）気候変動の緩和

再生可能エネルギーの普及など、化石燃料に依存しない脱炭素社会を実現し、大気中の温室効果ガス濃度を安全なレベルに維持する必要があります。

すでに気候変動の影響は世界中で起こっており、気象災害の激甚化など人命を脅かすレベルにあります。

国際社会は、国連気候変動枠組み条約・パリ協定において、世界の平均気温の上昇を設定しました。産業革命と比較して、世界共通の長期目標を2度に設定、ひとまずは1.5度に抑えることを掲げて取り組みを進めています（参照：[パリ協定の概要｜外務省](https://www.mofa.go.jp/mofaj/ic/ch/page1w_000119.html)）。

具体的な取り組みとして、2050年までに温室効果ガスの排出量を正味でゼロにするほか、影響を受けやすく対策が難しい途上国への支援などが掲げられています。

### （2）環境と調和した農業

農地での肥料使用や、農地や耕地への給水などの灌漑（かんがい）による淡水の使用を適正なレベルに保ち、環境と農業が調和するための取り組みが重要になります。

適正なレベルに保つことは、生物地球化学的循環や淡水資源を限界範囲内に抑えることにつながるため、農家や国によって、さまざまなレベルでの対策が進められています。たとえば、農地の土づくりを進めることで、化学肥料や農薬の使用量を抑える対策があります。

ただし、農地の拡大を抑えることは生物圏の健全化に寄与する一方、食料生産とのバランスも考えなければいけません。

### （3）生物多様性の保全

生物圏の健全性を保つために、生物種の絶滅を抑える取り組みが重要です。この取り組みは、気候変動や土地利用に関する効果も見込めます。

国際社会では、昆明・モントリオール枠組みを策定し、2030年までに自然を回復軌道に乗せるための取り組みを進めています。保護地を増やしたり、自然と調和した農業・林業を進めたりするなどの対策が含まれます（参照：[昆明・モントリオール生物多様性枠組｜環境省](https://www.biodic.go.jp/biodiversity/about/treaty/gbf/kmgbf.html)）。

## 5.プラネタリーバウンダリーの課題

九つあるプラネタリーバウンダリーのうち六つで限界を超えた危険水準に達しており、ほかの三つも徐々に限界に近づいています。このような地球の危機的な状況を解決するうえでの課題を解説します。

| 課題 | 概要 |
| --- | --- |
| 土地をめぐる競合の解消 | 住宅地や放牧地などの拡大によって、里山や熱帯林が失われている |
| 自然を活用した対策 | 気候変動への対策が期待通り進んでいないため、自然を活用した対策が求められる |
| 持続可能な都市へ | 大都市の出現によって、大量の資源消費と環境への負荷が加速している |

### （1）土地をめぐる競合の解消

自然が減少しているなか、利用できる地球上の土地は限られているため、土地の使い方で競合が起こっています。

たとえば、都市の周辺では住宅地を広げるために自然が残る里山が失われたり、熱帯林でもアブラヤシ園や放牧地のために原生林が犠牲になったりしています。

土地を適正に利用することは、自然の生態系を残して生物圏の健全さを保つことにつながります。また、農作物の生産に必要な分だけの肥料や水を使うようにすることで、農地での余分な資源の使用も抑制できるでしょう。

### （2）自然を活用した対策

気候変動への対策は、化石燃料からの排出量を抑えることが基本になりますが、電気やガソリンの使用とも結びついており、期待通りに進んでいません。

大気中のCO2を固定して地中に埋める隔離技術も研究されていますが、将来、大気に漏れ出るリスクもあります。そこで、リスクが低く生物多様性の保全などと調和して進められる「自然に基づく対策（NbS）」が注目されています。

たとえば、樹木の光合成により大気のCO2を減らすことで、気候変動や海洋酸性化の防止が期待できます。

### （3）持続可能な都市へ

今や世界人口の半分以上を占める大都市の出現が、生活様式を変化させ、大量の資源消費と環境への負荷を加速させています。

都市では食料や水、資材を周辺の広い地域から集めて消費し、その代わりに温室効果ガス、エアロゾル、成層圏オゾン破壊物質、プラスチックなどの新規化学物質を大気や河川を通じて都市外へ放出しています。

環境に優しい都市への変革は、世界で大きな課題となっています。緑地など都市の構造だけでなく住民の意識を変えて脱炭素化を進めることが重要です。都市外への廃棄物の排出を抑えることは、都市そのものだけでなく地球全体の持続可能性を高めることにつながります。（参照：[持続可能な都市をめざして｜国立環境研究所](https://www.nies.go.jp/kanko/kankyogi/55/04-09.html)）。

## 6.地球の健康診断の先にあるもの

プラネタリーバウンダリーは、人間に例えると健康診断のようなものです。そのため、要検査・要治療とされた項目を見過ごすことは致命的な結果につながりかねません。健康診断と同様に定期的に検査し、状態の変化を敏感に察知することで、高リスクな項目の悪化を防ぎ、対策を促せます。

この記事で解説した9項目は、互いに独立しているわけではありません。たとえば気候変動と土地利用、生物圏の健全さは森林の保全が共通項になるように、相互につながっています。

つまり、複数の要因を同時に解決し、持続可能な社会を実現する対策を考えていくことが重要です。SDGsの目標達成は、持続可能な社会を実現する手がかりになるでしょう。

また、プラネタリーバウンダリーは、科学的な評価によって限界とリスクが判断されていることが重要です。正しい限界を設定するには、地球に関する理解を深めなければいけません。リスクを察知するには、継続的にモニタリングする必要があります。

そして、そのような限界とリスクに関する理解を持続可能な社会に向けた変革につなげるには、科学と社会の緊密なコミュニケーションが不可欠です。
